细胞器

细胞器的分类

根据细胞器是否具有膜结构，可将其分为两大类：

  • 膜性细胞器：具有膜结构的细胞器，如细胞核、线粒体、内质网、高尔基体、溶酶体、过氧化物酶体等
  • 非膜性细胞器：不具有膜结构的细胞器，如核糖体、中心体、细胞骨架等

主要细胞器的结构与功能

细胞核

结构

  • 核膜：双层膜结构，有核孔复合体
  • 核仁：rRNA合成和核糖体亚基组装的场所
  • 染色质：DNA与组蛋白组成的复合物
  • 核基质：核内的蛋白质网络

功能

  • 遗传信息储存：储存DNA
  • 基因表达调控：转录调控
  • 核糖体亚基组装：在核仁中进行

线粒体

结构

  • 外膜：光滑，有孔蛋白
  • 内膜：向内折叠形成嵴，有ATP合酶
  • 膜间隙：内外膜之间的空间
  • 基质：内膜包围的空间，含有DNA、核糖体等

功能

  • 细胞呼吸：进行有氧呼吸，产生ATP
  • 细胞凋亡：释放细胞色素c，启动凋亡程序
  • 钙离子储存：调节细胞内钙离子浓度

内质网

结构

  • 粗面内质网：表面附着核糖体
  • 滑面内质网：表面光滑，无核糖体附着

功能

  • 蛋白质合成与加工：粗面内质网
  • 脂质合成：滑面内质网
  • 解毒：滑面内质网
  • 钙离子储存：滑面内质网

高尔基体

结构

  • 顺面：靠近内质网的一侧
  • 中间区室：位于顺面和反面之间
  • 反面：远离内质网的一侧

功能

  • 蛋白质加工与分拣：糖基化、磷酸化等
  • 脂质修饰：糖脂合成
  • 溶酶体形成：初级溶酶体的形成

溶酶体

结构

  • 单层膜结构：内含多种水解酶
  • 酸性环境：pH约为4.5-5.0

功能

  • 细胞内消化：降解大分子物质
  • 自噬：降解细胞自身组分
  • 细胞外消化：释放酶到细胞外

过氧化物酶体

结构

  • 单层膜结构：内含多种氧化酶
  • 晶体核心：含有尿酸氧化酶等

功能

  • 脂肪酸氧化：β-氧化
  • 过氧化氢分解：含有过氧化氢酶
  • 胆固醇合成：部分步骤

核糖体

结构

  • 大亚基：含有肽酰转移酶活性中心
  • 小亚基：含有mRNA结合位点

功能

  • 蛋白质合成：翻译mRNA

细胞骨架

组成

  • 微管：由微管蛋白组成
  • 微丝：由肌动蛋白组成
  • 中间纤维：由多种蛋白质组成

功能

  • 维持细胞形态
  • 细胞运动
  • 细胞内运输
  • 细胞分裂

细胞器之间的相互关系

分泌途径

分泌途径是指蛋白质从内质网合成到细胞外释放的过程：

  1. 内质网合成：蛋白质在粗面内质网合成
  2. 高尔基体加工：蛋白质在高尔基体进行糖基化等修饰
  3. 分泌囊泡运输：蛋白质包装到分泌囊泡中
  4. 胞吐作用：囊泡与细胞膜融合，释放内容物

内吞途径

内吞途径是指细胞外物质进入细胞的过程：

  1. 胞吞作用：细胞膜内陷形成囊泡
  2. 早期内体：囊泡与早期内体融合
  3. 晚期内体：早期内体成熟为晚期内体
  4. 溶酶体降解：晚期内体与溶酶体融合，降解内容物

细胞器间的膜运输

细胞器之间通过囊泡运输进行物质交换：

  • COPII囊泡：从内质网到高尔基体
  • COPI囊泡：从高尔基体到内质网
  • 网格蛋白囊泡：从高尔基体到溶酶体，或从细胞膜到内体

细胞器的生物发生

线粒体和叶绿体的内共生起源

线粒体和叶绿体起源于内共生细菌，具有以下证据：

  • 双层膜结构：类似细菌的外膜和内膜
  • 环状DNA：类似细菌基因组
  • 70S核糖体：类似细菌核糖体
  • 自主分裂：类似细菌的二分裂

其他细胞器的形成

其他细胞器主要通过膜的分裂和融合形成：

  • 内质网：通过膜的分裂和融合扩展
  • 高尔基体：通过膜的分裂和融合形成扁平囊泡
  • 溶酶体：通过高尔基体出芽形成
  • 过氧化物酶体：通过已有过氧化物酶体分裂形成
