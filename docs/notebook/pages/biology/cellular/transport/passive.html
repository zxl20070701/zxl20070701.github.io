被动运输

被动运输的基本概念

被动运输是指物质顺浓度梯度或顺电化学梯度进行的跨膜运输，不消耗能量（ATP）。被动运输的驱动力是浓度梯度或电化学梯度，物质总是从高浓度区域向低浓度区域移动，直到达到平衡状态。

简单扩散

定义与特点

简单扩散是指小分子脂溶性物质直接通过脂质双分子层的被动运输方式。这种方式不需要膜蛋白的协助，完全依赖于物质的浓度梯度。

影响因素

  • 浓度梯度：梯度越大，扩散速率越快
  • 脂溶性：脂溶性越强，扩散速率越快
  • 分子大小：分子越小，扩散速率越快
  • 膜厚度：膜越薄，扩散速率越快
  • 温度：温度越高，扩散速率越快

通过简单扩散跨膜的物质

  • 气体分子：O₂、CO₂、N₂
  • 脂溶性分子：乙醇、甘油、脂肪酸、类固醇激素
  • 疏水性小分子：苯、甲苯

易化扩散

定义与特点

易化扩散是指水溶性物质在膜蛋白协助下进行的被动运输。这种方式需要特定的膜蛋白（通道蛋白或载体蛋白）参与，但仍顺浓度梯度进行，不消耗能量。

通道蛋白介导的易化扩散

特点

  • 高选择性：每种通道蛋白只允许特定离子或小分子通过
  • 高速度：每秒可通过数百万个离子
  • 门控性：通道蛋白具有"门"的功能，可开可关

类型

  • 电压门控通道：受膜电位变化调控，如Na⁺通道、K⁺通道
  • 配体门控通道：受化学物质结合调控，如乙酰胆碱受体通道
  • 机械门控通道：受机械力刺激调控，如听觉毛细胞中的通道
  • 应力激活通道：受膜张力变化调控

载体蛋白介导的易化扩散

特点

  • 特异性：每种载体蛋白只运输特定物质
  • 饱和性：运输速率有上限，当所有载体蛋白都被占用时达到最大速率
  • 竞争性抑制：结构相似的物质可竞争同一载体蛋白

机制

载体蛋白介导的易化扩散通过构象变化实现物质的跨膜运输：

  1. 物质结合到载体蛋白的一侧
  2. 载体蛋白发生构象变化
  3. 物质被释放到膜的另一侧
  4. 载体蛋白恢复原构象

代表性载体蛋白

  • 葡萄糖转运蛋白（GLUT）：介导葡萄糖的跨膜运输
  • 氨基酸转运蛋白：介导氨基酸的跨膜运输
  • 核苷酸转运蛋白：介导核苷酸的跨膜运输

渗透

定义与特点

渗透是指水分子通过半透膜从低溶质浓度区域向高溶质浓度区域的被动运输。半透膜允许水分子自由通过，但限制溶质分子的通过。

渗透压

渗透压是指阻止水分子渗透所需的压力。渗透压的大小与溶质浓度成正比，溶质浓度越高，渗透压越大。

溶液类型

  • 等渗溶液：与细胞内液渗透压相等的溶液，细胞体积不变
  • 低渗溶液：渗透压低于细胞内液的溶液，水进入细胞，细胞膨胀
  • 高渗溶液：渗透压高于细胞内液的溶液，水离开细胞，细胞皱缩

水通道蛋白（水孔蛋白）

水通道蛋白（aquaporin）是一类专门介导水分子跨膜运输的膜蛋白。1988年，彼得·阿格雷发现水通道蛋白，并因此获得2003年诺贝尔化学奖。

结构特点

  • 四聚体结构：由四个亚基组成，每个亚基形成一个水通道
  • 选择性：只允许水分子通过，排除离子和其他小分子
  • 高速度：每秒可通过数十亿个水分子

功能与分布

  • 肾脏：近曲小管和集合管，调节水的重吸收
  • 红细胞：快速调节细胞体积
  • 植物：液泡膜和质膜，调节细胞膨压
  • 眼睛：晶状体和角膜，维持透明度

离子通道的电生理特性

膜电位

膜电位是指细胞膜内外两侧的电位差，主要由离子浓度梯度和离子选择性通透性决定。静息膜电位通常为-70mV（内负外正）。

平衡电位

平衡电位是指某种离子净流动为零时的膜电位，可用能斯特方程计算： 

E = (RT/zF) × ln([ion]out/[ion]in)

其中，E是平衡电位，R是气体常数，T是绝对温度，z是离子价数，F是法拉第常数，[ion]out和[ion]in分别是细胞外和细胞内的离子浓度。

动作电位

动作电位是指细胞膜电位发生的快速、短暂、可传播的变化，主要由Na⁺和K⁺通道的依次开放和关闭引起。

动作电位的阶段

  1. 静息期：膜电位维持在静息水平
  2. 去极化期：Na⁺通道开放，Na⁺内流，膜电位升高
  3. 反极化期：膜电位达到正值
  4. 复极化期：K⁺通道开放，K⁺外流，膜电位下降
  5. 超极化期：膜电位暂时低于静息电位

被动运输的生理意义

维持细胞内外物质平衡

被动运输使细胞能够快速调节细胞内外物质的浓度，维持细胞内环境的稳定。

细胞信号转导

离子通道的开放和关闭是细胞信号转导的重要机制，如神经冲动的传导、肌肉收缩的调节等。

细胞体积调节

水通道蛋白介导的水分子运输是细胞体积调节的重要机制，如肾脏对尿液的浓缩和稀释。

物质吸收与分泌

肠道上皮细胞通过被动运输机制吸收营养物质，如葡萄糖、氨基酸等。
