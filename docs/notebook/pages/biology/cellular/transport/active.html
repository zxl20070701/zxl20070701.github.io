主动运输与胞吞、胞吐

主动运输的基本概念

主动运输是指物质逆浓度梯度或逆电化学梯度进行的跨膜运输，需要消耗能量（ATP）。主动运输的驱动力是能量（主要是ATP水解），物质从低浓度区域向高浓度区域移动，维持细胞内外的浓度差。

原发性主动运输

定义与特点

原发性主动运输是指直接利用ATP水解提供能量的主动运输方式。这种方式通过ATP驱动的泵蛋白实现物质的跨膜运输。

Na⁺-K⁺泵

结构

Na⁺-K⁺泵是一种P型ATP酶，由α和β两个亚基组成。α亚基含有ATP结合位点和离子结合位点，β亚基具有调节功能。

工作机制

  1. 细胞内3个Na⁺结合到泵蛋白内侧
  2. ATP水解，泵蛋白磷酸化，构象变化
  3. 3个Na⁺被释放到细胞外
  4. 细胞外2个K⁺结合到泵蛋白外侧
  5. 泵蛋白去磷酸化，构象恢复
  6. 2个K⁺被释放到细胞内

生理意义

  • 维持细胞膜电位：建立和维持Na⁺/K⁺梯度，形成膜电位
  • 维持细胞渗透压：防止细胞因渗透压变化而膨胀或皱缩
  • 为继发性主动运输提供驱动力：建立Na⁺梯度，驱动葡萄糖、氨基酸等的协同运输

Ca²⁺泵

结构

Ca²⁺泵也是一种P型ATP酶，存在于细胞膜、内质网膜和肌浆网膜上。

工作机制

Ca²⁺泵通过ATP水解提供能量，将Ca²⁺从细胞质泵入内质网或细胞外，维持细胞内低钙浓度（约10⁻⁷ M）。

生理意义

  • 肌肉收缩调节：肌浆网Ca²⁺泵调节肌肉收缩和舒张
  • 信号转导：维持细胞内低钙浓度，使Ca²⁺可作为信号分子
  • 细胞凋亡调控：内质网Ca²⁺释放参与细胞凋亡

质子泵

类型

  • P型质子泵：存在于植物细胞膜、真菌细胞膜
  • V型质子泵：存在于溶酶体膜、内体膜、液泡膜
  • F型质子泵：存在于线粒体内膜、叶绿体类囊体膜，也称为ATP合酶

功能

  • 建立质子梯度：为ATP合成提供驱动力
  • 维持细胞器内酸性环境：如溶酶体、内体的酸性环境
  • 调节细胞pH值：维持细胞质pH稳定

继发性主动运输

定义与特点

继发性主动运输是指利用离子梯度（主要是Na⁺梯度）提供能量的主动运输方式。这种方式不直接消耗ATP，而是利用原发性主动运输建立的离子梯度驱动物质的跨膜运输。

同向运输（协同运输）

机制

同向运输是指两种物质同时向同一方向运输的过程。通常是Na⁺顺浓度梯度进入细胞，同时带动另一种物质（如葡萄糖、氨基酸）逆浓度梯度进入细胞。

代表性转运体

  • SGLT1：小肠上皮细胞中的Na⁺-葡萄糖协同转运蛋白
  • SGLT2：肾脏近曲小管中的Na⁺-葡萄糖协同转运蛋白
  • GAT：神经元的Na⁺-氨基酸协同转运蛋白

反向运输（对向运输）

机制

反向运输是指两种物质向相反方向运输的过程。通常是Na⁺顺浓度梯度进入细胞，同时另一种物质逆浓度梯度排出细胞。

代表性转运体

  • Na⁺/Ca²⁺交换体：心肌细胞中，3个Na⁺进入细胞，1个Ca²⁺排出细胞
  • Na⁺/H⁺交换体：调节细胞内pH值，1个Na⁺进入细胞，1个H⁺排出细胞

胞吞作用

定义与特点

胞吞作用是指细胞通过细胞膜内陷形成囊泡，将细胞外物质摄入细胞内的过程。胞吞作用可以摄入大分子物质、颗粒物质甚至整个细胞。

类型

吞噬作用

吞噬作用是指细胞摄入大颗粒物质（如细菌、细胞碎片）的过程。主要发生在巨噬细胞、中性粒细胞等专业吞噬细胞中。

胞饮作用

胞饮作用是指细胞摄入液体和小分子物质的过程。所有真核细胞都能进行胞饮作用，是一种非选择性的物质摄入方式。

受体介导的胞吞作用

受体介导的胞吞作用是指细胞通过特异性受体识别和结合细胞外物质，然后通过胞吞作用将其摄入细胞内的过程。这种方式具有高度特异性和高效性。

机制

  1. 识别与结合：细胞外物质与细胞膜上的受体结合
  2. 膜内陷：细胞膜内陷形成囊泡
  3. 囊泡形成：囊泡从细胞膜分离，进入细胞质
  4. 囊泡运输：囊泡与内体融合
  5. 物质处理：物质被分拣、降解或循环利用

代表性例子

  • 低密度脂蛋白（LDL）摄入：通过LDL受体介导的胞吞作用
  • 转铁蛋白摄入：通过转铁蛋白受体介导的胞吞作用
  • 病毒入侵：某些病毒利用受体介导的胞吞作用进入细胞

胞吐作用

定义与特点

胞吐作用是指细胞通过囊泡与细胞膜融合，将细胞内物质释放到细胞外的过程。胞吐作用可以释放分泌蛋白、神经递质、代谢废物等。

类型

组成型胞吐作用

组成型胞吐作用是指持续进行的胞吐作用，不需要特定信号刺激。这种方式主要分泌细胞外基质蛋白、膜蛋白等。

调节型胞吐作用

调节型胞吐作用是指受信号刺激才发生的胞吐作用。这种方式主要分泌激素、神经递质、消化酶等，需要Ca²⁺等信号分子的参与。

机制

  1. 囊泡形成：物质包装到囊泡中
  2. 囊泡运输：囊泡沿细胞骨架运输到细胞膜附近
  3. 囊泡锚定：囊泡锚定在细胞膜上
  4. 囊泡融合：囊泡与细胞膜融合，形成融合孔
  5. 物质释放：囊泡内容物释放到细胞外
  6. 囊泡回收：囊泡膜被回收利用

代表性例子

  • 神经递质释放：神经元通过调节型胞吐作用释放神经递质
  • 激素分泌：内分泌细胞通过调节型胞吐作用释放激素
  • 消化酶分泌：胰腺腺泡细胞通过调节型胞吐作用释放消化酶

膜泡运输的调控

囊泡形成

囊泡形成需要衣被蛋白（如网格蛋白、COPI、COPII）的参与，这些蛋白帮助囊泡从供体膜出芽形成。

囊泡运输

囊泡运输需要分子马达（如驱动蛋白、动力蛋白、肌球蛋白）沿细胞骨架（微管、微丝）运输囊泡。

囊泡锚定与融合

囊泡锚定与融合需要SNARE蛋白的参与，包括v-SNARE（囊泡上的SNARE蛋白）和t-SNARE（靶膜上的SNARE蛋白）的特异性识别和结合。

主动运输与膜泡运输的生理意义

维持细胞内外物质浓度差

主动运输和膜泡运输使细胞能够逆浓度梯度运输物质，维持细胞内外的浓度差，这对于细胞的正常功能至关重要。

细胞信号转导

神经递质的释放和受体的内化都是通过膜泡运输实现的，这是细胞信号转导的重要机制。

细胞免疫防御

巨噬细胞通过吞噬作用清除病原体，B细胞通过受体介导的胞吞作用摄取抗原，这些都是细胞免疫防御的重要机制。

细胞营养吸收

肠道上皮细胞通过协同运输吸收葡萄糖、氨基酸等营养物质，这是细胞营养吸收的重要机制。
