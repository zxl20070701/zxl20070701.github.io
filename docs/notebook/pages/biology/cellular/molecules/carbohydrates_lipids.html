糖类和脂质

糖类——生命的燃料

糖类的定义与通式

糖类是多羟基醛或多羟基酮及其缩合物和衍生物的总称，大多数糖类的通式为(CH₂O)n，因此又称为碳水化合物。

糖类的分类

根据水解情况，糖类可分为三类：

  • 单糖：不能水解的最简单糖类
  • 二糖：由两个单糖分子脱水缩合而成
  • 多糖：由多个单糖分子脱水缩合而成

单糖

结构特点

单糖根据碳原子数目可分为丙糖（3碳）、丁糖（4碳）、戊糖（5碳）、己糖（6碳）等。根据官能团可分为醛糖和酮糖。

重要单糖

  • 葡萄糖（C₆H₁₂O₆）：最重要的能源物质，是细胞呼吸的主要底物
  • 果糖：最甜的天然糖，存在于水果和蜂蜜中
  • 半乳糖：乳糖的组成成分
  • 核糖：RNA的组成成分
  • 脱氧核糖：DNA的组成成分

二糖

二糖是由两个单糖分子通过糖苷键连接而成，常见的二糖包括：

  • 蔗糖：葡萄糖+果糖，存在于甘蔗、甜菜中
  • 麦芽糖：葡萄糖+葡萄糖，淀粉水解产物
  • 乳糖：葡萄糖+半乳糖，存在于乳汁中

多糖

多糖是由多个单糖分子通过糖苷键连接而成的高分子化合物，主要包括：

储能多糖

  • 淀粉：植物储能物质，由直链淀粉和支链淀粉组成
  • 糖原：动物储能物质，主要储存在肝脏和肌肉中

结构多糖

  • 纤维素：植物细胞壁的主要成分，人类不能消化
  • 几丁质：真菌细胞壁和昆虫外骨骼的成分
  • 果胶：植物细胞间质的成分

脂质——生命的储能物质

脂质的定义与特点

脂质是一类不溶于水而溶于有机溶剂（如乙醚、氯仿、苯等）的生物大分子，主要由碳、氢、氧三种元素组成，有些脂质还含有磷、氮等元素。

脂质的分类

根据结构和功能，脂质可分为以下几类：

  • 脂肪：甘油三酯，主要功能是储能和保温
  • 磷脂：构成细胞膜的主要成分
  • 固醇：如胆固醇，调节细胞膜的流动性
  • 萜类：如维生素A、E、K等

脂肪（甘油三酯）

结构特点

脂肪由一分子甘油和三分子脂肪酸通过酯键连接而成。根据脂肪酸的饱和程度，可分为饱和脂肪酸和不饱和脂肪酸。

物理性质

饱和脂肪酸的脂肪在室温下为固态（如动物脂肪），不饱和脂肪酸的脂肪在室温下为液态（如植物油）。

生理功能

  • 储能：脂肪是生物体最重要的储能物质，其储能效率是糖类的两倍多
  • 保温：皮下脂肪层具有保温作用
  • 缓冲：内脏周围的脂肪具有缓冲保护作用
  • 绝缘：神经纤维周围的脂肪具有绝缘作用

磷脂

结构特点

磷脂由甘油、脂肪酸和磷酸基团组成，磷酸基团上通常连接一个含氮碱基（如胆碱、乙醇胺等）。

分子结构

磷脂分子具有双亲性：脂肪酸链是疏水的，磷酸基团是亲水的。这种特性使磷脂在水环境中自发形成双层结构，是构成细胞膜的基础。

生理功能

  • 构成细胞膜：磷脂双分子层是细胞膜的基本骨架
  • 形成细胞器膜：线粒体膜、内质网膜等
  • 信号转导：某些磷脂代谢产物可作为信号分子

固醇

结构特点

固醇是一类具有四环三萜结构的脂质，包括胆固醇、植物固醇、麦角固醇等。

生理功能

  • 调节膜流动性：胆固醇插入磷脂双分子层，调节膜的流动性
  • 合成其他物质：胆固醇是合成胆汁酸、性激素、肾上腺皮质激素的前体
  • 维生素D的合成：皮肤中的7-脱氢胆固醇在紫外线照射下转化为维生素D₃

糖类和脂质的代谢联系

糖脂互变

糖类和脂质在代谢过程中可以相互转化：

  • 糖转化为脂肪：当糖类摄入过多时，可在肝脏中转化为脂肪储存
  • 脂肪转化为糖：脂肪分解产生的甘油可转化为葡萄糖，脂肪酸只能部分转化为葡萄糖

能量代谢

糖类和脂质都是重要的能源物质：

  • 糖类：快速供能物质，但储能效率低
  • 脂肪：长期储能物质，储能效率高
1克脂肪完全氧化可释放约38 kJ能量，而1克糖类完全氧化仅释放约17 kJ能量。
