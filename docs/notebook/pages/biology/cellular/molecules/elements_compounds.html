元素和化合物

细胞的元素组成

细胞由多种化学元素组成，这些元素按照含量可分为大量元素和微量元素两大类。

大量元素

大量元素是指在细胞中含量占细胞总重量万分之一以上的元素，包括：

  • C（碳）：构成有机物的基本骨架，约占细胞干重的50%
  • H（氢）：参与构成水和有机物，约占细胞干重的7%
  • O（氧）：构成水和有机物，约占细胞干重的20%
  • N（氮）：构成蛋白质和核酸，约占细胞干重的10%
  • P（磷）：构成核酸、磷脂和ATP，约占细胞干重的3%
  • S（硫）：构成某些氨基酸和维生素，约占细胞干重的1%
  • K（钾）、Ca（钙）、Mg（镁）、Na（钠）、Cl（氯）等：维持渗透压和酸碱平衡

微量元素

微量元素在细胞中含量极少，但对生命活动至关重要，包括：

  • Fe（铁）：构成血红蛋白、细胞色素
  • Cu（铜）：多种酶的辅因子
  • Zn（锌）：多种酶的辅因子，参与DNA合成
  • Mn（锰）：某些酶的激活剂
  • Mo（钼）：某些酶的辅因子
  • Co（钴）：维生素B12的组成成分
  • I（碘）：甲状腺激素的组成成分
  • Se（硒）：谷胱甘肽过氧化物酶的组成成分
微量元素虽然含量极少，但缺乏会导致严重的生理功能障碍。例如，铁缺乏会导致贫血，碘缺乏会导致甲状腺肿大。

细胞中的化合物

细胞中的化合物可分为无机化合物和有机化合物两大类。

无机化合物

主要包括水和无机盐：

  • 水：细胞中含量最多的化合物，约占细胞鲜重的70%-90%
  • 无机盐：以离子形式存在，如Na⁺、K⁺、Ca²⁺、Mg²⁺、Cl⁻、HCO₃⁻等

有机化合物

主要包括以下几类：

  • 糖类：提供能量和结构支持
  • 脂质：储能、构成膜结构
  • 蛋白质：生命活动的主要承担者
  • 核酸：遗传信息的携带者

生物大分子的组成特点

碳骨架

碳原子具有特殊的电子构型（最外层4个电子），可以与其他碳原子形成稳定的共价键，构成链状、环状或分支状的碳骨架。这种特性使碳成为构成有机物的基本元素。

官能团

在碳骨架上连接不同的官能团，可形成各种具有特定功能的有机化合物。常见的官能团包括：

  • 羟基（-OH）：存在于醇类和糖类中
  • 羧基（-COOH）：存在于有机酸中
  • 氨基（-NH₂）：存在于氨基酸中
  • 磷酸基（-PO₄）：存在于核酸和ATP中

生物大分子的连接方式

生物大分子通常由小分子（单体）通过脱水缩合反应连接而成，通过水解反应分解为单体。

脱水缩合

两个分子连接时，脱去一个水分子（H₂O），形成新的化学键（如肽键、糖苷键、磷酸二酯键）。

水解

在水分子参与下，生物大分子分解为小分子，是脱水缩合的逆反应。

细胞中元素和化合物的动态平衡

细胞内的元素和化合物处于动态平衡状态，这种平衡受到严格的调控：

  • 物质代谢：细胞不断进行物质的合成与分解
  • 能量代谢：物质的合成与分解伴随着能量的储存与释放
  • 信息调控：基因表达调控物质的合成与分解
  • 环境适应：细胞根据外界环境变化调整代谢活动
