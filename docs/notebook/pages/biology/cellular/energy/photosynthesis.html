光合作用与能量转化

光合作用的基本概念

光合作用是绿色植物、藻类和某些光合细菌利用光能将CO₂和H₂O转化为有机物，并释放O₂的过程。光合作用是地球上最重要的化学反应，为几乎所有生命提供能量和有机物质。

光合作用每年固定约1000亿吨CO₂，产生约7000亿吨有机物质，释放约5000亿吨O₂。

光合作用的反应式 

6CO₂ + 6H₂O + 光能 → C₆H₁₂O₆ + 6O₂

光合作用的两个阶段

光反应

发生部位

光反应发生在叶绿体类囊体膜上。

主要过程

  1. 光能吸收：叶绿素等光合色素吸收光能
  2. 水光解：水分子分解产生氧气、质子和电子
  3. 电子传递：电子通过电子传递链传递
  4. ATP合成：通过光合磷酸化合成ATP
  5. NADPH形成：NADP⁺还原为NADPH

产物

光反应产生ATP和NADPH，这些物质用于暗反应。

暗反应（卡尔文循环）

发生部位

暗反应发生在叶绿体基质中。

主要过程

  1. CO₂固定：CO₂与RuBP结合形成3-磷酸甘油酸
  2. 还原阶段：3-磷酸甘油酸还原为甘油醛-3-磷酸
  3. 再生阶段：RuBP再生，继续进行CO₂固定

产物

暗反应产生甘油醛-3-磷酸，可用于合成葡萄糖等有机物。

光合色素

叶绿素

叶绿素是光合作用的主要色素，包括：

  • 叶绿素a：主要色素，吸收红光（约660nm）
  • 叶绿素b：辅助色素，吸收蓝光（约640nm）

类胡萝卜素

类胡萝卜素是辅助色素，包括：

  • 胡萝卜素：橙色，吸收蓝绿光
  • 叶黄素：黄色，吸收蓝光

藻胆素

藻胆素存在于红藻和蓝藻中，包括：

  • 藻红蛋白：红色，吸收绿光
  • 藻蓝蛋白：蓝色，吸收橙红光

光合作用的单位

光系统

光系统是光合作用的基本单位，包括：

  • 光系统I（PSI）：吸收远红光（约700nm）
  • 光系统II（PSII）：吸收红光（约680nm）

反应中心

反应中心是光系统的核心部分，包括：

  • P700：PSI的反应中心
  • P680：PSII的反应中心

天线复合体

天线复合体由光合色素组成，负责捕获光能并将能量传递给反应中心。

光合电子传递链

Z方案

光合电子传递链按照Z方案进行：

  1. 光能激发PSII的P680，使其失去电子
  2. 电子通过电子传递链传递到PSI的P700
  3. 光能激发PSI的P700，使其失去电子
  4. 电子最终传递给NADP⁺，形成NADPH
  5. 水分子分解补充PSII失去的电子

光合磷酸化

光合磷酸化是指利用质子梯度合成ATP的过程：

  • 非循环光合磷酸化：产生ATP和NADPH
  • 循环光合磷酸化：只产生ATP，不产生NADPH

卡尔文循环

碳固定

卡尔文循环通过RuBisCO酶催化，将CO₂固定到RuBP（核酮糖-1,5-二磷酸）上，形成3-磷酸甘油酸。

还原阶段

3-磷酸甘油酸被ATP和NADPH还原为甘油醛-3-磷酸。

再生阶段

部分甘油醛-3-磷酸用于RuBP再生，继续进行CO₂固定。

产物

每固定6分子CO₂，产生1分子葡萄糖，消耗18分子ATP和12分子NADPH。

光合作用的调控

光调控

光通过以下机制调控光合作用：

  • 光强：影响光合速率
  • 光质：不同波长的光对光合作用的影响不同
  • 光周期：影响植物的光合作用节律

CO₂浓度调控

CO₂浓度是影响光合作用的重要因素：

  • CO₂饱和点：光合速率达到最大时的CO₂浓度
  • CO₂补偿点：光合速率与呼吸速率相等时的CO₂浓度

温度调控

温度影响光合作用相关酶的活性：

  • 最适温度：光合作用速率最高的温度
  • 温度范围：C3植物一般为15-25℃，C4植物一般为30-40℃

水分调控

水分通过影响气孔开度调控CO₂的吸收，进而影响光合作用。

C3、C4和CAM植物

C3植物

C3植物是最常见的植物类型，通过卡尔文循环直接固定CO₂。代表植物有小麦、水稻、大豆等。

C4植物

C4植物通过C4途径先固定CO₂，再通过卡尔文循环固定CO₂。代表植物有玉米、甘蔗、高粱等。

CAM植物

CAM植物在夜间固定CO₂，在白天进行卡尔文循环。代表植物有仙人掌、景天科植物等。

光合作用的生态意义

碳循环

光合作用是碳循环的关键环节，固定大气中的CO₂，减少温室效应。

氧循环

光合作用释放O₂，维持大气中的氧气含量，支持需氧生物的生存。

食物链基础

光合作用产生的有机物是食物链的基础，为几乎所有生物提供能量和物质基础。

气候调节

植物通过光合作用吸收CO₂，调节全球气候，减缓全球变暖。

光合作用的应用

农业应用

  • 作物育种：选育高光合效率的作物品种
  • 栽培技术：优化光照、CO₂浓度、温度等条件
  • 温室种植：人工控制环境条件，提高光合效率

生物技术应用

  • 人工光合作用：模拟光合作用过程，开发新能源技术
  • 基因工程：改造植物光合系统，提高光合效率
  • 生物燃料：利用光合作用生产生物燃料

环境应用

  • 碳汇：通过植树造林增加碳汇，减少大气CO₂
  • 生态修复：利用植物光合作用修复退化生态系统
  • 环境监测：通过监测植物光合作用评估环境质量
