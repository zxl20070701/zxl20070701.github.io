生产函数：短期与长期

生产函数的定义

生产函数表示在给定技术条件下，生产要素投入量与最大产出量之间的技术关系。它反映了企业的技术约束。

一般形式：Q = f(L, K, M, ...)

其中Q表示产出量，L表示劳动投入，K表示资本投入，M表示原材料等其他投入。

短期生产函数

短期是指至少有一种生产要素固定不变的时间期间。在短期分析中，通常假设资本K固定，劳动L可变。

总产量(TP)

总产量是指在给定固定要素投入下，可变要素投入所能生产的最大产出量。

TP = f(L, K̄)

平均产量(AP)

平均产量是指每单位可变要素投入的平均产出量。

APₗ = TP/L

边际产量(MP)

边际产量是指在其他要素投入不变的情况下，增加一单位可变要素投入所带来的额外产出量。

MPₗ = dTP/dL

长期生产函数

长期是指所有生产要素都可以调整的时间期间。在长期中，企业可以调整所有投入要素的数量。

等产量线

等产量线表示能够生产相同产出水平的两种要素投入的各种组合的轨迹。

生产函数的齐次性

如果生产函数满足f(tL, tK) = tⁿf(L, K)，则称其为n次齐次函数。

  • n > 1：规模报酬递增
  • n = 1：规模报酬不变
  • n < 1：规模报酬递减

常见的生产函数形式

柯布-道格拉斯生产函数

Q = AL^αK^β

特点：

  • 要素替代弹性恒为1
  • α和β分别表示劳动和资本的产出弹性
  • α + β表示规模报酬性质

固定比例生产函数

Q = min{L/a, K/b}

特点：

  • 要素之间完全互补
  • 要素替代弹性为0

CES生产函数

Q = A[αL^ρ + (1-α)K^ρ]^(1/ρ)

特点：

  • 要素替代弹性为1/(1-ρ)
  • 当ρ→0时，趋近于柯布-道格拉斯函数
  • 当ρ→-∞时，趋近于固定比例函数

生产函数的性质

单调性

如果要素投入增加，产出不会减少，即∂f/∂L ≥ 0，∂f/∂K ≥ 0。

拟凹性

生产函数的等产量线凸向原点，反映了边际技术替代率递减规律。

可加性

f(L₁+L₂, K₁+K₂) ≥ f(L₁, K₁) + f(L₂, K₂)，表示生产具有规模经济效应。

技术进步的影响

中性技术进步

技术进步使所有要素的边际产量同比例增加。

劳动节约型技术进步

技术进步使资本的边际产量相对增加更多。

资本节约型技术进步

技术进步使劳动的边际产量相对增加更多。

生产函数的估计

参数估计方法

常用的生产函数参数估计方法包括：

  • 最小二乘法：对线性化后的生产函数进行回归
  • 极大似然法：基于生产函数的统计分布进行估计
  • 非参数方法：如数据包络分析(DEA)

数据问题

生产函数估计面临的主要问题包括：

  • 测量误差：产出和投入数据的准确性
  • 同时性偏差：投入决策与随机冲击的相关性
  • 选择性偏差：企业进入和退出市场的影响
