短期成本曲线

短期成本的分类

在短期分析中，企业的成本可以分为固定成本和可变成本两大类。

总成本概念

总固定成本(TFC)

总固定成本是指不随产量变化而变化的成本，即使产量为零也必须支付。在短期内，这些成本是固定的。

  • 特征：与产量无关，在短期内固定不变
  • 例子：厂房租金、设备折旧、管理人员工资、保险费

总可变成本(TVC)

总可变成本是指随产量变化而变化的成本，当产量为零时，可变成本也为零。

  • 特征：与产量正相关，随产量增加而增加
  • 例子：原材料成本、直接劳动成本、能源费用

总成本(TC)

总成本是总固定成本与总可变成本之和。

TC = TFC + TVC

总成本曲线与总可变成本曲线平行，两者之间的垂直距离等于总固定成本。

平均成本概念

平均固定成本(AFC)

平均固定成本是总固定成本除以产量。

AFC = TFC/Q

平均固定成本曲线是一条向右下方倾斜的曲线，随着产量增加而递减。

平均可变成本(AVC)

平均可变成本是总可变成本除以产量。

AVC = TVC/Q

平均可变成本曲线通常呈U形，先下降后上升。

平均总成本(AC)

平均总成本是总成本除以产量，也是平均固定成本与平均可变成本之和。

AC = TC/Q = AFC + AVC

平均总成本曲线也呈U形，位于平均可变成本曲线之上，两者之间的垂直距离等于平均固定成本。

边际成本(MC)

定义

边际成本是指增加一单位产量所引起的总成本的增量。

MC = dTC/dQ = dTVC/dQ

边际成本只与可变成本有关，因为固定成本在短期内不变。

边际成本曲线的特征

边际成本曲线通常呈U形，反映了边际报酬递减规律的影响。

短期成本曲线之间的关系

边际成本与平均成本的关系

  • 当MC < AC时，平均成本递减
  • 当MC > AC时，平均成本递增
  • 当MC = AC时，平均成本达到最小值

边际成本与平均可变成本的关系

  • 当MC < AVC时，平均可变成本递减
  • 当MC > AVC时，平均可变成本递增
  • 当MC = AVC时，平均可变成本达到最小值

成本曲线的几何关系

边际成本曲线分别通过平均可变成本曲线和平均总成本曲线的最低点。

成本曲线的推导

从生产函数推导成本函数

给定生产函数Q = f(L, K̄)和要素价格w、r，可以通过成本最小化问题推导出成本函数。

成本函数的性质

  • 非负性：成本不能为负
  • 单调性：成本随产量增加而增加
  • 连续性：成本函数通常是连续的
  • 凸性：边际成本通常递增

成本曲线的移动

要素价格变化的影响

当要素价格变化时，成本曲线会发生移动：

  • 工资率上升：可变成本曲线向上移动，固定成本曲线不变
  • 利率上升：固定成本曲线向上移动，可变成本曲线不变

技术进步的影响

技术进步会使成本曲线向下移动，提高生产效率。

成本曲线的实证应用

企业供给决策

企业的短期供给曲线是其边际成本曲线在平均可变成本最低点以上的部分。

停产决策

当价格低于平均可变成本最低点时，企业会选择停产。

盈亏平衡分析

通过成本曲线可以确定企业的盈亏平衡点。

成本曲线的弹性分析

成本弹性

成本弹性衡量成本对产量变化的敏感程度。

E_c = (dC/C)/(dQ/Q)

规模弹性

规模弹性是成本弹性的倒数，反映规模经济程度。

多产品企业的成本分析

联合成本

多产品企业可能面临联合成本，需要在不同产品间进行成本分配。

范围经济

多产品企业可能享受范围经济，即同时生产多种产品的成本低于分别生产的成本。

成本函数的估计

计量经济学方法

使用统计方法估计成本函数，常用的函数形式包括线性函数、二次函数、三次函数等。

数据要求

成本函数估计需要准确的成本数据和产出数据，以及相关的要素价格数据。

估计问题

  • 测量误差：成本数据的准确性问题
  • 同时性偏差：产出决策与成本之间的相互影响
  • 选择性偏差：企业进入和退出市场的影响
