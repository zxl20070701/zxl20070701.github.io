铁 (Fe)

基本性质

铁是元素周期表第26号元素，原子序数26，原子量55.85，电子构型[Ar]3d⁶4s²。作为第一过渡系的代表元素，铁具有多种氧化态(-2到+6)，其中+2和+3价最稳定。铁是重要的铁磁性材料，居里温度768°C，在工业和生物系统中具有核心地位。

原子结构特征

铁原子的3d轨道部分填充，导致其具有多种氧化态和丰富的配位化学。第一、第二、第三电离能分别为762.5、1561.9和2957.4 kJ/mol。3d电子参与成键，形成复杂的配位化合物，这是过渡金属的特征。

同素异形体

铁具有多种晶体结构：α-Fe(体心立方，<912°C)、γ-Fe(面心立方，912-1394°C)、δ-Fe(体心立方，>1394°C)。这些相变对钢的热处理具有重要意义。α-Fe是铁磁性的，而γ-Fe是顺磁性的。

化学反应性

氧化还原特性

铁的标准电极电势：Fe²⁺/Fe = -0.44 V，Fe³⁺/Fe²⁺ = +0.77 V。铁易失去电子被氧化，在潮湿空气中形成铁锈：4Fe + 3O₂ + 6H₂O → 4Fe(OH)₃。铁锈是复杂的水合氧化铁，结构松散，不能提供保护。

与非金属的反应

铁与氧气反应生成多种氧化物：FeO、Fe₂O₃、Fe₃O₄。与硫反应生成FeS，与卤素反应生成FeX₂或FeX₃。铁与碳形成重要的合金系统，碳含量决定钢和铸铁的性能差异。

配位化学

铁形成多种配位化合物，如[Fe(CN)₆]⁴⁻、[Fe(CN)₆]³⁻、[Fe(H₂O)₆]²⁺等。这些配合物的颜色、磁性和反应活性与其电子构型密切相关。铁配合物在催化、分析和生物系统中发挥重要作用。

重要化合物与转化

氧化物体系

氧化亚铁(FeO)是碱性氧化物，具有NaCl型结构，但在<570°C时不稳定。氧化铁(Fe₂O₃)存在α和γ两种晶型，α-Fe₂O₃是赤铁矿的主要成分。四氧化三铁(Fe₃O₄)是混合价态氧化物(Fe²⁺Fe³⁺₂O₄)，具有反尖晶石结构，是磁铁矿的主要成分。

盐类化合物

硫酸亚铁(FeSO₄·7H₂O)俗称绿矾，是重要的还原剂和营养补充剂。氯化铁(FeCl₃)是强Lewis酸，在有机合成中用作催化剂。铁氰化钾(K₄[Fe(CN)₆])和铁氰化钾(K₃[Fe(CN)₆])是重要分析试剂。

羰基化合物

五羰基铁(Fe(CO)₅)是重要有机金属化合物，由铁粉与CO在200°C、200 atm下反应制得。Fe(CO)₅是挥发性液体，可用作抗爆剂和制备纯铁。十二羰基三铁(Fe₃(CO)₁₂)具有金属-金属键，是簇合物化学的重要研究对象。

冶金与材料科学

钢铁冶炼

高炉炼铁：Fe₂O₃ + 3CO → 2Fe + 3CO₂，使用焦炭作为还原剂和热源。转炉炼钢通过氧化去除生铁中的碳、硅、锰、磷、硫等杂质。电弧炉炼钢利用电能熔化废钢，是现代钢铁工业的重要组成部分。

钢的热处理

退火处理消除内应力，改善塑性和韧性。淬火处理获得马氏体组织，提高硬度和强度。回火处理调整淬火钢的硬度和韧性平衡。表面硬化如渗碳、氮化处理提高表面耐磨性而保持芯部韧性。

不锈钢与合金钢

不锈钢含铬>10.5%，形成Cr₂O₃保护膜。奥氏体不锈钢(如304)含18%Cr-8%Ni，具有优异耐腐蚀性。马氏体不锈钢可通过热处理强化。工具钢含钨、钼、钒等合金元素，具有高硬度和红硬性。

生物无机化学

血红蛋白与肌红蛋白

血红蛋白(Hb)是四聚体蛋白，每个亚基含一个血红素铁。Fe²⁺离子与卟啉环配位，形成八面体结构，第六配位位点可逆结合O₂。氧合血红蛋白(HbO₂)与脱氧血红蛋白(Hb)的结构差异解释了协同效应。

铁硫蛋白

铁硫簇([Fe-S])是重要电子传递体，存在于线粒体呼吸链和光合作用中。最简单的[2Fe-2S]簇含有两个铁离子通过硫原子桥联。铁硫蛋白在氧化还原反应中通过Fe²⁺/Fe³⁺的相互转化传递电子。

铁代谢与疾病

人体含铁约4-5 g，主要存在于血红蛋白中。铁的吸收受铁调素(hepcidin)调控。铁过载导致血色病，铁缺乏引起贫血。铁离子通过Fenton反应催化活性氧产生：Fe²⁺ + H₂O₂ → Fe³⁺ + OH⁻ + ·OH，与神经退行性疾病相关。
