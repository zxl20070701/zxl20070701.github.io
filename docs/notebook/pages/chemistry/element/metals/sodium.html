钠 (Na)

基本性质

钠是元素周期表第11号元素，原子序数11，原子量22.99，电子构型[Ne]3s¹。作为碱金属族的代表元素，钠具有典型的金属特性：银白色光泽、良好的导电导热性、较低的密度(0.97 g/cm³)和熔点(97.8°C)。

原子结构特征

钠原子最外层仅有一个3s电子，这个价电子与原子核的结合能较弱（第一电离能495.8 kJ/mol），导致钠极易失去电子形成Na⁺离子。这种电子构型决定了钠的强还原性和活泼的化学性质。

物理性质

钠晶体为体心立方结构，配位数8。由于金属键较弱，钠的硬度低（莫氏硬度0.5），可用刀切割。液态钠具有独特的物理性质：在98-883°C范围内为液态，这一特性使其成为核反应堆冷却剂的理想选择。

化学反应性

与水的反应

钠与水发生剧烈反应：2Na + 2H₂O → 2NaOH + H₂↑。反应放热剧烈，可使钠熔化成小球并在水面上快速移动。反应生成的氢气可能自燃，产生黄色火焰。反应机理涉及电子从钠原子转移到水分子，形成Na⁺和OH⁻离子。

与非金属的反应

钠与卤素反应剧烈：2Na + Cl₂ → 2NaCl，反应放出大量热并发出明亮黄光。与氧气的反应产物取决于条件：常温下生成氧化钠(Na₂O)，过量氧气中燃烧生成过氧化钠(Na₂O₂)，高压氧气中可生成超氧化钠(NaO₂)。

与酸的反应

钠与酸反应比与水反应更剧烈：2Na + 2HCl → 2NaCl + H₂↑。由于酸中H⁺浓度更高，反应速率显著加快。在非氧化性酸中，钠主要作为还原剂，将H⁺还原为H₂。

化合物与转化

氧化物与氢氧化物

氧化钠(Na₂O)是白色固体，离子晶体结构，与水反应生成氢氧化钠：Na₂O + H₂O → 2NaOH。氢氧化钠(NaOH)是强碱，完全电离，具有强腐蚀性。工业上通过电解饱和食盐水制备：2NaCl + 2H₂O → 2NaOH + Cl₂↑ + H₂↑。

盐类化合物

钠盐普遍具有良好的水溶性，这是钠离子水合能高(-405 kJ/mol)的结果。氯化钠(NaCl)是最重要的钠盐，立方晶系结构，每个Na⁺被6个Cl⁻八面体配位。碳酸钠(Na₂CO₃)俗称纯碱，工业上通过索尔维法制备。

有机金属化合物

钠与醇反应生成醇钠：2Na + 2ROH → 2RONa + H₂↑。醇钠是强碱，在有机合成中用作碱催化剂。钠汞齐(Na-Hg)是温和的还原剂，可选择性还原羰基化合物而不影响碳碳双键。

工业应用与制备

金属钠的生产

工业上通过电解熔融氯化钠制备金属钠：2NaCl(l) → 2Na(l) + Cl₂(g)。采用唐斯电解槽，以石墨为阳极，铁为阴极，在约600°C下操作。为降低熔点，常加入氯化钙形成共熔体。

应用领域

金属钠主要用作还原剂制备钛、锆等金属：TiCl₄ + 4Na → Ti + 4NaCl。在有机合成中，钠用于制备醇钠、金属有机化合物等。钠蒸气灯发出特征黄光(589 nm)，广泛应用于路灯照明。

生物与环境化学

钠离子是生物体内重要的电解质，参与神经冲动传导、肌肉收缩等生理过程。人体每日需摄入约2-3 g钠离子，主要通过食盐摄取。钠离子与钾离子的跨膜浓度梯度是细胞膜电位形成的基础。
