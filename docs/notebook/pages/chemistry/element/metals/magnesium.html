镁 (Mg)

基本性质

镁是元素周期表第12号元素，原子序数12，原子量24.31，电子构型[Ne]3s²。作为碱土金属族的代表元素，镁具有典型的金属特性：银白色光泽、中等密度(1.74 g/cm³)、熔点650°C。镁是地壳中第八丰富的元素，主要存在于菱镁矿(MgCO₃)和白云石(CaMg(CO₃)₂)中。

原子结构特征

镁原子最外层有两个3s电子，第二电离能(1450 kJ/mol)明显高于第一电离能(737 kJ/mol)，但两者都低于同周期的钠。这种电子构型使镁倾向于形成+2价离子，具有较强的还原性，但活性低于碱金属。

晶体结构

镁晶体为密排六方结构(hcp)，配位数12，这种结构赋予了镁良好的力学性能。镁的晶格参数a=3.21 Å，c=5.21 Å，c/a比值为1.62，接近理想密排值1.63。

化学反应性

与水的反应

镁与冷水反应缓慢：Mg + 2H₂O → Mg(OH)₂ + H₂↑，生成的氢氧化镁形成保护层阻止反应进一步进行。与热水反应较快：Mg + 2H₂O → MgO + H₂↑。镁与水蒸气的反应在高温下显著加速。

与非金属的反应

镁与氧气反应生成氧化镁：2Mg + O₂ → 2MgO，反应发出耀眼白光，这是镁的重要特征反应。镁与氮气在高温下反应生成氮化镁：3Mg + N₂ → Mg₃N₂。与卤素反应剧烈：Mg + Cl₂ → MgCl₂。

与酸的反应

镁与非氧化性酸反应放出氢气：Mg + 2HCl → MgCl₂ + H₂↑。与氧化性酸反应复杂，如与浓硝酸反应生成硝酸镁、氮氧化物和水：Mg + 4HNO₃ → Mg(NO₃)₂ + 2NO₂ + 2H₂O。

化合物与转化

氧化物与氢氧化物

氧化镁(MgO)是白色离子晶体，熔点2852°C，具有NaCl型结构。MgO是碱性氧化物，与水缓慢反应生成氢氧化镁：MgO + H₂O → Mg(OH)₂。氢氧化镁是弱碱，Ksp=1.8×10⁻¹¹，在医药上用作抗酸剂。

卤化物

氯化镁(MgCl₂)是重要镁盐，易潮解，可形成多种水合物。无水氯化镁通过熔融电解制备金属镁：MgCl₂(l) → Mg(l) + Cl₂(g)。氟化镁(MgF₂)是难溶盐，用于光学镀膜材料。

碳酸盐与硫酸盐

碳酸镁(MgCO₃)是白色粉末，加热分解：MgCO₃ → MgO + CO₂↑。硫酸镁(MgSO₄)俗称泻盐，易溶于水，形成七水合物MgSO₄·7H₂O，医学上用作泻药和抗惊厥剂。

金属镁的生产与应用

工业制备方法

金属镁主要通过电解法和热还原法生产。电解法使用熔融氯化镁，在约750°C下电解。热还原法使用硅铁还原氧化镁：2MgO + Si → 2Mg + SiO₂，在1200°C下于真空炉中进行。

合金应用

镁合金具有密度低、比强度高、减震性能好等优点。AZ91D(Mg-9Al-1Zn)是最常用的镁合金，用于汽车零部件、航空航天结构件。镁铝合金具有优异的电磁屏蔽性能，用于电子设备外壳。

有机镁化合物

格氏试剂(RMgX)是最重要的有机镁化合物，由镁与卤代烷在无水乙醚中反应制得：R-X + Mg → RMgX。格氏试剂是强亲核试剂，可与羰基化合物反应制备醇类，在有机合成中具有重要地位。

生物化学功能

镁离子是生物体内第四丰富的阳离子，参与300多种酶促反应。作为ATP的辅因子，镁离子稳定ATP的负电荷，促进磷酸基团转移。叶绿素分子中心的镁离子是光合作用的关键组分。镁离子还参与蛋白质合成、DNA复制和神经传导等生理过程。
