铝 (Al)

基本性质

铝是元素周期表第13号元素，原子序数13，原子量26.98，电子构型[Ne]3s²3p¹。作为第三主族的代表元素，铝具有独特的两性性质：既表现金属性又表现非金属性。铝密度低(2.70 g/cm³)、导电性好、耐腐蚀，是现代工业中最重要的结构材料之一。

原子结构特征

铝原子最外层有三个电子(3s²3p¹)，电离能分别为577.5、1816.7和2744.8 kJ/mol。这种电子构型使铝倾向于形成+3价离子，但由于高电荷密度，Al³⁺在水溶液中强烈水解，形成[Al(H₂O)₆]³⁺等配合物。

晶体结构

铝晶体为面心立方结构(fcc)，配位数12，晶格参数a=4.05 Å。这种密堆积结构赋予了铝良好的延展性和导电性。铝的原子半径为143 pm，离子半径(Al³⁺)为53 pm，高电荷小半径导致强烈的极化能力。

化学反应性

钝化现象

铝在空气中迅速形成致密氧化膜(Al₂O₃)，厚度约5-10 nm，这层氧化膜具有极高的稳定性，阻止了铝的进一步氧化。这种自钝化特性是铝具有优异耐腐蚀性的根本原因。氧化膜在特定条件下可被破坏，如汞齐化后铝会迅速氧化。

与酸和碱的反应

铝与盐酸反应：2Al + 6HCl → 2AlCl₃ + 3H₂↑。与氧化性酸如浓硝酸反应时，铝发生钝化而不溶解。铝与强碱反应表现出两性：2Al + 2OH⁻ + 6H₂O → 2[Al(OH)₄]⁻ + 3H₂↑，生成四羟基合铝酸盐。

与非金属的反应

铝与氧气在高温下反应：4Al + 3O₂ → 2Al₂O₃，反应放出大量热(ΔH=-1675 kJ/mol)，这一性质被用于铝热反应。铝与卤素反应剧烈：2Al + 3Cl₂ → 2AlCl₃。铝与氮气在高温下反应生成氮化铝：2Al + N₂ → 2AlN。

化合物与转化

氧化物与氢氧化物

氧化铝(Al₂O₃)具有多种晶型，α-Al₂O₃(刚玉)是最稳定的形式，硬度仅次于金刚石。γ-Al₂O₃具有多孔结构，比表面积大，是重要的催化剂载体。氢氧化铝Al(OH)₃是两性氢氧化物，可溶于酸和强碱：Al(OH)₃ + OH⁻ → [Al(OH)₄]⁻。

卤化物

氯化铝(AlCl₃)是重要Lewis酸，在Friedel-Crafts反应中作催化剂。无水AlCl₃为共价化合物，形成二聚体Al₂Cl₆，具有氯桥结构。氟化铝(AlF₃)是离子晶体，用于铝电解生产中的电解质组分。

铝酸盐与配合物

铝酸盐如Na[Al(OH)₄]和NaAlO₂是重要的工业中间体。铝离子易形成配合物，如[AlF₆]³⁻、Al(acac)₃等。这些配合物在分析化学和材料科学中具有重要应用。

工业生产与应用

霍尔-埃鲁法

工业上通过霍尔-埃鲁法电解制备铝：2Al₂O₃ + 3C → 4Al + 3CO₂。电解槽使用冰晶石(Na₃AlF₆)作为助熔剂，降低Al₂O₃的熔点至约950°C。电解过程消耗大量电能，每吨铝需约15,000 kWh。

铝合金

铝合金按系列分类：1xxx(纯铝)、2xxx(Al-Cu)、3xxx(Al-Mn)、4xxx(Al-Si)、5xxx(Al-Mg)、6xxx(Al-Mg-Si)、7xxx(Al-Zn-Mg-Cu)。2024合金(Al-Cu-Mg)用于航空结构件，7075合金具有超高强度，用于飞机机身。

铝热反应

铝热反应利用铝的强还原性：2Al + Fe₂O₃ → 2Fe + Al₂O₃，反应温度可达2500°C，用于焊接铁轨、提取高熔点金属如铬、锰等。反应机理涉及铝的氧化和金属氧化物的还原。

表面工程与腐蚀防护

阳极氧化

阳极氧化是在电解条件下使铝表面形成致密氧化膜的过程。硫酸阳极氧化可形成多孔氧化膜，厚度5-25 μm，可进行染色处理。硬质阳极氧化膜厚度可达100 μm，硬度接近刚玉。

化学转化膜

铬酸盐处理形成黄色到彩虹色的转化膜，提供良好的腐蚀防护和涂装基底。无铬转化处理如钛锆处理，是环保替代方案。磷铬酸盐处理用于航空工业。
