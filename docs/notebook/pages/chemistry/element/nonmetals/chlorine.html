氯 (Cl)

基本性质

氯是元素周期表第17号元素，原子序数17，原子量35.45，电子构型[Ne]3s²3p⁵。作为卤素的代表元素，氯具有极强的电负性(3.16)和氧化性。氯气(Cl₂)是黄绿色气体，具有刺激性气味和强毒性。氯是地壳中第21丰富的元素，主要以氯化钠形式存在于海水和盐矿中。

原子结构特征

氯原子价电子构型为3s²3p⁵，仅差一个电子达到稳定八隅体结构，因此具有强氧化性。氯的第一电子亲和能(-349 kJ/mol)很高，易获得电子形成Cl⁻离子。氯的价层有空的3d轨道，可扩展八隅体形成高价化合物。

分子氯的特性

Cl₂分子键长1.99 Å，键能242 kJ/mol，较F-F键(158 kJ/mol)强但较Br-Br键(193 kJ/mol)弱。氯气在水中的溶解度较高(20°C时约7 g/L)，部分水解：Cl₂ + H₂O ⇌ HCl + HOCl。氯气易被活性炭吸附，这一性质用于防毒面具。

单质氯的化学性质

氧化性反应

氯气是强氧化剂，标准电极电势Cl₂/Cl⁻ = +1.36 V。氯气与金属反应生成氯化物：2Fe + 3Cl₂ → 2FeCl₃。与非金属反应如：H₂ + Cl₂ → 2HCl，反应剧烈并爆炸。氯气可氧化Br⁻和I⁻：Cl₂ + 2Br⁻ → 2Cl⁻ + Br₂。

与水的反应

氯气溶于水发生歧化反应：Cl₂ + H₂O ⇌ HCl + HOCl，平衡常数K=4×10⁻⁴。次氯酸(HOCl)是弱酸，Ka=3×10⁻⁸，具有强氧化性，是漂白和消毒的活性成分。光照下HOCl分解：2HOCl → 2HCl + O₂。

与碱的反应

氯气与冷稀碱反应生成次氯酸盐：Cl₂ + 2OH⁻ → Cl⁻ + ClO⁻ + H₂O。与热浓碱反应生成氯酸盐：3Cl₂ + 6OH⁻ → 5Cl⁻ + ClO₃⁻ + 3H₂O。工业上通过电解食盐水制备氯气：2NaCl + 2H₂O → 2NaOH + Cl₂ + H₂。

氯化物化学

离子氯化物

碱金属和碱土金属氯化物多为离子晶体，如NaCl、KCl、CaCl₂。NaCl具有面心立方结构，每个Na⁺被6个Cl⁻八面体配位。氯化物溶解度规律：多数氯化物易溶，但AgCl、PbCl₂、Hg₂Cl₂难溶。

共价氯化物

非金属氯化物多为共价化合物，如CCl₄、SiCl₄、PCl₅。这些化合物易水解：SiCl₄ + 4H₂O → Si(OH)₄ + 4HCl。过渡金属氯化物如FeCl₃、AlCl₃具有共价性，可形成二聚体如Al₂Cl₆。

配位氯化物

氯离子是良好配体，可形成多种配合物，如[CoCl₄]²⁻、[PtCl₆]²⁻。氯配合物的稳定性遵循软硬酸碱理论：软酸金属如Hg²⁺、Pt²⁺形成稳定氯配合物。氯桥配合物如[Cu₂Cl₆]²⁻含有氯桥键。

含氧酸及其盐

次氯酸及其盐

次氯酸钠(NaClO)是重要漂白剂和消毒剂，通过氯气与NaOH反应制备。次氯酸钙(Ca(ClO)₂)是漂白粉的有效成分。次氯酸盐在酸性条件下释放Cl₂：ClO⁻ + Cl⁻ + 2H⁺ → Cl₂ + H₂O。

氯酸及其盐

氯酸钾(KClO₃)是强氧化剂，加热分解：2KClO₃ → 2KCl + 3O₂。氯酸钠(NaClO₃)用于制备二氧化氯(ClO₂)，是高效漂白剂。氯酸盐与有机物混合可形成爆炸性混合物。

高氯酸及其盐

高氯酸(HClO₄)是最强的无机酸，浓溶液具有强氧化性。高氯酸盐如NH₄ClO₄用作火箭推进剂。高氯酸根(ClO₄⁻)是弱配体，但可稳定高价金属离子如[Fe(H₂O)₅(ClO₄)]²⁺。

有机氯化学

氯代烃合成

自由基氯化：CH₄ + Cl₂ → CH₃Cl + HCl，通过链式反应机理进行。亲电氯化使用Lewis酸催化剂如FeCl₃：C₆H₆ + Cl₂ → C₆H₅Cl + HCl。氯化试剂如SOCl₂、PCl₅用于醇的氯化：ROH + SOCl₂ → RCl + SO₂ + HCl。

氯代烃反应

氯代烃可发生亲核取代(SN1/SN2)和消除(E1/E2)反应。格氏试剂制备：RCl + Mg → RMgCl。武兹反应：2RCl + 2Na → R-R + 2NaCl。氯代芳烃可通过亲核芳香取代反应(SNAr)进行官能团转化。

重要有机氯化合物

四氯化碳(CCl₄)曾用作灭火剂和干洗剂，因毒性已限制使用。氯仿(CHCl₃)是重要溶剂和合成中间体。DDT是有效杀虫剂，因环境持久性已被禁用。聚氯乙烯(PVC)是重要塑料，由氯乙烯聚合制备。

氯的配位与催化化学

金属氯配合物

氯离子是重要配体，可形成从简单[CoCl₄]²⁻到复杂[Re₂Cl₈]²⁻的各种配合物。金属氯配合物的颜色与d-d跃迁有关，如[CuCl₄]²⁻黄色、[CoCl₄]²⁻蓝色。氯桥配合物如[Cu₂Cl₆]²⁻含有μ₂-Cl桥键。

氯在催化中的作用

氯化钯(PdCl₂)是重要催化剂，用于瓦克过程：C₂H₄ + PdCl₂ + H₂O → CH₃CHO + Pd + 2HCl。氯化铜(CuCl₂)催化乙炔水合：C₂H₂ + H₂O → CH₃CHO。齐格勒-纳塔催化剂使用TiCl₄/AlEt₃催化烯烃聚合。

氯配体的取代反应

金属氯配合物中的Cl⁻可被其他配体取代，反应速率与金属中心和配体性质有关。反位效应序列：CN⁻ ~ CO > NO₂⁻ > I⁻ > Br⁻ > Cl⁻ > NH₃ > OH⁻ > H₂O。这一规律在配合物合成中具有指导意义。

氯的环境与生物化学

氯在生物体中的作用

氯离子是细胞外液的主要阴离子，维持渗透压和酸碱平衡。胃酸中的HCl有助于食物消化和杀菌。髓过氧化物酶催化Cl⁻氧化产生次氯酸，参与免疫反应。氯离子通道蛋白调控细胞膜电位。

氯代有机物的环境影响

氯氟烃(CFC)破坏臭氧层：CFCl₃ + hν → CFCl₂ + Cl·，Cl·催化O₃分解。持久性有机氯污染物如二噁英、多氯联苯(PCB)在食物链中富集。水处理中产生的消毒副产物如三卤甲烷(THM)具有潜在致癌性。

氯的毒性与防护

氯气LC₅₀(大鼠吸入10分钟)为293 ppm，刺激呼吸道并导致肺水肿。氯气中毒急救需立即脱离现场，给予氧气治疗。氯气检测可使用碘化钾淀粉试纸变蓝。防护措施包括佩戴防毒面具和化学防护服。
