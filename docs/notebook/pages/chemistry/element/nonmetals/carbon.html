碳 (C)

基本性质

碳是元素周期表第6号元素，原子序数6，原子量12.01，电子构型[He]2s²2p²。作为第四主族的头号元素，碳具有独特的成键能力，可形成稳定的单键、双键和三键。碳是构成有机化合物的基本元素，其同素异形体包括金刚石、石墨、富勒烯、碳纳米管和石墨烯等。

原子结构特征

碳原子的价电子构型为2s²2p²，可形成sp³、sp²、sp三种杂化轨道，分别对应四面体、平面三角形和直线型几何构型。碳的电负性(2.55)适中，既能失去电子形成CO、CO₂，又能获得电子形成CH₄等化合物。

同素异形体

金刚石是三维网络结构，每个碳原子以sp³杂化与四个相邻碳原子形成σ键，形成极其坚硬的晶体。石墨是层状结构，每个碳原子以sp²杂化形成平面六元环，层间通过范德华力结合。石墨烯是单层石墨，具有优异的电学和力学性能。

无机碳化学

碳氧化物

一氧化碳(CO)是线性分子，C≡O三键键能1072 kJ/mol，是重要的还原剂和配体。二氧化碳(CO₂)是直线型分子，具有Lewis酸碱双重性，可与水反应生成碳酸。干冰是固态CO₂，在-78.5°C升华。

碳酸盐与碳酸氢盐

碳酸钙(CaCO₃)存在方解石和文石两种晶型，是石灰石和大理石的主要成分。碳酸氢钠(NaHCO₃)俗称小苏打，是两性物质。碳酸盐的热稳定性随金属离子半径增大而增加：BeCO₃ < MgCO₃ < CaCO₃ < SrCO₃ < BaCO₃。

碳化物

离子型碳化物如CaC₂含有C₂²⁻离子，水解生成乙炔：CaC₂ + 2H₂O → Ca(OH)₂ + C₂H₂↑。共价型碳化物如SiC具有金刚石结构，是重要磨料和高温材料。间充型碳化物如Fe₃C存在于钢中，影响钢的硬度和脆性。

有机碳化学基础

烃类化合物

烷烃(CnH₂n₊₂)是饱和烃，仅含C-C和C-H单键。烯烃(CnH₂n)含C=C双键，可发生加成反应。炔烃(CnH₂n₋₂)含C≡C三键，具有酸性氢。芳香烃如苯具有特殊的π电子离域结构，稳定性高。

官能团化学

羟基(-OH)形成醇和酚，可发生氧化、酯化等反应。羰基(C=O)形成醛和酮，是重要反应中心。羧基(-COOH)形成羧酸，具有酸性和酯化活性。氨基(-NH₂)形成胺，具有碱性和亲核性。

反应机理

亲核取代反应(SN1/SN2)涉及碳-杂原子键的形成与断裂。亲电加成反应(如烯烃与HX)遵循马氏规则。消除反应(E1/E2)生成烯烃。自由基反应涉及碳自由基中间体，如烷烃的卤化反应。

碳的同素异形体化学

富勒烯化学

C₆₀是截角二十面体结构，含有12个五元环和20个六元环，具有芳香性。富勒烯可作为电子受体，形成电荷转移复合物。金属富勒烯(M@C₆₀)将金属原子封装在碳笼内，具有独特性质。

碳纳米管

单壁碳纳米管(SWCNT)可视为石墨烯卷曲形成的圆柱体，具有手性矢量(n,m)表征。碳纳米管具有优异的力学强度和杨氏模量，是理想的纳米材料。功能化修饰可改善其溶解性和生物相容性。

石墨烯化学

石墨烯是单层sp²杂化碳原子组成的二维材料，具有线性色散关系，载流子表现为无质量狄拉克费米子。石墨烯可通过机械剥离、化学气相沉积等方法制备，在电子器件、传感器等领域有广泛应用。

生物碳化学

生物大分子

蛋白质由氨基酸通过肽键连接，具有四级结构。核酸(DNA/RNA)由核苷酸通过磷酸二酯键连接，携带遗传信息。多糖如纤维素、淀粉由单糖通过糖苷键连接。脂类是细胞膜的主要成分。

代谢途径

糖酵解将葡萄糖分解为丙酮酸，产生ATP和NADH。三羧酸循环是碳骨架彻底氧化的途径，产生大量还原当量。光合作用将CO₂固定为有机物，是地球上最重要的碳固定过程。

同位素化学

碳的稳定同位素¹²C(98.9%)和¹³C(1.1%)在自然界中的丰度比可用于追踪碳源。放射性同位素¹⁴C(t₁/₂=5730年)用于放射性碳定年。同位素标记技术(如¹³C-NMR)是研究反应机理的重要工具。
