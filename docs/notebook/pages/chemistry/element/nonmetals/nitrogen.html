氮 (N)

基本性质

氮是元素周期表第7号元素，原子序数7，原子量14.01，电子构型[He]2s²2p³。作为第五主族的头号元素，氮具有独特的三重键形成能力。氮气(N₂)占空气的78%，由于N≡N三键键能高达945 kJ/mol，氮气在常温下化学性质稳定。氮是生命必需元素，构成氨基酸、核酸等生物大分子。

原子结构特征

氮原子的价电子构型为2s²2p³，可形成三个共价键。氮的电负性(3.04)较高，仅次于氟、氧，易形成极性共价键。氮的离子半径小(N³⁻为171 pm)，电荷密度高，导致氮化物具有高晶格能。

分子氮的特性

N₂分子具有三重键：一个σ键和两个π键，键长1.10 Å。N₂的分子轨道电子排布为(σ2s)²(σ*2s)²(π2p)⁴(σ2p)²，键级为3。N₂的第一电离能(1503 kJ/mol)很高，电子亲和能接近零，解释了其化学惰性。

氮的固定与循环

工业固氮

哈伯-博施工艺：N₂ + 3H₂ ⇌ 2NH₃，ΔH = -92.4 kJ/mol，使用铁催化剂在400-500°C、200-300 atm下进行。该过程需要高能耗，但为农业提供重要氮肥原料。催化剂表面N₂解离吸附是速率控制步骤。

生物固氮

根瘤菌含有固氮酶，可将N₂还原为NH₃：N₂ + 8H⁺ + 8e⁻ + 16ATP → 2NH₃ + H₂ + 16ADP + 16Pi。固氮酶由铁蛋白和钼铁蛋白组成，对氧敏感。固氮过程消耗大量ATP，是生物体内最耗能的反应之一。

氮循环

氮在自然界中经历固氮、硝化、反硝化等过程。硝化细菌将NH₄⁺氧化为NO₃⁻：NH₄⁺ → NO₂⁻ → NO₃⁻。反硝化细菌在厌氧条件下将NO₃⁻还原为N₂。人类活动(如化肥使用)显著改变了全球氮循环。

氮的化合物

氨及其衍生物

氨(NH₃)是三角锥形分子，键角107.8°，具有孤对电子，是良好Lewis碱。氨水溶液呈弱碱性：NH₃ + H₂O ⇌ NH₄⁺ + OH⁻，Kb = 1.8×10⁻⁵。铵盐如NH₄Cl、(NH₄)₂SO₄是重要氮肥。

氮氧化物

一氧化氮(NO)是顺磁性分子，含奇数电子，是重要的信号分子。二氧化氮(NO₂)是红棕色气体，存在二聚平衡：2NO₂ ⇌ N₂O₄。硝酸(HNO₃)是强酸，具有强氧化性，与金属反应产物复杂。

氮的含氧酸

亚硝酸(HNO₂)是弱酸，不稳定，是强氧化剂。硝酸(HNO₃)通过奥斯特瓦尔德工艺制备：NH₃ → NO → NO₂ → HNO₃。王水(浓HCl:浓HNO₃ = 3:1)可溶解金和铂，形成[AuCl₄]⁻和[PtCl₆]²⁻。

有机氮化学

胺类化合物

胺类(RNH₂、R₂NH、R₃N)根据氮原子上取代基数分为伯、仲、叔胺。胺的碱性随烷基取代增加而增强：NH₃ < RNH₂ < R₂NH < R₃N(气相)。季铵盐(R₄N⁺X⁻)是重要表面活性剂和相转移催化剂。

氨基酸与肽

α-氨基酸是蛋白质的基本单元，通式为H₂N-CHR-COOH。氨基酸具有两性性质，等电点(pI)时净电荷为零。肽键(-CO-NH-)是氨基酸间的酰胺键，具有部分双键性质，限制肽链构象。

杂环化合物

吡啶是六元氮杂环，具有芳香性，碱性(pKb=8.8)弱于脂肪胺。吡咯是五元氮杂环，氮的孤对电子参与芳香体系，碱性极弱。嘌呤和嘧啶是核酸碱基，形成氢键配对(A-T, G-C)。

氮配位化学

金属氮配合物

过渡金属可形成多种氮配合物，如[Co(NH₃)₆]³⁺、[Fe(CN)₆]³⁻。氮分子配合物如[Ru(NH₃)₅(N₂)]²⁺是研究固氮机理的重要模型。金属卟啉配合物在催化氧化反应中具有重要作用。

氮杂环卡宾

N-杂环卡宾(NHC)是重要的有机催化剂和配体，如咪唑-2-亚基。NHC具有强σ给电子能力和弱π接受能力，稳定高价金属配合物。NHC配合物在交叉偶联反应中表现出优异活性。

氮宾化学

氮宾(Nitrene, RN:)是氮的类似卡宾的物种，含六价电子。氮宾可通过叠氮化合物光解或热解产生：RN₃ → RN: + N₂。氮宾可插入C-H键、加成到双键，在有机合成中有重要应用。
