物质的状态

物质通常以三种基本状态存在：固态、液态和气态。此外，还有等离子态和玻色-爱因斯坦凝聚态等特殊状态。

固态

固态物质具有固定的形状和体积。其粒子排列紧密，振动幅度小，只能在固定位置附近振动。

液态

液态物质具有固定的体积但无固定形状，会适应容器的形状。其粒子间距离比固态大，可以自由滑动。

气态

气态物质既无固定形状也无固定体积，会完全充满容器。其粒子间距离很大，运动速度快，相互碰撞频繁。

相变

物质在不同状态之间的转变称为相变：

  • 熔化：固态 → 液态
  • 凝固：液态 → 固态
  • 汽化：液态 → 气态
  • 凝结：气态 → 液态
  • 升华：固态 → 气态
  • 凝华：气态 → 固态

等离子态

等离子态是物质的第四种状态，由电离的气体组成，含有自由电子和离子。太阳和其他恒星主要由等离子体组成。
