化学反应

化学反应是指原子或分子重新排列形成新物质的过程。在反应过程中，原子本身不发生变化，只是它们的组合方式发生了改变。

化学反应的类型

化合反应

两种或多种物质结合生成一种新物质的反应。通式：A + B → AB

分解反应

一种物质分解成两种或多种较简单物质的反应。通式：AB → A + B

置换反应

一种单质与一种化合物反应，生成另一种单质和另一种化合物的反应。通式：A + BC → AC + B

复分解反应

两种化合物相互交换成分，生成两种新化合物的反应。通式：AB + CD → AD + CB

化学反应的特征

  • 颜色变化
  • 气体产生
  • 沉淀形成
  • 温度变化
  • 发光发热

化学方程式

化学方程式是用化学式表示化学反应的式子。书写化学方程式必须遵循质量守恒定律，即反应前后各元素的原子数目相等。

影响反应速率的因素

  • 反应物浓度
  • 温度
  • 催化剂
  • 反应物接触面积
  • 压强（对气体反应）
