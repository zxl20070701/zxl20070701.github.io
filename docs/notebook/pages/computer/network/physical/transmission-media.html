传输介质

传输介质是网络中数据传输的物理通道，是构成计算机网络物理层的基础。不同的传输介质具有不同的特性，适用于不同的网络环境和应用场景。选择合适的传输介质对于网络性能、成本和可靠性都有重要影响。

有线传输介质

1. 双绞线 (Twisted Pair Cable)

双绞线是最常用的有线传输介质，由两根绝缘铜导线按一定密度相互缠绕而成。

1.1 非屏蔽双绞线 (UTP - Unshielded Twisted Pair)

  • 结构特点：无金属屏蔽层，仅依靠双绞结构减少干扰
  • 分类标准：按照EIA/TIA-568标准分为Cat1-Cat8多个等级
    • Cat5e：支持100MHz，用于100BASE-TX和1000BASE-T
    • Cat6：支持250MHz，用于1000BASE-T和10GBASE-T（短距离）
    • Cat6a：支持500MHz，用于10GBASE-T（100米）
    • Cat7：支持600MHz，全屏蔽设计，支持10GBASE-T
    • Cat8：支持2000MHz，用于数据中心25/40GBASE-T
  • 优点：成本低、安装简单、灵活性高
  • 缺点：抗干扰能力较弱、传输距离有限（100米）
  • 典型应用：以太网、电话系统、综合布线

1.2 屏蔽双绞线 (STP - Shielded Twisted Pair)

  • 结构特点：每对线或整个电缆有金属屏蔽层
  • 屏蔽类型：
    • FTP (Foiled Twisted Pair)：铝箔屏蔽
    • SSTP (Shielded Screened Twisted Pair)：编织网+铝箔双重屏蔽
  • 优点：抗电磁干扰能力强、串扰小
  • 缺点：成本高、安装复杂、接地要求高
  • 典型应用：工业环境、高干扰区域、数据中心

2. 同轴电缆 (Coaxial Cable)

同轴电缆由内导体、绝缘层、外导体屏蔽层和保护套组成，具有良好的抗干扰能力。

2.1 基带同轴电缆

  • RG-8/RG-11：粗同轴电缆，用于10BASE5以太网
  • RG-58：细同轴电缆，用于10BASE2以太网
  • 特性阻抗：50Ω，用于数字信号传输
  • 传输距离：500米（粗缆）/185米（细缆）

2.2 宽带同轴电缆

  • RG-59：用于有线电视系统，特性阻抗75Ω
  • RG-6：改进型，用于卫星电视、宽带接入
  • 频分复用：支持多频道同时传输
  • 传输距离：可达数公里

2.3 优缺点分析

  • 优点：抗干扰能力强、带宽高、传输距离远
  • 缺点：成本高、安装复杂、扩展性差
  • 现状：在现代以太网中已基本被双绞线和光纤取代

3. 光纤 (Fiber Optic Cable)

光纤利用光脉冲传输数据，具有极高的带宽和传输距离，是现代高速网络的首选介质。

3.1 光纤分类

  • 按传输模式：
    • 单模光纤(SMF)：纤芯直径8-10μm，传输单一模式，适用于长距离
    • 多模光纤(MMF)：纤芯直径50μm或62.5μm，传输多个模式，适用于短距离
  • 按折射率分布：
    • 阶跃型光纤：纤芯和包层折射率突变
    • 渐变型光纤：折射率从中心向外逐渐减小

3.2 光纤规格

  • 单模光纤：
    • G.652：标准单模光纤，广泛用于城域网和长途传输
    • G.655：非零色散位移光纤，适用于DWDM系统
    • G.657：弯曲不敏感光纤，适用于室内布线
  • 多模光纤：
    • OM1：62.5/125μm，支持1Gb/s至275米
    • OM2：50/125μm，支持1Gb/s至550米
    • OM3：激光优化，支持10Gb/s至300米
    • OM4：改进型OM3，支持10Gb/s至550米
    • OM5：宽带多模光纤，支持40/100Gb/s

3.3 光纤连接器

  • SC连接器：方形卡接式，常用于千兆网络
  • LC连接器：小型化设计，用于高密度应用
  • FC连接器：螺纹连接，用于精密仪器
  • ST连接器：卡口式，用于多模网络
  • MPO/MTP连接器：多芯连接器，用于40/100GbE

3.4 光纤优缺点

  • 优点：
    • 带宽极高：可达Tbps级别
    • 传输距离远：单模光纤可达数百公里
    • 抗干扰能力强：不受电磁干扰影响
    • 安全性高：难以窃听
    • 体积小、重量轻：便于布线
  • 缺点：
    • 成本高：材料和安装费用昂贵
    • 连接复杂：需要专业设备和技能
    • 脆弱性：弯曲半径有限，易断裂
    • 设备兼容性：需要光电转换设备

无线传输介质

1. 无线电波 (Radio Waves)

无线电波是频率在3kHz-300GHz之间的电磁波，广泛用于无线通信。

1.1 频段划分

  • VLF (3-30kHz)：甚低频，用于导航
  • LF (30-300kHz)：低频，用于导航和授时
  • MF (300kHz-3MHz)：中频，用于AM广播
  • HF (3-30MHz)：高频，用于短波通信
  • VHF (30-300MHz)：甚高频，用于FM广播、电视
  • UHF (300MHz-3GHz)：特高频，用于移动通信、WiFi
  • SHF (3-30GHz)：超高频，用于卫星通信、微波
  • EHF (30-300GHz)：极高频，用于毫米波通信

1.2 传播特性

  • 地波传播：沿地面传播，适用于低频
  • 天波传播：经电离层反射，适用于高频
  • 视距传播：直线传播，适用于甚高频以上
  • 散射传播：利用对流层散射，适用于特高频

2. 微波通信 (Microwave Communication)

微波是频率在1-300GHz之间的电磁波，主要用于视距通信。

  • 频段应用：
    • L波段(1-2GHz)：移动通信、GPS
    • S波段(2-4GHz)：雷达、WiFi 2.4GHz
    • C波段(4-8GHz)：卫星通信
    • X波段(8-12GHz)：雷达、卫星通信
    • Ku波段(12-18GHz)：卫星电视
    • K波段(18-27GHz)：毫米波通信
    • Ka波段(27-40GHz)：5G、卫星通信
  • 特点：方向性强、容量大、建设周期短
  • 限制：视距传播、受天气影响、需要中继站

3. 卫星通信 (Satellite Communication)

利用人造卫星作为中继站进行远距离通信。

3.1 卫星轨道分类

  • GEO (地球静止轨道)：高度35786km，覆盖范围广，延迟大(250ms)
  • MEO (中地球轨道)：高度5000-15000km，用于导航(GPS、北斗)
  • LEO (低地球轨道)：高度500-1500km，延迟小，需要多颗卫星组网
  • HEO (高椭圆轨道)：用于高纬度地区通信

3.2 卫星通信特点

  • 优点：覆盖范围广、广播能力强、不受地理限制
  • 缺点：传输延迟大、成本高、受天气影响
  • 应用：电视广播、互联网接入、远程教育、应急通信

4. 红外线通信 (Infrared Communication)

利用红外线进行短距离无线通信。

  • 波长范围：700nm-1mm，通常使用850nm-900nm
  • 标准协议：IrDA (Infrared Data Association)
  • 传输速率：传统IrDA 115.2Kbps，现代版本可达4Mbps
  • 特点：成本低、功耗低、安全性高（视距传播）
  • 限制：视距传播、易受障碍物阻挡、传输距离短
  • 应用：遥控器、早期手机数据传输、短距离设备互联

传输介质的选择因素

1. 性能要求

  • 带宽需求：根据应用选择满足带宽要求的介质
  • 传输距离：考虑介质的最大传输距离和中继需求
  • 延迟要求：实时应用需要低延迟介质
  • 可靠性要求：关键应用需要高可靠性介质

2. 环境因素

  • 电磁干扰：高干扰环境选择抗干扰能力强的介质
  • 物理环境：考虑温度、湿度、腐蚀性等因素
  • 布线条件：考虑管道空间、弯曲半径等限制
  • 安全要求：涉密环境选择安全性高的介质

3. 经济因素

  • 初期投资：介质成本、安装成本、设备成本
  • 维护成本：日常维护、故障修复、升级成本
  • 扩展性：未来升级和扩展的成本考虑
  • 生命周期：介质的使用寿命和折旧成本

4. 技术标准

  • 兼容性：与现有网络设备的兼容性
  • 标准化：选择符合国际标准的产品
  • 未来发展：考虑技术发展趋势和升级路径
  • 厂商支持：选择有良好技术支持的厂商
