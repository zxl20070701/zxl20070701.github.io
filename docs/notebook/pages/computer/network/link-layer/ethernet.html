以太网

以太网(Ethernet)是目前应用最广泛的局域网技术，由Xerox公司在1970年代初期开发，后经DEC、Intel和Xerox共同标准化。以太网技术以其简单、可靠、成本低廉等特点，成为局域网技术的事实标准。从最初10Mbps发展到今天的400Gbps，以太网技术不断演进，适应着不断增长的网络需求。

以太网发展历史

1. 早期发展

  • 1973年：Robert Metcalfe在Xerox PARC发明以太网
  • 1980年：DIX联盟发布Ethernet v1.0规范
  • 1982年：发布Ethernet v2.0规范(DIX Ethernet)
  • 1983年：IEEE 802.3标准正式发布

2. 标准演进

  • 10BASE5：1983年，粗同轴电缆，500米，10Mbps
  • 10BASE2：1985年，细同轴电缆，185米，10Mbps
  • 10BASE-T：1990年，双绞线，100米，10Mbps
  • 100BASE-TX：1995年，快速以太网，100Mbps
  • 1000BASE-T：1999年，千兆以太网，1Gbps
  • 10GBASE-T：2006年，万兆以太网，10Gbps

以太网标准体系

1. IEEE 802.3标准族

IEEE 802.3是以太网的核心标准，定义了物理层和数据链路层的MAC子层规范。

1.1 物理层标准

  • 编码方式：曼彻斯特编码、4B/5B、8B/10B、PAM等
  • 传输介质：同轴电缆、双绞线、光纤
  • 拓扑结构：总线型、星型、点对点

1.2 数据链路层标准

  • MAC子层：介质访问控制
  • LLC子层：逻辑链路控制(IEEE 802.2)
  • 帧格式：以太网帧结构定义

2. 以太网命名规则

以太网标准使用特定的命名规则：速率(Mbps/Gbps) + BASE + 介质类型

  • 速率：10、100、1000、10000、40000、100000等
  • BASE：基带传输(Baseband)
  • 介质类型：T(双绞线)、F(光纤)、L(长波长光纤)、S(短波长光纤)等

传统以太网

1. 10Mbps以太网

1.1 10BASE5 (粗缆以太网)

  • 传输介质：RG-8同轴电缆，直径约1cm
  • 最大长度：500米
  • 连接方式：使用N型连接器，通过收发器连接设备
  • 拓扑结构：总线型
  • 特点：可靠性高，但安装维护困难

1.2 10BASE2 (细缆以太网)

  • 传输介质：RG-58同轴电缆，直径约5mm
  • 最大长度：185米
  • 连接方式：使用BNC T型连接器
  • 拓扑结构：总线型
  • 特点：成本低，但可靠性差，单点故障影响全网

1.3 10BASE-T (双绞线以太网)

  • 传输介质：3类或更高UTP双绞线
  • 最大长度：100米
  • 拓扑结构：星型(使用集线器或交换机)
  • 编码方式：曼彻斯特编码
  • 特点：安装简便，故障隔离容易，成为主流

2. 以太网帧格式

2.1 Ethernet II帧格式 

| 前导码(7) | SFD(1) | 目的MAC(6) | 源MAC(6) | 类型(2) | 数据(46-1500) | FCS(4) |
  • 前导码：10101010模式，用于时钟同步
  • SFD：帧开始定界符，10101011
  • 类型字段：标识上层协议，如0x0800(IPv4)、0x86DD(IPv6)

2.2 IEEE 802.3帧格式 

| 前导码(7) | SFD(1) | 目的MAC(6) | 源MAC(6) | 长度(2) | LLC/SNAP | 数据 | FCS(4) |
  • 长度字段：指示数据字段长度
  • LLC首部：逻辑链路控制子层
  • SNAP首部：子网访问协议，用于协议标识

3. 介质访问控制

3.1 CSMA/CD算法

  • 载波监听：发送前检测信道状态
  • 冲突检测：边发送边监听，检测冲突
  • 退避算法：二进制指数退避
  • 限制：最大电缆长度2500米，最小帧长64字节

3.2 网络直径计算

  • 传播时延：信号在电缆中传播的最大时间
  • 发送时延：发送最小帧所需时间
  • 约束条件：发送时延 ≥ 2 × 传播时延
  • 计算结果：10Mbps以太网最大网络直径约2500米

快速以太网 (100Mbps)

1. 100BASE-TX

  • 传输介质：Cat5 UTP双绞线，使用两对线(一对发送，一对接收)
  • 最大长度：100米
  • 编码方式：4B/5B编码 + MLT-3调制
  • 信号速率：125MHz
  • 拓扑结构：星型

2. 100BASE-FX

  • 传输介质：多模光纤
  • 最大长度：400米(半双工)或2000米(全双工)
  • 编码方式：4B/5B编码 + NRZI调制
  • 连接器：ST或SC连接器

3. 100BASE-T4

  • 传输介质：Cat3 UTP双绞线，使用全部四对线
  • 编码方式：8B/6T编码
  • 特点：兼容现有Cat3布线系统
  • 应用：已淘汰，被100BASE-TX取代

4. 自动协商 (Auto-Negotiation)

  • 功能：自动选择最佳工作模式
  • 协商内容：速率(10/100Mbps)、双工模式(半/全双工)、流控
  • 优先级：100BASE-TX全双工 > 100BASE-TX半双工 > 10BASE-T全双工 > 10BASE-T半双工
  • 信号机制：使用快速链路脉冲(FLP)交换能力信息

千兆以太网 (1Gbps)

1. 1000BASE-T

  • 传输介质：Cat5e或更高UTP双绞线，使用全部四对线
  • 最大长度：100米
  • 编码方式：PAM-5编码 + 4D-PAM5调制
  • 信号速率：每对线250Mbps，总计1000Mbps
  • 技术特点：全双工工作，每对线同时收发

2. 1000BASE-SX

  • 传输介质：多模光纤(850nm波长)
  • 最大长度：220米(62.5μm光纤)或550米(50μm光纤)
  • 编码方式：8B/10B编码
  • 连接器：SC或LC连接器

3. 1000BASE-LX

  • 传输介质：单模或多模光纤(1300nm波长)
  • 最大长度：550米(多模)或5000米(单模)
  • 编码方式：8B/10B编码
  • 应用：长距离连接

4. 载波扩展与帧突发

4.1 载波扩展

  • 目的：保持最小帧长64字节的同时支持更高速率
  • 机制：短帧添加填充位延长至512字节
  • 影响：增加开销，降低效率

4.2 帧突发

  • 目的：提高短帧传输效率
  • 机制：连续发送多个短帧，只扩展第一个帧
  • 限制：最大突发长度8192字节

万兆以太网 (10Gbps)

1. 10GBASE标准

1.1 10GBASE-T

  • 传输介质：Cat6a或Cat7双绞线
  • 最大长度：100米
  • 编码方式：PAM-16 + LDPC纠错
  • 技术特点：全双工工作，无CSMA/CD

1.2 10GBASE-SR

  • 传输介质：多模光纤(850nm)
  • 最大长度：26米(OM1)或300米(OM3)
  • 编码方式：64B/66B
  • 连接器：LC连接器

1.3 10GBASE-LR

  • 传输介质：单模光纤(1310nm)
  • 最大长度：10公里
  • 编码方式：64B/66B
  • 应用：城域网连接

1.4 10GBASE-ER

  • 传输介质：单模光纤(1550nm)
  • 最大长度：40公里
  • 编码方式：64B/66B
  • 应用：广域网连接

2. 技术特点

  • 全双工模式：不再使用CSMA/CD
  • 编码效率：64B/66B编码，效率97%
  • 接口类型：XENPAK、X2、XFP、SFP+

高速以太网 (40G/100G/400G)

1. 40 Gigabit Ethernet

  • 标准：IEEE 802.3ba-2010
  • 物理层：4×10Gbps通道并行传输
  • 编码：64B/66B编码
  • 应用：数据中心核心交换机互联

2. 100 Gigabit Ethernet

  • 标准：IEEE 802.3ba-2010
  • 物理层选项：
    • 100GBASE-SR4：多模光纤，100米
    • 100GBASE-LR4：单模光纤，10公里
    • 100GBASE-ER4：单模光纤，40公里
  • 编码：64B/66B编码
  • 接口：CFP、CFP2、CFP4、QSFP28

3. 400 Gigabit Ethernet

  • 标准：IEEE 802.3bs-2017
  • 物理层：8×50Gbps或4×100Gbps通道
  • 编码：256b/257b编码
  • 接口：QSFP-DD、OSFP

以太网交换机技术

1. 交换机工作原理

  • 地址学习：通过源MAC地址学习端口映射
  • 帧转发：基于目的MAC地址转发帧
  • 过滤：丢弃不需要转发的帧
  • 消除冲突域：每个端口独立冲突域

2. 交换模式

2.1 存储转发 (Store-and-Forward)

  • 工作机制：接收完整帧，校验CRC后转发
  • 优点：错误检测，支持不同速率端口
  • 缺点：延迟较大

2.2 直通交换 (Cut-Through)

  • 工作机制：收到目的MAC后立即转发
  • 优点：延迟最小
  • 缺点：无法检测错误帧，要求同速率端口

2.3 碎片隔离 (Fragment-Free)

  • 工作机制：存储前64字节后转发
  • 优点：过滤冲突碎片，延迟较小
  • 应用：折中方案

3. 高级交换技术

3.1 多层交换

  • 二层交换：基于MAC地址
  • 三层交换：基于IP地址，集成路由功能
  • 四层交换：基于端口号，支持负载均衡

3.2 交换架构

  • 共享内存：集中式交换架构
  • 交叉开关：矩阵式交换架构
  • Clos网络：多级交换架构

以太网供电技术 (PoE)

1. PoE标准

1.1 IEEE 802.3af (PoE)

  • 最大功率：15.4W(端口输出)
  • 工作电压：44-57V DC
  • 供电方式：方案A(数据线对)、方案B(备用线对)

1.2 IEEE 802.3at (PoE+)

  • 最大功率：30W(端口输出)
  • 向后兼容：兼容802.3af设备
  • 增强功能：支持更高功率设备

1.3 IEEE 802.3bt (PoE++)

  • 类型3：最大功率60W
  • 类型4：最大功率100W
  • 应用场景：高功率设备如笔记本电脑、显示器

2. PoE工作原理

  • 检测阶段：检测连接设备是否支持PoE
  • 分类阶段：确定设备所需功率级别
  • 供电阶段：提供稳定直流电源
  • 断电阶段：设备断开时停止供电

以太网发展趋势

1. 更高速率

  • 800Gbps：正在标准化中
  • 1.6Tbps：研究阶段
  • 技术挑战：信号完整性、功耗、散热

2. 新兴应用

  • 数据中心：支持云计算、大数据
  • 5G承载：前传、中传、回传网络
  • 工业以太网：实时性、可靠性要求
  • 车载网络：汽车电子系统互联

3. 技术演进

  • 光通信：硅光子技术、相干光通信
  • 智能网卡：网卡卸载、可编程网络
  • 时间敏感网络：TSN标准支持实时应用
  • 绿色以太网：节能技术、能效优化
