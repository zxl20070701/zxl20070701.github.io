基础概念

计算机网络是将多台计算机通过通信线路连接起来，实现资源共享和信息交换的系统。理解计算机网络的基础知识对于现代信息技术领域至关重要。

1. 计算机网络概述

计算机网络的基本概念和分类是理解网络技术的基础。

1.1 网络分类

  • 局域网 (LAN)：覆盖范围小，通常在建筑物或校园内
  • 城域网 (MAN)：覆盖一个城市范围
  • 广域网 (WAN)：覆盖范围大，如互联网
  • 个人区域网 (PAN)：个人设备间的短距离通信

1.2 网络拓扑结构

  • 总线型：所有设备连接到一条主干线
  • 星型：所有设备连接到一个中心节点
  • 环型：设备连接成环状
  • 网状：设备间有多条连接路径

2. OSI七层模型

OSI（开放系统互联）参考模型将网络通信分为七个层次，每一层都有特定的功能和协议。

  1. 物理层：传输比特流，定义物理连接标准
  2. 数据链路层：提供可靠的数据传输，处理错误检测和纠正
  3. 网络层：负责数据包的路由选择和转发
  4. 传输层：提供端到端的通信服务，确保数据完整传输
  5. 会话层：建立、管理和终止会话
  6. 表示层：处理数据格式转换、加密解密
  7. 应用层：为应用程序提供网络服务接口

3. TCP/IP四层模型

TCP/IP模型是实际应用中更常用的网络模型，由以下四层组成：

  1. 网络接口层：对应OSI的物理层和数据链路层
  2. 网络层：主要协议是IP，负责寻址和路由
  3. 传输层：主要协议是TCP和UDP
  4. 应用层：包含各种应用协议如HTTP、FTP、SMTP等

4. IP地址与子网划分

4.1 IP地址分类

  • IPv4：32位地址，通常表示为四个十进制数（如192.168.1.1）
  • IPv6：128位地址，解决IPv4地址耗尽问题

4.2 子网掩码

子网掩码用于划分网络地址和主机地址，如255.255.255.0表示前24位是网络地址。

4.3 CIDR表示法

CIDR（无类别域间路由）使用斜线表示法，如192.168.1.0/24表示前24位为网络地址。

5. 网络设备

5.1 常见网络设备

  • 集线器 (Hub)：物理层设备，简单转发数据
  • 交换机 (Switch)：数据链路层设备，根据MAC地址转发数据
  • 路由器 (Router)：网络层设备，根据IP地址进行路由选择
  • 网关 (Gateway)：连接不同网络的设备
  • 防火墙 (Firewall)：网络安全设备，控制网络访问

6. 网络安全基础

6.1 常见安全威胁

  • 病毒和恶意软件：破坏系统或窃取信息
  • 拒绝服务攻击 (DoS/DDoS)：使网络服务不可用
  • 中间人攻击：窃听或篡改通信内容
  • 钓鱼攻击：诱骗用户提供敏感信息

6.2 安全防护措施

  • 防火墙：控制进出网络的数据流
  • 加密技术：保护数据传输安全
  • 身份认证：验证用户身份
  • 访问控制：限制资源访问权限

7. 无线网络技术

7.1 Wi-Fi标准

  • 802.11a/b/g：早期Wi-Fi标准
  • 802.11n：支持MIMO技术，提高传输速率
  • 802.11ac：5GHz频段，更高速度
  • 802.11ax (Wi-Fi 6)：最新标准，提高效率

7.2 无线安全

  • WEP：早期加密标准，安全性较低
  • WPA/WPA2：目前主流的安全标准
  • WPA3：最新安全标准，增强保护

8. 网络性能指标

8.1 关键性能指标

  • 带宽：网络传输数据的能力，单位通常为Mbps或Gbps
  • 延迟：数据从发送到接收所需的时间
  • 吞吐量：实际数据传输速率
  • 丢包率：传输过程中丢失数据包的比例
  • 抖动：延迟的变化程度

9. 网络管理基础

9.1 网络管理功能

  • 配置管理：管理网络设备配置
  • 性能管理：监控网络性能指标
  • 故障管理：检测和处理网络故障
  • 安全管理：保护网络安全
  • 计费管理：记录网络资源使用情况

10. 云计算与网络

10.1 云服务模型

  • IaaS：基础设施即服务，提供虚拟化的计算资源
  • PaaS：平台即服务，提供开发和部署环境
  • SaaS：软件即服务，提供应用程序服务

10.2 云网络特点

  • 虚拟化：网络资源虚拟化，提高灵活性
  • 弹性扩展：根据需求动态调整资源
  • 按需服务：按使用量付费
  • 广泛接入：通过互联网随时随地访问
