数制转换

数制系统概述

数制是表示数值的方法，计算机中常用的数制包括二进制、八进制、十进制和十六进制。理解不同数制及其转换是计算机科学的基础。

常用数制系统

1. 二进制（Binary）

基数为2，使用数字0和1表示。

  • 特点：与计算机硬件直接对应，便于逻辑运算
  • 表示：后缀B或下标2，如1011B或1011₂
  • 权重：2^n，从右到左依次为2⁰, 2¹, 2², 2³...

2. 八进制（Octal）

基数为8，使用数字0-7表示。

  • 特点：二进制数字的紧凑表示，每3位二进制对应1位八进制
  • 表示：后缀O或Q，或下标8，如17O或17₈
  • 权重：8^n，从右到左依次为8⁰, 8¹, 8², 8³...

3. 十进制（Decimal）

基数为10，使用数字0-9表示。

  • 特点：人类日常使用的数制，便于理解和计算
  • 表示：通常不加后缀，如123
  • 权重：10^n，从右到左依次为10⁰, 10¹, 10², 10³...

4. 十六进制（Hexadecimal）

基数为16，使用数字0-9和字母A-F表示。

  • 特点：二进制数字的紧凑表示，每4位二进制对应1位十六进制
  • 表示：后缀H或下标16，如1FH或1F₁₆
  • 权重：16^n，从右到左依次为16⁰, 16¹, 16², 16³...

数制转换方法

1. 任意进制转十进制

按权展开求和法：将每一位数字乘以对应位的权值，然后求和。

例：1011.11₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ + 1×2⁻¹ + 1×2⁻² = 8 + 0 + 2 + 1 + 0.5 + 0.25 = 11.75₁₀

例：3A.2₁₆ = 3×16¹ + 10×16⁰ + 2×16⁻¹ = 48 + 10 + 0.125 = 58.125₁₀

2. 十进制转任意进制

整数部分：除基取余法

将十进制整数不断除以目标进制的基数，取余数，直到商为0，余数倒序排列。

例：25₁₀ → 二进制

25 ÷ 2 = 12 余 1

12 ÷ 2 = 6 余 0

6 ÷ 2 = 3 余 0

3 ÷ 2 = 1 余 1

1 ÷ 2 = 0 余 1

结果：11001₂

小数部分：乘基取整法

将十进制小数不断乘以目标进制的基数，取整数部分，直到小数部分为0或达到所需精度。

例：0.625₁₀ → 二进制

0.625 × 2 = 1.25 → 取整1

0.25 × 2 = 0.5 → 取整0

0.5 × 2 = 1.0 → 取整1

结果：0.101₂

3. 二进制与八进制、十六进制转换

二进制 ↔ 八进制

每3位二进制对应1位八进制，不足3位补0。

例：1011011.101₂ → 八进制

分组：001 011 011.101

转换：1 3 3.5

结果：133.5₈

二进制 ↔ 十六进制

每4位二进制对应1位十六进制，不足4位补0。

例：1011011.101₂ → 十六进制

分组：0101 1011.1010

转换：5 B.A

结果：5B.A₁₆

编码系统

1. BCD码（Binary-Coded Decimal）

用4位二进制表示1位十进制数字。

  • 有权码：8421码、2421码等
  • 无权码：余3码、格雷码等
  • 应用：数字显示、财务计算

2. 格雷码（Gray Code）

相邻两个数只有一位不同，减少转换错误。

  • 特点：循环性、反射性
  • 应用：角度编码器、数字通信

数制转换的应用

计算机系统中的数制

  • 机器内部：二进制
  • 程序设计：十六进制（便于表示地址）
  • 用户界面：十进制
  • 文件权限：八进制（Unix/Linux）
