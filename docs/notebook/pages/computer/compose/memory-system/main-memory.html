主存储器（SRAM/DRAM）

主存储器概述

主存储器（Main Memory）是计算机系统中用于临时存储程序和数据的核心存储部件，直接与CPU交换信息。主存储器通常由SRAM（静态随机存取存储器）和DRAM（动态随机存取存储器）组成，两者在结构、性能和应用方面各有特点。

SRAM（静态随机存取存储器）

1. SRAM基本原理

SRAM使用双稳态触发器存储信息，只要保持电源供应，数据就能长期保存，无需刷新。

基本结构

  • 存储单元：6个晶体管组成的双稳态电路
  • 字线（Word Line）：控制存储单元的选通
  • 位线（Bit Line）：传输数据信号

工作原理

  • 读操作：字线激活，位线检测存储单元状态
  • 写操作：字线激活，位线施加写入信号
  • 保持状态：字线不激活，存储单元保持原有状态

2. SRAM特点

  • 速度快：访问时间通常在1-10纳秒
  • 功耗较高：每个存储单元需要持续供电
  • 集成度低：每个存储单元需要6个晶体管
  • 成本高：芯片面积大，制造成本高

3. SRAM应用

  • CPU缓存：L1、L2、L3缓存
  • 寄存器堆：处理器内部寄存器
  • 高速缓冲：需要快速访问的场合

DRAM（动态随机存取存储器）

1. DRAM基本原理

DRAM使用电容存储信息，电容充电表示1，放电表示0。由于电容会漏电，需要定期刷新。

基本结构

  • 存储单元：1个晶体管+1个电容
  • 字线：控制晶体管的导通
  • 位线：传输数据信号

工作原理

  • 读操作：字线激活，晶体管导通，检测位线电压变化
  • 写操作：字线激活，晶体管导通，位线施加写入电压
  • 刷新操作：定期读取并重新写入数据，补充电容电荷

2. DRAM特点

  • 速度较慢：访问时间通常在50-100纳秒
  • 功耗较低：存储单元简单，静态功耗小
  • 集成度高：每个存储单元只需1个晶体管+1个电容
  • 成本低：芯片面积小，制造成本低
  • 需要刷新：每几毫秒需要刷新一次

3. DRAM类型

SDRAM（同步DRAM）

  • 与系统时钟同步工作
  • 支持突发传输模式
  • 广泛应用于PC内存

DDR SDRAM（双倍数据速率SDRAM）

  • DDR：在时钟上升沿和下降沿都传输数据
  • DDR2：改进的信号完整性和功耗
  • DDR3：更高的数据传输速率
  • DDR4：进一步降低功耗，提高容量
  • DDR5：最新标准，更高带宽和容量

GDDR（图形DDR）

  • 专为图形应用优化
  • 高带宽设计
  • 用于显卡显存

主存储器组织结构

1. 存储单元组织

  • 位结构：每个存储单元存储1位信息
  • 字结构：多个位组成一个字（通常为8位、16位、32位或64位）
  • 地址空间：每个存储单元有唯一的地址

2. 地址译码

  • 线性译码：每个地址对应一个译码器输出，简单但硬件开销大
  • 矩阵译码：将地址分为行地址和列地址，减少译码器数量
  • 双译码：同时译码行地址和列地址，提高效率

3. 存储器芯片结构

  • 存储阵列：存储单元的二维阵列
  • 行译码器：选择激活的行
  • 列译码器：选择激活的列
  • 读写电路：数据输入输出缓冲
  • 控制逻辑：控制读写时序

主存储器性能指标

1. 存储容量

  • 芯片容量：单个存储器芯片的容量
  • 模块容量：存储器模块的总容量
  • 系统容量：整个系统的最大内存容量

2. 访问时间

  • 读取时间：从发出读命令到数据稳定输出的时间
  • 写入时间：从发出写命令到数据写入完成的时间
  • 周期时间：连续两次访问之间的最小时间间隔

3. 带宽

  • 数据宽度：一次能传输的数据位数
  • 传输速率：单位时间内传输的数据量
  • 突发传输：连续传输多个数据单元的模式

主存储器接口技术

1. 总线接口

  • 地址总线：传输存储器地址
  • 数据总线：传输数据
  • 控制总线：传输读写控制信号

2. 时序控制

  • 同步接口：与系统时钟同步
  • 异步接口：不依赖系统时钟
  • 握手协议：通过应答信号协调传输

3. 错误检测与纠正

  • 奇偶校验：简单的错误检测
  • ECC：错误检测和纠正
  • Chipkill：更高级的错误纠正技术

主存储器发展趋势

1. 容量提升

  • 工艺进步：更小的工艺节点，更高密度
  • 3D堆叠：垂直堆叠增加容量
  • 新型存储单元：如3D NAND、MRAM等

2. 速度提升

  • 接口升级：从DDR到DDR5，带宽不断提升
  • 协议优化：改进的通信协议减少延迟
  • 近存计算：减少数据移动，提高速度

3. 功耗优化

  • 低功耗模式：支持多种省电模式
  • 动态电压调节：根据负载调整电压
  • 新型材料：如相变存储器、阻变存储器等
