内存与存储系统

存储器层次结构

计算机存储系统采用层次结构设计，目的是在速度、容量和成本之间取得平衡：

  1. 寄存器（Registers）：CPU内部的高速存储，速度最快，容量最小
  2. 高速缓存（Cache）：分为L1、L2、L3三级，速度较快，容量较小
  3. 主存储器（Main Memory）：即内存条，速度中等，容量中等
  4. 辅助存储器（Secondary Storage）：硬盘、SSD等，速度较慢，容量较大
  5. 离线存储器（Offline Storage）：磁带、光盘等，速度最慢，容量可很大

内存的基本概念

内存（Memory） 是计算机的主存储器，用于临时存储正在运行的程序和数据。内存直接与CPU交换数据，是计算机运行速度的关键因素之一。

内存的分类

计算机内存按照不同的标准可以分为多种类型：

  • 按读写特性：
    • RAM（随机存取存储器）：可读写，断电后数据丢失
    • ROM（只读存储器）：只能读取，断电后数据不丢失
  • 按技术实现：
    • DRAM（动态随机存取存储器）：需要定期刷新，容量大，成本低
    • SRAM（静态随机存取存储器）：不需要刷新，速度快，成本高

虚拟内存

虚拟内存（Virtual Memory） 是一种内存管理技术，它使得应用程序认为自己拥有连续的可用内存，而实际上这些内存可能是分散在物理内存和磁盘上的。

虚拟内存的主要优点：

  • 允许运行比物理内存更大的程序
  • 提供内存保护，防止程序间相互干扰
  • 简化内存管理，为每个进程提供独立的地址空间
  • 支持内存共享，多个进程可以共享相同的代码和数据

存储设备类型

常见的计算机存储设备包括：

  • 硬盘驱动器（HDD）：机械式存储，容量大，成本低，速度较慢
  • 固态硬盘（SSD）：电子式存储，速度快，抗震性好，成本较高
  • 光盘（CD/DVD/Blu-ray）：光学存储，适合长期保存数据
  • USB闪存盘：便携式存储，方便携带，容量适中
  • 存储卡（SD卡、TF卡等）：用于相机、手机等移动设备

内存管理技术

现代操作系统采用多种内存管理技术提高系统效率：

  • 分页（Paging）：将内存划分为固定大小的页面进行管理
  • 分段（Segmentation）：将内存划分为不同大小的段，每段有特定用途
  • 段页式管理：结合分段和分页的优点
  • 内存映射文件：将文件内容映射到进程的地址空间
