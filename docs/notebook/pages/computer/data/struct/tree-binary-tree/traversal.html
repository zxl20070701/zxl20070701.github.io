二叉树遍历

遍历的基本概念

二叉树的遍历（traversing binary tree）是指从根结点出发，按照某种次序依次访问二叉树中所有结点，使得每个结点被访问一次且仅被访问一次。遍历是二叉树最基本的运算，是二叉树上进行其他运算的基础。

遍历的重要性

  • 数据访问：按照特定顺序访问树中的所有数据
  • 树的操作：许多树的操作都基于遍历实现
  • 算法基础：许多树算法都以遍历为基础

遍历的分类

深度优先遍历

深度优先遍历按照深度方向进行遍历，包括前序遍历、中序遍历和后序遍历。

广度优先遍历

广度优先遍历按照层次顺序进行遍历，即层序遍历。

深度优先遍历

前序遍历（Pre-order Traversal）

前序遍历的基本思想是：若二叉树为空，则空操作返回；否则先访问根结点，然后前序遍历左子树，再前序遍历右子树。

递归实现 

void PreOrderTraverse(BiTree T) {
    if (T) {
        printf("%d ", T->data);        // 访问根结点
        PreOrderTraverse(T->lchild);   // 遍历左子树
        PreOrderTraverse(T->rchild);   // 遍历右子树
    }
}

非递归实现（使用栈） 

void PreOrderTraverse_NonRecursive(BiTree T) {
    Stack S;
    InitStack(&S);
    BiTree p = T;

    while (p || !StackEmpty(S)) {
        if (p) {
            printf("%d ", p->data);  // 访问根结点
            Push(&S, p);
            p = p->lchild;           // 遍历左子树
        } else {
            Pop(&S, &p);
            p = p->rchild;           // 遍历右子树
        }
    }
}

中序遍历（In-order Traversal）

中序遍历的基本思想是：若二叉树为空，则空操作返回；否则从根结点开始（注意并不是先访问根结点），中序遍历根结点的左子树，然后是访问根结点，最后中序遍历右子树。

递归实现 

void InOrderTraverse(BiTree T) {
    if (T) {
        InOrderTraverse(T->lchild);    // 遍历左子树
        printf("%d ", T->data);        // 访问根结点
        InOrderTraverse(T->rchild);    // 遍历右子树
    }
}

非递归实现（使用栈） 

void InOrderTraverse_NonRecursive(BiTree T) {
    Stack S;
    InitStack(&S);
    BiTree p = T;

    while (p || !StackEmpty(S)) {
        if (p) {
            Push(&S, p);
            p = p->lchild;           // 遍历左子树
        } else {
            Pop(&S, &p);
            printf("%d ", p->data);  // 访问根结点
            p = p->rchild;           // 遍历右子树
        }
    }
}

后序遍历（Post-order Traversal）

后序遍历的基本思想是：若二叉树为空，则空操作返回；否则从左到右先叶子后结点的方式遍历访问左右子树，最后是访问根结点。

递归实现 

void PostOrderTraverse(BiTree T) {
    if (T) {
        PostOrderTraverse(T->lchild);  // 遍历左子树
        PostOrderTraverse(T->rchild);  // 遍历右子树
        printf("%d ", T->data);        // 访问根结点
    }
}

非递归实现（使用栈） 

void PostOrderTraverse_NonRecursive(BiTree T) {
    Stack S;
    InitStack(&S);
    BiTree p = T, r = NULL;

    while (p || !StackEmpty(S)) {
        if (p) {
            Push(&S, p);
            p = p->lchild;  // 遍历左子树
        } else {
            GetTop(S, &p);
            if (p->rchild && p->rchild != r) {
                p = p->rchild;  // 遍历右子树
            } else {
                Pop(&S, &p);
                printf("%d ", p->data);  // 访问根结点
                r = p;
                p = NULL;
            }
        }
    }
}

广度优先遍历

层序遍历（Level-order Traversal）

层序遍历的基本思想是：从树的第一层（即根结点）开始，从上而下逐层遍历，在同一层中，按从左到右的顺序对结点逐个访问。

实现（使用队列） 

void LevelOrderTraverse(BiTree T) {
    if (!T) return;

    Queue Q;
    InitQueue(&Q);
    EnQueue(&Q, T);

    while (!QueueEmpty(Q)) {
        BiTree p;
        DeQueue(&Q, &p);
        printf("%d ", p->data);

        if (p->lchild) EnQueue(&Q, p->lchild);
        if (p->rchild) EnQueue(&Q, p->rchild);
    }
}

遍历的应用

前序遍历的应用

  • 复制二叉树
  • 获取二叉树的深度
  • 前缀表达式求值

中序遍历的应用

  • 二叉搜索树的中序遍历结果是有序序列
  • 中缀表达式求值

后序遍历的应用

  • 计算二叉树的结点个数
  • 计算二叉树的叶子结点个数
  • 后缀表达式求值
  • 销毁二叉树

层序遍历的应用

  • 计算二叉树的宽度
  • 寻找二叉树的最左/最右结点
  • 序列化和反序列化二叉树

遍历序列的性质

前序+中序确定唯一二叉树

给定一棵二叉树的前序遍历序列和中序遍历序列，可以唯一确定这棵二叉树。

后序+中序确定唯一二叉树

给定一棵二叉树的后序遍历序列和中序遍历序列，可以唯一确定这棵二叉树。

前序+后序不能确定唯一二叉树

给定一棵二叉树的前序遍历序列和后序遍历序列，不能唯一确定这棵二叉树。

遍历的复杂度分析

遍历方法 时间复杂度 空间复杂度（递归） 空间复杂度（非递归）
前序遍历 O(n) O(h) O(h)
中序遍历 O(n) O(h) O(h)
后序遍历 O(n) O(h) O(h)
层序遍历 O(n) O(w) O(w)

其中，n是二叉树的结点数，h是二叉树的高度，w是二叉树的最大宽度。

遍历的优化技巧

Morris遍历

Morris遍历是一种不需要栈或递归的二叉树遍历算法，可以在O(1)空间复杂度下完成遍历。

线索化遍历

通过线索化二叉树，可以在不使用栈的情况下实现遍历。
