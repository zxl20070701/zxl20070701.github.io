特殊矩阵

特殊矩阵的定义

特殊矩阵是指具有特殊结构的矩阵，这些矩阵在存储和运算时可以采用特殊的压缩存储方法，以节省存储空间。常见的特殊矩阵包括对称矩阵、三角矩阵、对角矩阵等。

对称矩阵

定义

若n阶矩阵A中的元素满足性质：aᵢⱼ = aⱼᵢ（1 ≤ i, j ≤ n），则称A为n阶对称矩阵。

压缩存储方法

对称矩阵中的元素关于主对角线对称，因此可以为每一对对称元素分配一个存储空间，将n²个元素压缩存储到n(n+1)/2个存储空间中。可以按行优先或列优先的顺序存储下三角（包括对角线）中的元素。

按行优先存储

将下三角中的元素按行优先顺序存储到一维数组SA[n(n+1)/2]中，元素aᵢⱼ（i ≥ j）在数组SA中的下标k与(i,j)的对应关系为：

k = i(i-1)/2 + j-1 （当i ≥ j时）

对于上三角中的元素aᵢⱼ（i < j），由于aᵢⱼ = aⱼᵢ，因此可以通过访问aⱼᵢ来获得。

地址计算公式 

// 获取对称矩阵中元素a[i][j]在一维数组中的位置
int getIndex(int i, int j) {
    if (i >= j) {
        return i * (i - 1) / 2 + j - 1;
    } else {
        return j * (j - 1) / 2 + i - 1;
    }
}

三角矩阵

定义

以主对角线划分，三角矩阵有上三角矩阵和下三角矩阵两种。上三角矩阵是指矩阵下三角（不包括对角线）中的元素均为常数c或零的n阶矩阵；下三角矩阵是指矩阵上三角（不包括对角线）中的元素均为常数c或零的n阶矩阵。

上三角矩阵

压缩存储方法

重复元素c共享一个存储空间，其余的元素正好有n(n+1)/2个，将上三角矩阵压缩存储到数组SA[n(n+1)/2+1]中。元素aᵢⱼ在数组SA中的下标k与(i,j)的对应关系为：

k = (i-1)(2n-i+2)/2 + (j-i) （当i ≤ j时）

k = n(n+1)/2 （当i > j时，存储常数c）

下三角矩阵

压缩存储方法

与上三角矩阵类似，将下三角矩阵压缩存储到数组SA[n(n+1)/2+1]中。元素aᵢⱼ在数组SA中的下标k与(i,j)的对应关系为：

k = i(i-1)/2 + j-1 （当i ≥ j时）

k = n(n+1)/2 （当i < j时，存储常数c）

地址计算公式 

// 上三角矩阵元素位置计算
int getUpperTriIndex(int i, int j, int n) {
    if (i <= j) {
        return (i - 1) * (2 * n - i + 2) / 2 + (j - i);
    } else {
        return n * (n + 1) / 2; // 存储常数c的位置
    }
}

// 下三角矩阵元素位置计算
int getLowerTriIndex(int i, int j, int n) {
    if (i >= j) {
        return i * (i - 1) / 2 + j - 1;
    } else {
        return n * (n + 1) / 2; // 存储常数c的位置
    }
}

对角矩阵

定义

对角矩阵是指所有的非零元素都集中在以主对角线为中心的带状区域中的矩阵，即除了主对角线上和直接在对角线上、下方若干条对角线上的元素之外，所有其他的元素皆为零。

三对角矩阵

三对角矩阵是一种特殊的对角矩阵，其中非零元素仅出现在主对角线及其相邻的两条对角线上。对于n阶三对角矩阵，非零元素总数为3n-2。

压缩存储方法

将三对角矩阵的非零元素按行优先顺序存储到数组SA[3n-2]中。元素aᵢⱼ在数组SA中的下标k与(i,j)的对应关系为：

当|i-j| ≤ 1时，k = 2i + j - 3

地址计算公式 

// 三对角矩阵元素位置计算
int getTridiagonalIndex(int i, int j) {
    if (abs(i - j) <= 1) {
        return 2 * i + j - 3;
    } else {
        return -1; // 零元素，不存储
    }
}

稀疏矩阵

定义

稀疏矩阵是指矩阵中大多数元素为零，只有少量非零元素的矩阵。通常认为非零元素个数远小于矩阵元素总数的矩阵为稀疏矩阵。

压缩存储方法

稀疏矩阵的压缩存储方法将在专门的章节中详细介绍，主要包括三元组顺序表和十字链表两种方法。

特殊矩阵的运算

矩阵转置

对于压缩存储的特殊矩阵，转置运算需要重新计算元素的位置。以对称矩阵为例，由于其对称性，转置后矩阵不变，因此不需要实际进行转置操作。

矩阵加法

两个相同类型的特殊矩阵相加，结果仍然是同类型的特殊矩阵。可以利用压缩存储的特性，只对非零元素进行运算。

矩阵乘法

特殊矩阵的乘法运算可以利用其结构特性进行优化。例如，三角矩阵与向量的乘法可以避免对零元素的运算。

特殊矩阵的应用场景

  • 数值分析：有限元方法、差分方法等数值计算中经常出现特殊矩阵
  • 图像处理：图像变换、滤波等操作中使用的变换矩阵
  • 机器学习：协方差矩阵、核矩阵等通常是对称矩阵
  • 图论：图的邻接矩阵、拉普拉斯矩阵等
  • 信号处理：傅里叶变换、小波变换等使用的变换矩阵
