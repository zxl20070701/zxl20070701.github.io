多维数组

数组的定义

数组是由n（n≥1）个相同类型的数据元素构成的有限序列，每个数据元素称为一个数组元素，每个元素在n个线性关系中的序号称为该元素的下标，下标的取值范围称为数组的维界。数组是线性表的推广，一维数组可以看作是一个线性表，二维数组可以看作是"数据元素是一维数组"的一维数组，三维数组可以看作是"数据元素是二维数组"的一维数组，以此类推。

数组的特点

  • 元素类型相同：数组中所有元素的数据类型必须相同
  • 下标有序：元素的下标具有固定的次序关系
  • 随机访问：可以通过下标直接访问任意元素
  • 结构固定：数组的维数和维界一旦确定，其结构就不变

一维数组

一维数组是最简单的数组形式，可以看作是一个线性表。设一维数组A[n] = {a₀, a₁, ..., aₙ₋₁}，每个元素占用L个存储单元，则元素aᵢ的存储地址为：

LOC(aᵢ) = LOC(a₀) + i × L

其中，LOC(a₀)是数组的起始地址，也称为基地址。

二维数组

行优先存储

行优先存储（Row-major order）是将数组元素按行向量排列，第i+1个行向量紧接在第i个行向量后面。对于二维数组A[m][n]，元素aᵢⱼ的存储地址为：

LOC(aᵢⱼ) = LOC(a₀₀) + (i × n + j) × L

列优先存储

列优先存储（Column-major order）是将数组元素按列向量排列，第j+1个列向量紧接在第j个列向量后面。对于二维数组A[m][n]，元素aᵢⱼ的存储地址为：

LOC(aᵢⱼ) = LOC(a₀₀) + (j × m + i) × L

存储示例

设二维数组A[3][4]，其逻辑结构为： 

A = [ a₀₀  a₀₁  a₀₂  a₀₃ ]
    [ a₁₀  a₁₁  a₁₂  a₁₃ ]
    [ a₂₀  a₂₁  a₂₂  a₂₃ ]

行优先存储顺序：a₀₀, a₀₁, a₀₂, a₀₃, a₁₀, a₁₁, a₁₂, a₁₃, a₂₀, a₂₁, a₂₂, a₂₃

列优先存储顺序：a₀₀, a₁₀, a₂₀, a₀₁, a₁₁, a₂₁, a₀₂, a₁₂, a₂₂, a₀₃, a₁₃, a₂₃

三维数组

三维数组可以看作是由多个二维数组组成的一维数组。设三维数组A[m][n][p]，元素aᵢⱼₖ的存储地址计算如下：

行优先存储

LOC(aᵢⱼₖ) = LOC(a₀₀₀) + (i × n × p + j × p + k) × L

列优先存储

LOC(aᵢⱼₖ) = LOC(a₀₀₀) + (k × m × n + j × m + i) × L

n维数组

对于n维数组A[b₁][b₂]...[bₙ]，其中bᵢ是第i维的长度，元素aᵢ₁ᵢ₂...ᵢₙ的存储地址可以推广为：

行优先存储

LOC(aᵢ₁ᵢ₂...ᵢₙ) = LOC(a₀₀...₀) + (i₁ × b₂ × b₃ × ... × bₙ + i₂ × b₃ × ... × bₙ + ... + iₙ₋₁ × bₙ + iₙ) × L

列优先存储

LOC(aᵢ₁ᵢ₂...ᵢₙ) = LOC(a₀₀...₀) + (iₙ × bₙ₋₁ × ... × b₁ + iₙ₋₁ × bₙ₋₂ × ... × b₁ + ... + i₂ × b₁ + i₁) × L

数组的存储结构

顺序存储结构

数组通常采用顺序存储结构，因为数组的维数和维界是固定的，且需要随机访问元素。顺序存储结构可以充分利用存储空间，实现随机访问。

数组的抽象数据类型 

ADT Array {
    数据对象：D = {aⱼ₁ⱼ₂...ⱼₙ | n(>0)称为数组的维数，bᵢ是数组第i维的长度，jᵢ是数组元素的第i维下标，aⱼ₁ⱼ₂...ⱼₙ ∈ ElemSet}

    数据关系：R = {R₁, R₂, ..., Rₙ}
        Rᵢ = {<aⱼ₁...ⱼᵢ...ⱼₙ, aⱼ₁...ⱼᵢ₊₁...ⱼₙ> | 0 ≤ jₖ ≤ bₖ-1, 1 ≤ k ≤ n且k≠i, 0 ≤ jᵢ ≤ bᵢ-2,
              aⱼ₁...ⱼᵢ...ⱼₙ, aⱼ₁...ⱼᵢ₊₁...ⱼₙ ∈ D, i = 2, ..., n}

    基本操作：
        InitArray(&A, n, bound₁, ..., boundₙ)
        DestroyArray(&A)
        Value(A, &e, index₁, ..., indexₙ)
        Assign(&A, e, index₁, ..., indexₙ)
}

数组的基本操作

数组初始化 InitArray(&A, n, bound₁, ..., boundₙ)

若维数n和各维长度合法，则构造相应的数组A，并返回OK。

销毁数组 DestroyArray(&A)

销毁数组A。

取值操作 Value(A, &e, index₁, ..., indexₙ)

A是n维数组，e为元素变量，随后是n个下标值。若各下标不超界，则e赋值为所指定的A的元素值，并返回OK。

赋值操作 Assign(&A, e, index₁, ..., indexₙ)

A是n维数组，e为元素变量，随后是n个下标值。若下标不超界，则将e的值赋给所指定的A的元素，并返回OK。

数组的应用

矩阵运算

二维数组是表示矩阵的自然数据结构，可以用于实现各种矩阵运算，如矩阵加法、矩阵乘法、矩阵转置等。

图像处理

数字图像可以表示为二维数组，其中每个元素表示一个像素的颜色值。三维数组可以用于表示彩色图像（RGB三个通道）。

科学计算

在科学计算中，多维数组用于存储和处理多维数据，如物理场、气象数据、金融数据等。

游戏开发

在游戏开发中，二维数组常用于表示游戏地图、棋盘等，三维数组可以用于表示三维游戏世界。
