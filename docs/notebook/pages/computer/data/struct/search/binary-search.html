二分查找

二分查找的定义

二分查找（Binary Search），又称折半查找，是一种高效的查找算法。它的基本思想是：在有序表中，取中间位置的元素与目标元素进行比较，如果相等则查找成功；如果目标元素小于中间元素，则在左半部分继续查找；如果目标元素大于中间元素，则在右半部分继续查找。重复这个过程，直到找到目标元素或确定目标元素不存在。

二分查找的前提条件

  • 有序性：数据必须是有序的
  • 随机访问：必须支持随机访问（如数组）
  • 稳定性：在查找过程中数据不会发生变化

二分查找的实现

递归实现 

int BinarySearch_Recursive(int arr[], int left, int right, int key) {
    if (left > right) {
        return -1;  // 查找失败
    }

    int mid = left + (right - left) / 2;  // 防止溢出

    if (arr[mid] == key) {
        return mid;  // 查找成功
    } else if (arr[mid] > key) {
        return BinarySearch_Recursive(arr, left, mid - 1, key);  // 在左半部分查找
    } else {
        return BinarySearch_Recursive(arr, mid + 1, right, key);  // 在右半部分查找
    }
}

非递归实现 

int BinarySearch_Iterative(int arr[], int n, int key) {
    int left = 0;
    int right = n - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;  // 防止溢出

        if (arr[mid] == key) {
            return mid;  // 查找成功
        } else if (arr[mid] > key) {
            right = mid - 1;  // 在左半部分查找
        } else {
            left = mid + 1;  // 在右半部分查找
        }
    }

    return -1;  // 查找失败
}

查找第一个等于目标值的元素 

int BinarySearch_FirstEqual(int arr[], int n, int key) {
    int left = 0;
    int right = n - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (arr[mid] >= key) {
            right = mid - 1;
        } else {
            left = mid + 1;
        }
    }

    if (left < n && arr[left] == key) {
        return left;
    }
    return -1;
}

查找最后一个等于目标值的元素 

int BinarySearch_LastEqual(int arr[], int n, int key) {
    int left = 0;
    int right = n - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (arr[mid] <= key) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    if (right >= 0 && arr[right] == key) {
        return right;
    }
    return -1;
}

二分查找的性能分析

时间复杂度

  • 最好情况：O(1)，目标元素正好是中间元素
  • 最坏情况：O(log n)，需要查找到最后一个元素
  • 平均情况：O(log n)

空间复杂度

  • 递归实现：O(log n)，递归调用栈的深度
  • 非递归实现：O(1)，只需要常数级别的额外空间

二分查找的优化

插值查找

插值查找是二分查找的改进，它根据目标值在范围中的位置来估计中间位置，而不是简单地取中间位置。

斐波那契查找

斐波那契查找利用斐波那契数列的性质来确定分割点，在某些情况下比二分查找更高效。

三分查找

三分查找将查找区间分成三部分，适用于某些特殊的函数优化问题。

二分查找的应用场景

有序数组查找

当数据存储在有序数组中且需要频繁查找时，二分查找是最佳选择。

查找边界

二分查找的变种可以用于查找有序数组中满足特定条件的边界。

数值计算

在数值计算中，二分查找可以用于求解方程的根。

优化问题

在某些优化问题中，二分查找可以用于寻找最优解。

int BinarySearch_FirstGreaterEqual(int arr[], int n, int key) {
    int left = 0;
    int right = n - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (arr[mid] >= key) {
            right = mid - 1;
        } else {
            left = mid + 1;

    return left;  // 返回第一个大于等于key的元素位置
}
