图的遍历

图遍历的基本概念

图的遍历（Graph Traversal）是指从图的某一顶点出发，对图中所有顶点访问一次且仅访问一次。图的遍历是图的基本运算，是求解图的连通性问题、拓扑排序、关键路径等问题的基础。

遍历的重要性

  • 连通性检查：判断图的连通性
  • 路径查找：寻找两点间的路径
  • 拓扑排序：对有向无环图进行拓扑排序
  • 最小生成树：寻找图的最小生成树

深度优先搜索（DFS）

基本概念

深度优先搜索（Depth-First Search，DFS）类似于树的先根遍历，是树的先根遍历的推广。它的基本思想是：从图中某个顶点v出发，访问此顶点，然后依次从v的未被访问的邻接点出发深度优先遍历图，直至图中所有和v有路径相通的顶点都被访问到；若此时图中尚有顶点未被访问，则另选图中一个未曾被访问的顶点作起始点，重复上述过程，直至图中所有顶点都被访问到为止。

DFS的实现（递归） 

#define MAX_VERTEX_NUM 100
int visited[MAX_VERTEX_NUM];  // 访问标志数组

void DFS(GraphAdjList G, int i) {
    EdgeNode *p;
    visited[i] = 1;
    printf("%c ", G.adjList[i].data);

    p = G.adjList[i].firstedge;
    while (p) {
        if (!visited[p->adjvex]) {
            DFS(G, p->adjvex);
        }
        p = p->next;
    }
}

void DFSTraverse(GraphAdjList G) {
    int i;
    for (i = 0; i < G.numVertexes; i++) {
        visited[i] = 0;  // 初始化访问标志数组
    }

    for (i = 0; i < G.numVertexes; i++) {
        if (!visited[i]) {
            DFS(G, i);
        }
    }
}

DFS的实现（非递归） 

void DFS_NonRecursive(GraphAdjList G, int i) {
    int j;
    EdgeNode *p;
    Stack S;
    InitStack(&S);

    printf("%c ", G.adjList[i].data);
    visited[i] = 1;
    Push(&S, i);

    while (!StackEmpty(S)) {
        Pop(&S, &i);
        p = G.adjList[i].firstedge;
        while (p) {
            j = p->adjvex;
            if (!visited[j]) {
                printf("%c ", G.adjList[j].data);
                visited[j] = 1;
                Push(&S, j);
                break;
            }
            p = p->next;
        }
        if (p == NULL && !StackEmpty(S)) {
            Pop(&S, &i);
        }
    }
}

广度优先搜索（BFS）

基本概念

广度优先搜索（Breadth-First Search，BFS）类似于树的按层次遍历的过程。它的基本思想是：从图中某个顶点v出发，访问此顶点，然后依次访问v的各个未被访问过的邻接点，然后分别从这些邻接点出发依次访问它们的邻接点，并使"先被访问的顶点的邻接点"先于"后被访问的顶点的邻接点"被访问，直至图中所有已被访问的顶点的邻接点都被访问到；若此时图中尚有顶点未被访问，则另选图中一个未曾被访问的顶点作起始点，重复上述过程，直至图中所有顶点都被访问到为止。

BFS的实现 

void BFS(GraphAdjList G, int i) {
    int j;
    EdgeNode *p;
    Queue Q;
    InitQueue(&Q);

    printf("%c ", G.adjList[i].data);
    visited[i] = 1;
    EnQueue(&Q, i);

    while (!QueueEmpty(Q)) {
        DeQueue(&Q, &i);
        p = G.adjList[i].firstedge;
        while (p) {
            j = p->adjvex;
            if (!visited[j]) {
                printf("%c ", G.adjList[j].data);
                visited[j] = 1;
                EnQueue(&Q, j);
            }
            p = p->next;
        }
    }
}

void BFSTraverse(GraphAdjList G) {
    int i;
    for (i = 0; i < G.numVertexes; i++) {
        visited[i] = 0;  // 初始化访问标志数组
    }

    for (i = 0; i < G.numVertexes; i++) {
        if (!visited[i]) {
            BFS(G, i);
        }
    }
}

DFS与BFS的比较

特性 DFS BFS
数据结构 栈（递归或显式） 队列
遍历顺序 深度优先 广度优先
空间复杂度 O(h) O(w)
时间复杂度 O(V+E) O(V+E)
适用场景 寻找路径、连通分量 最短路径、层次遍历

其中，h是图的高度，w是图的最大宽度，V是顶点数，E是边数。

图遍历的应用

连通分量

通过一次遍历可以找到一个连通分量中的所有顶点。

拓扑排序

对有向无环图进行拓扑排序。

最短路径

BFS可以用于求解无权图的最短路径问题。

最小生成树

DFS和BFS都可以用于生成图的生成树。

图遍历的复杂度分析

时间复杂度

  • 邻接矩阵存储：O(V²)
  • 邻接表存储：O(V+E)

空间复杂度

  • 邻接矩阵存储：O(V)
  • 邻接表存储：O(V)

图遍历的优化

双端队列BFS

使用双端队列可以实现0-1权值图的最短路径算法。

A*算法

在BFS基础上加入启发式函数，可以加速路径搜索。

迭代加深DFS

结合了DFS的空间效率和BFS的完备性，适用于状态空间较大的问题。
