网络

网络配置查看

ifconfig

查看网络接口配置信息： 

    ifconfig

查看特定网络接口（如 en0）： 

    ifconfig en0

networksetup

列出所有网络服务： 

    networksetup -listallnetworkservices

查看特定服务的详细信息： 

    networksetup -getinfo "Wi-Fi"

网络连接测试

ping

测试网络连通性： 

    ping www.baidu.com

指定 ping 次数： 

    ping -c 4 www.baidu.com

traceroute

跟踪数据包路径： 

    traceroute www.baidu.com

端口和网络服务

lsof

查看端口占用情况： 

    lsof -i :8080

查看所有监听端口： 

    lsof -i -P | grep LISTEN

netstat

查看网络连接状态： 

    netstat -an | grep LISTEN
