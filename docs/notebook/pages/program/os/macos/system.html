系统信息

系统基本信息

system_profiler

查看详细的系统硬件和软件信息： 

    system_profiler

查看特定类型的信息（如硬件）： 

    system_profiler SPHardwareDataType

sw_vers

查看 macOS 版本信息： 

    sw_vers

硬件信息

sysctl

查看 CPU 信息： 

    sysctl -n machdep.cpu.brand_string

查看内存信息： 

    sysctl -n hw.memsize

system_profiler 硬件信息

查看 CPU 信息： 

    system_profiler SPHardwareDataType | grep "Processor Name"

查看内存信息： 

    system_profiler SPHardwareDataType | grep "Memory"

磁盘和存储

df

查看磁盘使用情况： 

    df -h

diskutil

查看磁盘列表： 

    diskutil list

查看特定磁盘信息： 

    diskutil info disk0

系统运行状态

uptime

查看系统运行时间和负载： 

    uptime

top

实时查看系统资源使用情况： 

    top
