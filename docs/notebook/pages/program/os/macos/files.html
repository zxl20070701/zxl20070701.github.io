文件操作

文件和目录操作

基本文件操作

创建目录： 

    mkdir new_folder

创建多级目录： 

    mkdir -p parent/child/grandchild

复制文件： 

    cp source.txt destination.txt

复制目录： 

    cp -r source_folder destination_folder

移动或重命名文件： 

    mv old_name.txt new_name.txt

删除文件： 

    rm file.txt

删除目录： 

    rm -r folder_name

查找文件

使用 find 命令查找文件： 

    find /path/to/search -name "filename.txt"

查找特定类型的文件： 

    find . -name "*.txt"

使用 locate 命令快速查找（需要先更新数据库）： 

sudo /usr/libexec/locate.updatedb
locate filename.txt

文件内容查看和编辑

cat

查看文件内容： 

    cat file.txt

less 和 more

分页查看文件内容： 

less large_file.txt
more large_file.txt

head 和 tail

查看文件开头部分： 

    head -n 20 file.txt

查看文件末尾部分： 

    tail -n 20 file.txt

实时查看文件更新： 

    tail -f log_file.txt

权限和属性

chown

更改文件所有者： 

    sudo chown username file.txt

chmod

更改文件权限： 

chmod 755 script.sh
chmod +x script.sh
