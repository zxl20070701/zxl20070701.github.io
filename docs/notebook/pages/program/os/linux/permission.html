权限管理

文件权限基础

ls -l

查看文件权限： 

    ls -l file.txt

查看目录权限： 

    ls -ld directory/

权限表示法

权限由三组字符表示：所有者、组、其他用户 

-rwxr-xr-x  1 user group  1234 Jan 1 12:00 file.txt

权限字符含义： 

r - 读权限 (read)
w - 写权限 (write)
x - 执行权限 (execute)
- - 无权限

修改权限

chmod

使用数字方式修改权限： 

    chmod 755 file.txt

权限数字对应关系： 

7 = rwx (4+2+1)
6 = rw- (4+2)
5 = r-x (4+1)
4 = r-- (4)
0 = --- (0)

使用符号方式修改权限： 

chmod u+x file.txt    # 给所有者添加执行权限
chmod g-w file.txt    # 移除组的写权限
chmod o=r file.txt    # 设置其他用户只有读权限
chmod a+r file.txt    # 给所有用户添加读权限

递归修改权限

递归修改目录及其内容权限： 

    chmod -R 755 directory/

文件所有者和组

chown

修改文件所有者： 

    chown username file.txt

同时修改所有者和组： 

    chown username:groupname file.txt

只修改组： 

    chown :groupname file.txt

chgrp

修改文件组（专门用于修改组）： 

    chgrp groupname file.txt

递归修改

递归修改目录及其内容的所有者和组： 

    chown -R username:groupname directory/

特殊权限

SUID (Set User ID)

设置 SUID 权限： 

    chmod u+s file.txt

SGID (Set Group ID)

设置 SGID 权限： 

    chmod g+s file.txt

设置目录的 SGID，使新建文件继承目录的组： 

    chmod g+s directory/

Sticky Bit

设置粘滞位（通常用于 /tmp 目录）： 

    chmod +t directory/

查看和修改默认权限

umask

查看当前 umask： 

    umask

设置 umask： 

    umask 022

umask 值与默认权限的关系： 

文件默认权限 = 666 - umask
目录默认权限 = 777 - umask
