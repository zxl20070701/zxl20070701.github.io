系统信息

系统基本信息

uname

查看系统内核信息： 

    uname -a

查看内核版本： 

    uname -r

lsb_release

查看 Linux 发行版信息： 

    lsb_release -a

cat /etc/os-release

查看操作系统详细信息： 

    cat /etc/os-release

硬件信息

lscpu

查看 CPU 信息： 

    lscpu

free

查看内存使用情况： 

    free -h

lsblk

查看块设备信息： 

    lsblk

lspci

查看 PCI 设备： 

    lspci

lsusb

查看 USB 设备： 

    lsusb

磁盘和存储

df

查看磁盘使用情况： 

    df -h

du

查看目录大小： 

    du -sh /path/to/directory

fdisk

查看磁盘分区： 

    sudo fdisk -l

系统运行状态

uptime

查看系统运行时间和负载： 

    uptime

top / htop

实时查看系统资源使用情况： 

top
# 或者安装 htop
htop

vmstat

查看虚拟内存统计： 

    vmstat 1 5
