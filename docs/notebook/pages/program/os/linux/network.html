网络

网络配置查看

ip addr / ifconfig

查看网络接口配置信息： 

ip addr show
# 或者
ifconfig

查看特定网络接口： 

ip addr show eth0
# 或者
ifconfig eth0

ip route

查看路由表： 

    ip route show

网络连接测试

ping

测试网络连通性： 

    ping www.baidu.com

指定 ping 次数： 

    ping -c 4 www.baidu.com

traceroute

跟踪数据包路径： 

    traceroute www.baidu.com

mtr

网络诊断工具（需要安装）： 

    mtr www.baidu.com

端口和网络服务

netstat

查看网络连接状态： 

    netstat -tuln

查看监听端口： 

    netstat -tuln | grep LISTEN

ss

更现代的网络连接查看工具： 

    ss -tuln

lsof

查看端口占用情况： 

    lsof -i :8080

DNS 相关

nslookup

查询 DNS 记录： 

    nslookup www.baidu.com

dig

更详细的 DNS 查询工具： 

    dig www.baidu.com

host

简单的 DNS 查询： 

    host www.baidu.com
