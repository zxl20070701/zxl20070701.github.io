基本语法

定义

比如我们要开发一个插件 DemoPlugin ，直接新建文件 demp-plugin.js后写入如下代码： 

class DemoPlugin{
    constructor(options) {

        // 用户传递的参数
        this.options = options;
    }
    apply(compiler) {
        // 在这里写具体的业务代码
    }
}
module.exports = DemoPlugin;

使用

定义好了以后，就和普通的插件一样使用即可，比如在 webpack.config.js 文件中： 

const DemoPlugin = require('./demo-plugin.js');

module.exports = {
    plugins: [
        new DemoPlugin({
            // 也就是options
        })
    ]
};
