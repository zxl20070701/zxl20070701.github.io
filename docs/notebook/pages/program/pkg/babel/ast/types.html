@babel/types

官方文档地址： https://babel.dev/docs/babel-types

提供了一套用于创建和判断 AST 节点的工具，在遍历和修改 AST 时，常常需要使用这些工具来创建新节点或检查节点类型。

节点创建

  • types.identifier(name) - 创建标识符节点
  • types.stringLiteral(value) - 创建字符串字面量节点
  • types.numericLiteral(value) - 创建数值字面量节点
  • types.booleanLiteral(value) - 创建布尔字面量节点
  • types.nullLiteral() - 创建空字面量节点

对象创建

  • types.objectExpression(properties) - 创建对象表达式节点
  • types.objectProperty(key, value) - 创建对象属性节点
  • types.objectMethod(kind, key, params, body) - 创建对象方法节点

数组创建

  • types.arrayExpression(elements) - 创建数组表达式节点

函数创建

  • types.functionDeclaration(id, params, body) - 创建函数声明节点
  • types.functionExpression(id, params, body) - 创建函数表达式节点
  • types.arrowFunctionExpression(params, body, async) - 创建箭头函数表达式节点

表达式创建

  • types.binaryExpression(operator, left, right) - 创建二元表达式节点
