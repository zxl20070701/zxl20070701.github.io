AST 基础概念

什么是AST

AST（Abstract Syntax Tree，抽象语法树）是源代码的树形表示，它以树状结构表示编程语言的语法结构。每个节点代表源代码中的一个构造，如表达式、语句、声明等。

AST的作用

  • 代码转换：如Babel将ES6+代码转换为ES5
  • 代码分析：静态分析、linting、格式化
  • 代码优化：压缩、混淆
  • 代码生成：从AST生成目标代码

AST节点类型

AST中的节点有多种类型，常见的包括：

  • Program：整个程序的根节点
  • VariableDeclaration：变量声明
  • FunctionDeclaration：函数声明
  • ExpressionStatement：表达式语句
  • CallExpression：函数调用表达式
  • BinaryExpression：二元表达式

AST结构示例

以简单的JavaScript代码为例： 

const x = 1 + 2;

对应的AST结构大致如下： 

{
  "type": "Program",
  "body": [
    {
      "type": "VariableDeclaration",
      "declarations": [
        {
          "type": "VariableDeclarator",
          "id": {
            "type": "Identifier",
            "name": "x"
          },
          "init": {
            "type": "BinaryExpression",
            "operator": "+",
            "left": {
              "type": "Literal",
              "value": 1
            },
            "right": {
              "type": "Literal",
              "value": 2
            }
          }
        }
      ],
      "kind": "const"
    }
  ]
}
