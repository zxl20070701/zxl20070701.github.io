@babel/parser - 代码解析器

什么是@babel/parser

@babel/parser（原名babylon）是Babel的JavaScript解析器，用于将JavaScript源代码转换为抽象语法树（AST）。

安装与使用 

npm install --save-dev @babel/parser

const parser = require('@babel/parser');

const code = 'const x = 1 + 2;';
const ast = parser.parse(code);

console.log(ast);

解析选项

@babel/parser提供了多种选项来控制解析行为： 

const ast = parser.parse(code, {
  // 启用JSX语法
  plugins: ['jsx'],
  
  // 启用严格模式
  strictMode: true,
  
  // 启用装饰器语法
  plugins: ['decorators-legacy'],
  
  // 启用TypeScript
  plugins: ['typescript']
});

常用插件

  • jsx：支持JSX语法
  • typescript：支持TypeScript语法
  • decorators-legacy：支持装饰器语法
  • classProperties：支持类属性
  • asyncGenerators：支持异步生成器

parseExpression方法

如果只需要解析单个表达式，可以使用parseExpression方法： 

const expression = parser.parseExpression('1 + 2');
console.log(expression);

错误处理

当代码包含语法错误时，@babel/parser会抛出错误： 

try {
  const ast = parser.parse('const x = ;'); // 语法错误
} catch (error) {
  console.error('解析错误:', error.message);
}

实际应用示例

下面是一个完整的示例，展示如何使用@babel/parser解析代码并检查变量声明： 

const parser = require('@babel/parser');
const traverse = require('@babel/traverse').default;

const code = `
  const x = 1;
  let y = 2;
  var z = 3;
  const sum = x + y + z;
`;

const ast = parser.parse(code);

let variableCount = 0;
traverse(ast, {
  VariableDeclarator(path) {
    variableCount++;
    console.log(`找到变量: ${path.node.id.name}`);
  }
});

console.log(`总共找到 ${variableCount} 个变量声明`);
