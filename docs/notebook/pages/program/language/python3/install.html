如何搭建环境和运行？

安装

如果你已经安装过python或系统内置有，可以先查看版本： 

    python --version

如果版本不对或没有安装，可以去 https://www.python.org 下载后进行安装。

HelloWorld

文件名称：index.py 

#!/usr/bin/python3

print("你好呀！")

运行

执行下面命令即可运行： 

    python index.py

如果你看见类似下面的报错：

SyntaxError: Non-ASCII character '\xe4' in file index.py on line 3, but no encoding declared;

可以在文件开头（其实应该是第二行）添加下面语句： 

# coding=utf-8
