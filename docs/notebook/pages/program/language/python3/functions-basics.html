函数基础概念

函数是Python中的基本构建块，用于封装可重用的代码。函数可以接受参数并返回值。

语法 

def 函数名（参数列表）:
    函数体

函数定义和调用

基本函数定义 

def greet(name):
    """这是一个简单的问候函数"""
    return f"Hello, {name}!"

message = greet("Alice")
print(message)  # 输出: Hello, Alice!

没有返回值的函数 

def print_info(name, age):
    print(f"姓名: {name}, 年龄: {age}")

print_info("Bob", 25)  # 输出: 姓名: Bob, 年龄: 25

返回多个值的函数 

def get_name_and_age():
    return "Charlie", 30

name, age = get_name_and_age()
print(name, age)  # 输出: Charlie 30
