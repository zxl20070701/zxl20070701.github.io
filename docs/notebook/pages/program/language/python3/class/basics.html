类与对象基础

什么是类与对象？

在Python中，类（Class）是创建对象的蓝图或模板，而对象（Object）是类的实例。类定义了一组属性（数据）和方法（函数），对象则根据这些定义创建并拥有自己的具体值。

类的定义

使用class关键字来定义一个类： 

class MyClass:
    """这是一个简单的类"""
    
    # 类属性
    class_variable = "这是一个类属性"
    
    # 初始化方法
    def __init__(self, name):
        # 实例属性
        self.name = name
    
    # 实例方法
    def greet(self):
        return f"Hello, {self.name}!"

创建对象

通过调用类名并传递必要的参数来创建对象： 

# 创建对象
obj1 = MyClass("Alice")
obj2 = MyClass("Bob")

# 访问属性
print(obj1.name)  # 输出: Alice
print(obj2.name)  # 输出: Bob

# 调用方法
print(obj1.greet())  # 输出: Hello, Alice!
print(obj2.greet())  # 输出: Hello, Bob!

类属性与实例属性

类属性

类属性是定义在类中但在任何方法之外的变量，所有类的实例共享这个属性： 

class Dog:
    # 类属性
    species = "Canis familiaris"
    
    def __init__(self, name):
        self.name = name

# 所有Dog实例共享species属性
dog1 = Dog("Buddy")
dog2 = Dog("Lucy")

print(dog1.species)  # 输出: Canis familiaris
print(dog2.species)  # 输出: Canis familiaris
print(Dog.species)   # 输出: Canis familiaris

实例属性

实例属性是属于特定对象的属性，通过self在方法中定义： 

class Person:
    def __init__(self, name, age):
        # 实例属性
        self.name = name
        self.age = age
    
    def introduce(self):
        return f"My name is {self.name}, I'm {self.age} years old."

person = Person("Alice", 30)
print(person.name)  # 输出: Alice
print(person.age)   # 输出: 30

访问控制

公有属性和方法

默认情况下，Python中的所有属性和方法都是公有的，可以从类的外部访问： 

class Example:
    def __init__(self):
        self.public_var = "这是公有属性"
    
    def public_method(self):
        return "这是公有方法"

obj = Example()
print(obj.public_var)      # 可以访问
print(obj.public_method()) # 可以调用

私有属性和方法

通过在名称前添加双下划线__来定义私有属性和方法： 

class Example:
    def __init__(self):
        self.public_var = "这是公有属性"
        self.__private_var = "这是私有属性"
    
    def public_method(self):
        return "这是公有方法"
    
    def __private_method(self):
        return "这是私有方法"
    
    def access_private(self):
        # 在类内部可以访问私有成员
        return self.__private_var + " - " + self.__private_method()

obj = Example()
print(obj.public_var)           # 可以访问
print(obj.public_method())      # 可以调用
# print(obj.__private_var)      # 错误！不能直接访问
# print(obj.__private_method()) # 错误！不能直接调用
print(obj.access_private())     # 通过公有方法访问私有成员
注意：Python中的私有成员实际上是通过名称改写（name mangling）实现的，技术上仍然可以从外部访问，但不推荐这样做。

类变量与实例变量的区别 

class MyClass:
    # 类变量
    class_var = "类变量"
    
    def __init__(self, instance_var):
        # 实例变量
        self.instance_var = instance_var

# 创建实例
obj1 = MyClass("实例1")
obj2 = MyClass("实例2")

# 访问类变量
print(MyClass.class_var)  # 输出: 类变量
print(obj1.class_var)     # 输出: 类变量
print(obj2.class_var)     # 输出: 类变量

# 修改类变量
MyClass.class_var = "修改后的类变量"
print(obj1.class_var)     # 输出: 修改后的类变量
print(obj2.class_var)     # 输出: 修改后的类变量

# 为实例添加与类变量同名的实例变量
obj1.class_var = "实例1的类变量"
print(obj1.class_var)     # 输出: 实例1的类变量
print(obj2.class_var)     # 输出: 修改后的类变量
print(MyClass.class_var)  # 输出: 修改后的类变量
