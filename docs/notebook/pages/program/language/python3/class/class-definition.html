类的定义

使用class关键字定义类

使用class关键字来定义一个类： 

class MyClass:
    """这是一个简单的类"""
    
    # 类属性
    class_variable = "这是一个类属性"
    
    # 初始化方法
    def __init__(self, name):
        # 实例属性
        self.name = name
    
    # 实例方法
    def greet(self):
        return f"Hello, {self.name}!"

类定义的基本结构

  • 类名：遵循大驼峰命名法（PascalCase）
  • 文档字符串：描述类的用途和功能
  • 属性：类变量和实例变量
  • 方法：类中定义的函数

命名规范 

# 正确的类命名
class StudentInfo:
    pass

class BankAccount:
    pass

# 不推荐的下划线命名
class student_info:
    pass

class bank_account:
    pass
