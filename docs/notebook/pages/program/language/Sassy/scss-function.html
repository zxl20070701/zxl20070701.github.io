Sassy CSS 内置函数

颜色操作

lighten

使颜色变淡：

lighten($color, $amount)

例如：

color: lighten(#6b717f, 20%);

darken

使颜色变暗：

darken($color, $amount)

例如：

color: darken(#6b717f, 20%);

saturate

增加颜色的饱和度：

saturate($color, $amount)

例如：

color: saturate(#6b717f, 20%);

desaturate

降低颜色的饱和度：

desaturate($color, $amount)

例如：

color: desaturate(#6b717f, 20%);

adjust-hue

调整颜色的色相：

adjust-hue($color, $degrees)

例如：

color: adjust-hue(#6b717f, 180deg);

rgba

设置颜色的透明度：

rgba($color, $alpha)

例如：

color: rgba(#6b717f, 0.5);

mix

混合两种颜色：

mix($color1, $color2, $weight: 50%)

例如：

color: mix(#6b717f, #ffffff, 30%);

complement

返回颜色的补色：

complement($color)

例如：

color: complement(#6b717f);

invert

返回颜色的反色：

invert($color, $weight: 100%)

例如：

color: invert(#6b717f);

字符串操作

quote

给字符串添加引号：

quote($string)

例如：

content: quote(Hello World);

unquote

移除字符串的引号：

unquote($string)

例如：

font-family: unquote("Arial, sans-serif");

str-length

返回字符串的长度：

str-length($string)

例如：

width: str-length("Hello") * 10px;

str-insert

在字符串指定位置插入子字符串：

str-insert($string, $insert, $index)

例如：

content: str-insert("Hello", " World", 6);

str-index

返回子字符串在字符串中的索引位置：

str-index($string, $substring)

例如：

position: str-index("Hello World", "World");

str-slice

提取字符串的片段：

str-slice($string, $start-at, $end-at: -1)

例如：

content: str-slice("Hello World", 1, 5);

数字操作

percentage

将无单位数字转换为百分比：

percentage($number)

例如：

width: percentage(0.5);

round

四舍五入数字：

round($number)

例如：

width: round(15.6px);

ceil

向上取整：

ceil($number)

例如：

width: ceil(15.2px);

floor

向下取整：

floor($number)

例如：

width: floor(15.8px);

abs

返回数字的绝对值：

abs($number)

例如：

margin: abs(-10px);

min

返回一组数字中的最小值：

min($numbers...)

例如：

width: min(100px, 200px, 150px);

max

返回一组数字中的最大值：

max($numbers...)

例如：

width: max(100px, 200px, 150px);

random

返回随机数：

random($limit: null)

例如：

opacity: random();

列表操作

length

返回列表的长度：

length($list)

例如：

items: length(1px 2px 3px);

nth

返回列表中的第n个元素：

nth($list, $n)

例如：

margin: nth(10px 20px 30px, 2);

append

向列表添加元素：

append($list, $val, $separator: auto)

例如：

list: append(10px 20px, 30px);

join

合并两个列表：

join($list1, $list2, $separator: auto, $bracketed: auto)

例如：

list: join(10px 20px, 30px 40px);

zip

将多个列表合并为一个多维列表：

zip($lists...)

例如：

list: zip(1px 2px 3px, solid dashed dotted, red green blue);

index

返回元素在列表中的位置：

index($list, $value)

例如：

position: index(1px 2px 3px, 2px);

list-separator

返回列表的分隔符：

list-separator($list)

例如：

separator: list-separator(1px, 2px, 3px);

映射操作

map-get

从映射中获取值：

map-get($map, $key)

例如：

$font-weights: ("regular": 400, "medium": 500, "bold": 700);
font-weight: map-get($font-weights, bold);

map-merge

合并两个映射：

map-merge($map1, $map2)

例如：

$map1: ("a": 1, "b": 2);
$map2: ("c": 3, "d": 4);
$merged: map-merge($map1, $map2);

map-remove

从映射中移除键值对：

map-remove($map, $keys...)

例如：

$font-weights: map-remove($font-weights, "regular");

map-keys

返回映射中所有键的列表：

map-keys($map)

例如：

keys: map-keys(("a": 1, "b": 2, "c": 3));

map-values

返回映射中所有值的列表：

map-values($map)

例如：

values: map-values(("a": 1, "b": 2, "c": 3));

map-has-key

检查映射是否包含指定的键：

map-has-key($map, $key)

例如：

has-bold: map-has-key($font-weights, bold);

选择器操作

selector-append

将选择器附加到其他选择器：

selector-append($selectors...)

例如：

selector: selector-append(".btn", ":hover");

selector-extend

扩展选择器：

selector-extend($selector, $extendee, $extender)

例如：

selector: selector-extend(".btn", ".btn-primary", ".btn-large");

selector-replace

替换选择器中的部分：

selector-replace($selector, $original, $replacement)

例如：

selector: selector-replace(".btn-primary", "primary", "secondary");

selector-nest

嵌套选择器：

selector-nest($selectors...)

例如：

selector: selector-nest(".card", ".header", "h1");

selector-parse

解析选择器字符串：

selector-parse($selector)

例如：

parsed: selector-parse(".btn.btn-primary:hover");

is-superselector

检查一个选择器是否匹配另一个选择器的所有元素：

is-superselector($super, $sub)

例如：

result: is-superselector(".btn", ".btn-primary");

simple-selectors

将复杂选择器分解为简单选择器：

simple-selectors($selector)

例如：

selectors: simple-selectors(".btn.btn-primary:hover");

自检函数

feature-exists

检查某个特性是否存在：

feature-exists($feature)

例如：

has-feature: feature-exists("global-variable-shadowing");

variable-exists

检查变量是否存在：

variable-exists($name)

例如：

exists: variable-exists(my-variable);

global-variable-exists

检查全局变量是否存在：

global-variable-exists($name)

例如：

exists: global-variable-exists(my-global-variable);

function-exists

检查函数是否存在：

function-exists($name)

例如：

exists: function-exists(my-function);

mixin-exists

检查混合宏是否存在：

mixin-exists($name)

例如：

exists: mixin-exists(my-mixin);

其他实用函数

if

条件判断函数：

if($condition, $if-true, $if-false)

例如：

color: if($theme == dark, #fff, #000);

unique-id

生成唯一标识符：

unique-id()

例如：

id: unique-id();

call

动态调用函数：

call($name, $args...)

例如：

result: call(lighten, #000, 50%);
