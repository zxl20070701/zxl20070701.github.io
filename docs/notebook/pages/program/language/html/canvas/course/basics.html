什么是Canvas？

HTML5 Canvas 是一个可以使用JavaScript绘制图形的HTML元素。它提供了一个空白的画布，开发者可以在上面绘制各种图形、文字、图像等。

Canvas的基本结构

Canvas元素本身只是一个容器，所有的绘图工作都需要通过JavaScript来完成： 


    您的浏览器不支持Canvas

获取Canvas上下文

要在Canvas上绘图，首先需要获取绘图上下文： 

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d'); // 获取2D绘图上下文

Canvas坐标系统

Canvas使用一个二维坐标系统：

  • 原点(0,0)在Canvas的左上角
  • X轴向右增加
  • Y轴向下增加

坐标单位是像素(px)。

Canvas常用属性

属性 说明
width Canvas的宽度，默认为300像素
height Canvas的高度，默认为150像素
id Canvas的唯一标识符

浏览器兼容性

虽然现代浏览器都支持Canvas，但在使用时最好检查浏览器是否支持： 
if (canvas.getContext) {
    // 支持Canvas
    const ctx = canvas.getContext('2d');
} else {
    // 不支持Canvas，提供降级方案
}
