Shadow DOM详解

什么是Shadow DOM

Shadow DOM是Web Components技术套件的一部分，它提供了一种将隐藏的、独立的DOM树附加到元素上的方法。Shadow DOM允许你将标记结构、样式和行为隐藏起来，与页面上的其他代码隔离开来。

Shadow DOM的核心思想是封装，它允许组件开发者隐藏HTML结构、CSS样式和行为，防止与主文档冲突。

创建Shadow DOM

要为一个元素创建Shadow DOM，你需要调用attachShadow()方法： 

class MyElement extends HTMLElement {
    constructor() {
        super();
        // 创建Shadow DOM
        const shadow = this.attachShadow({mode: 'open'});
        
        // 添加内容到Shadow DOM
        shadow.innerHTML = `
            
            
这是Shadow DOM中的内容

        `;
    }
}

customElements.define('my-element', MyElement);

Shadow DOM的模式

Shadow DOM有两种模式：open和closed：

  • open ：可以通过JavaScript访问Shadow DOM（通过element.shadowRoot）
  • closed ：无法通过JavaScript从外部访问Shadow DOM

open模式示例 

class OpenElement extends HTMLElement {
    constructor() {
        super();
        this.attachShadow({mode: 'open'});
        this.shadowRoot.innerHTML = '
Open模式
';
    }
}

customElements.define('open-element', OpenElement);

// 可以访问Shadow DOM
const openEl = document.querySelector('open-element');
console.log(openEl.shadowRoot); // 返回ShadowRoot对象

closed模式示例 

class ClosedElement extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'closed'});
        shadow.innerHTML = '
Closed模式
';
    }
}

customElements.define('closed-element', ClosedElement);

// 无法访问Shadow DOM
const closedEl = document.querySelector('closed-element');
console.log(closedEl.shadowRoot); // 返回null

Shadow DOM的样式封装

Shadow DOM最重要的特性之一是样式封装。Shadow DOM内部的样式不会影响外部，外部的样式也不会影响内部。 

class StyledElement extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            
组件标题

            
        `;
    }
}

customElements.define('styled-element', StyledElement);

:host伪类选择器

:host伪类选择器允许你从Shadow DOM内部选择自定义元素本身： 

:host {
    display: block;
    background: #f0f0f0;
}

:host([disabled]) {
    opacity: 0.5;
}

:host(.active) {
    border-color: blue;
}

::slotted伪元素

::slotted()伪元素允许你为插入到slot中的内容设置样式： 

::slotted(*) {
    margin: 0;
}

::slotted(p) {
    color: blue;
}

::slotted(.highlight) {
    background-color: yellow;
}

事件重定向

在Shadow DOM中触发的事件会被重定向，使得事件看起来是从自定义元素本身触发的，而不是从Shadow DOM内部触发的： 

class EventElement extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
        `;
        
        const button = shadow.getElementById('inner-btn');
        button.addEventListener('click', (event) => {
            console.log('按钮被点击了');
            console.log('事件目标:', event.target); // 指向按钮
            console.log('当前目标:', event.currentTarget); // 指向按钮
        });
    }
}

customElements.define('event-element', EventElement);

// 监听自定义元素上的点击事件
document.querySelector('event-element').addEventListener('click', (event) => {
    console.log('自定义元素被点击了');
    console.log('事件目标:', event.target); // 指向event-element
    console.log('当前目标:', event.currentTarget); // 指向event-element
});
