样式封装与CSS变量

Web Components中的样式封装

Web Components的一个重要特性是样式封装。通过Shadow DOM，组件的样式被完全隔离，不会影响到外部文档，也不会被外部样式所影响。

这种封装机制确保了组件的可复用性和一致性，避免了样式冲突。

组件内部样式

在Web Components中，你可以在Shadow DOM内部定义样式： 

class StyledComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            
组件标题

            

                
这是组件的内容

                
            

        `;
    }
}

customElements.define('styled-component', StyledComponent);

CSS自定义属性（CSS变量）

CSS自定义属性（也称为CSS变量）是Web Components中实现主题化和可定制性的强大工具。它们允许用户从外部自定义组件的外观，而无需修改组件的内部样式。

定义CSS变量 

class ThemableComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            
可主题化组件

            

                
这是一个支持主题化的组件

                
                
            

        `;
    }
}

customElements.define('themable-component', ThemableComponent);

使用CSS变量自定义组件

使用组件时，可以通过CSS变量来自定义外观：

动态更新CSS变量

可以通过JavaScript动态更新CSS变量： 

class DynamicThemeComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            
动态主题组件

            

                
通过下面的控件可以实时调整组件样式

                
                

                    

                        
                        
                    

                    
                    

                        
                        
                    

                    
                    

                        
                        
                    

                    
                    

                        
                        
                        4px
                    

                    
                    

                        
                        
                        16px
                    

                    
                    
                

            

        `;
        
        // 绑定事件
        this.bindEvents();
    }
    
    bindEvents() {
        const shadow = this.shadowRoot;
        
        // 颜色选择器事件
        shadow.getElementById('primary-color').addEventListener('input', (e) => {
            this.style.setProperty('--primary-color', e.target.value);
        });
        
        shadow.getElementById('background-color').addEventListener('input', (e) => {
            this.style.setProperty('--background-color', e.target.value);
        });
        
        shadow.getElementById('text-color').addEventListener('input', (e) => {
            this.style.setProperty('--text-color', e.target.value);
        });
        
        // 滑块事件
        shadow.getElementById('border-radius').addEventListener('input', (e) => {
            const value = e.target.value + 'px';
            this.style.setProperty('--border-radius', value);
            shadow.getElementById('radius-value').textContent = value;
        });
        
        shadow.getElementById('padding').addEventListener('input', (e) => {
            const value = e.target.value + 'px';
            this.style.setProperty('--padding', value);
            shadow.getElementById('padding-value').textContent = value;
        });
        
        // 重置按钮
        shadow.getElementById('reset-btn').addEventListener('click', () => {
            this.style.setProperty('--primary-color', '#007acc');
            this.style.setProperty('--background-color', '#ffffff');
            this.style.setProperty('--text-color', '#333333');
            this.style.setProperty('--border-radius', '4px');
            this.style.setProperty('--padding', '16px');
            
            // 重置表单值
            shadow.getElementById('primary-color').value = '#007acc';
            shadow.getElementById('background-color').value = '#ffffff';
            shadow.getElementById('text-color').value = '#333333';
            shadow.getElementById('border-radius').value = '4';
            shadow.getElementById('padding').value = '16';
            shadow.getElementById('radius-value').textContent = '4px';
            shadow.getElementById('padding-value').textContent = '16px';
        });
    }
}

customElements.define('dynamic-theme-component', DynamicThemeComponent);

主题预设

创建预定义的主题类，让用户可以轻松切换主题： 

class ThemePresetsComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            
主题预设组件

            

                
这是一个支持多种预设主题的组件。点击下方按钮可以切换不同的主题。

                
当前主题会根据选择的主题自动调整颜色、背景等样式。

            

            
            

                
                
                
                
                
            

        `;
        
        this.bindEvents();
    }
    
    bindEvents() {
        const shadow = this.shadowRoot;
        const buttons = shadow.querySelectorAll('.theme-btn');
        
        buttons.forEach(button => {
            button.addEventListener('click', () => {
                // 移除所有主题类
                this.classList.remove('theme-dark', 'theme-success', 'theme-warning', 'theme-danger');
                
                // 添加选中的主题类
                const theme = button.getAttribute('data-theme');
                if (theme !== 'default') {
                    this.classList.add(theme);
                }
                
                // 更新按钮状态
                buttons.forEach(btn => btn.classList.remove('active'));
                button.classList.add('active');
            });
        });
    }
}

customElements.define('theme-presets-component', ThemePresetsComponent);
