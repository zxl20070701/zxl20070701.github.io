模板和插槽

HTML模板

HTML模板是Web Components的重要组成部分，它允许你定义可重用的HTML结构。<template>元素中的内容不会被渲染，但可以被JavaScript克隆和使用。

在Web Components中使用模板

在自定义元素中使用模板： 

class CardComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        // 获取模板
        const template = document.getElementById('my-template');
        const templateContent = template.content.cloneNode(true);
        
        // 设置内容
        const title = templateContent.querySelector('.card-title');
        const content = templateContent.querySelector('.card-content');
        
        title.textContent = this.getAttribute('title') || '默认标题';
        content.textContent = this.getAttribute('content') || '默认内容';
        
        shadow.appendChild(templateContent);
    }
}

customElements.define('card-component', CardComponent);

插槽（Slots）

插槽是Web Components中的一个强大特性，它允许你在组件中定义可插入内容的位置。元素作为占位符，可以在使用组件时填充内容。

基本插槽 

class SlotComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            

                

                    默认标题
                

                

                    默认内容
                

            

        `;
    }
}

customElements.define('slot-component', SlotComponent);

插槽的默认内容

插槽可以包含默认内容，当没有提供相应内容时显示： 

class DefaultSlotComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            

                
默认标题

                
这是默认的描述内容，当没有提供描述时会显示这段文字。

                

                    
                

            

        `;
    }
}

customElements.define('default-slot-component', DefaultSlotComponent);

插槽事件

当插槽内容发生变化时，会触发slotchange事件： 

class SlotChangeComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        shadow.innerHTML = `
            
            
            

                
插槽变化检测示例

                默认内容
                
插槽变化次数: 0

            

        `;
        
        // 监听插槽变化
        const slot = shadow.querySelector('slot[name="content"]');
        let changeCount = 0;
        
        slot.addEventListener('slotchange', () => {
            changeCount++;
            shadow.getElementById('change-count').textContent = changeCount;
            console.log('插槽内容已更改');
        });
    }
}

customElements.define('slot-change-component', SlotChangeComponent);

模板与插槽结合使用

模板和插槽可以结合使用，创建更灵活的可复用组件： 

class TemplateSlotComponent extends HTMLElement {
    constructor() {
        super();
        const shadow = this.attachShadow({mode: 'open'});
        
        // 创建模板
        const template = document.createElement('template');
        template.innerHTML = `
            
            
            

                

                    

                        A
                    

                    

                        

                            作者名
                        

                        

                            发布日期
                        

                    

                

                
                

                    文章标题
                

                
                

                    文章内容
                

                
                

                    
                        默认标签
                    
                

            

        `;
        
        shadow.appendChild(template.content.cloneNode(true));
    }
}
