变换

变换基础

SVG变换允许你对元素进行平移、旋转、缩放和倾斜操作。

translate 平移

translate变换用于移动元素的位置。

rotate 旋转

rotate变换用于旋转元素。

scale变换用于缩放元素。

skew 倾斜

skewX和skewY变换用于倾斜元素。

矩阵变换 (matrix)

matrix变换使用6个参数定义2D变换矩阵。

matrix(a, b, c, d, e, f)对应变换矩阵： ["matrix",[["a", "c", "e" ],["b", "d", "f"],[0, 0, 1]],"p2"]

组合变换

多个变换可以组合使用，按从左到右的顺序应用。

transform-origin

transform-origin属性可以指定变换的原点。
变换可以应用于任何SVG元素，包括形状、文本和组。组合变换时要注意顺序，不同的顺序会产生不同的结果。
