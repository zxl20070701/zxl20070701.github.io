动画

动画基础

SVG提供了多种方式创建动画，包括SMIL动画、CSS动画和JavaScript动画。

<animate> 元素

使用<animate>元素可以动画化元素的属性。

animate属性

  • attributeName: 要动画化的属性名
  • from: 起始值
  • to: 结束值
  • dur: 动画持续时间
  • repeatCount: 重复次数（indefinite表示无限循环）
  • begin: 动画开始时间
  • fill: 动画结束后的状态（freeze保持最后状态）

<animateTransform> 元素

用于动画化transform变换。

<animateMotion> 元素

使元素沿着路径移动。

关键帧动画

使用values属性定义多个关键帧。

CSS动画

使用CSS动画控制SVG元素。

路径动画

使用path属性定义动画路径。

组合动画

多个动画可以同时应用于一个元素。

事件触发动画

使用事件触发动画。
SVG动画提供了丰富的功能，可以创建复杂的交互式图形。SMIL动画在现代浏览器中得到广泛支持，但对于更复杂的动画需求，JavaScript动画可能更加灵活。
