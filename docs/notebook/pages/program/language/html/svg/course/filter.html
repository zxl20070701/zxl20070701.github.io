滤镜

滤镜基础

SVG滤镜使用<filter>元素定义，可以应用于任何图形元素。

模糊滤镜 <feGaussianBlur>

创建模糊效果。

阴影滤镜 <feDropShadow>

创建阴影效果。

颜色矩阵滤镜 <feColorMatrix>

调整颜色，如灰度、饱和度等。

颜色矩阵类型

  • matrix: 使用20个值的矩阵
  • saturate: 饱和度调整（0-1）
  • hueRotate: 色相旋转（角度）
  • luminanceToAlpha: 亮度转透明度

位移滤镜 <feOffset>

创建位移效果。

复合滤镜

多个滤镜效果可以组合使用。

光照滤镜

创建光照效果。
SVG滤镜功能强大，可以创建各种视觉效果。通过组合不同的滤镜元素，可以实现复杂的图像处理效果。
