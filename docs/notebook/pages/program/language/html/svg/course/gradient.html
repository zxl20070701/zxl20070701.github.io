渐变

线性渐变 <linearGradient>

线性渐变沿着一条直线从一个颜色过渡到另一个颜色。

线性渐变属性

  • x1, y1: 渐变起点的坐标
  • x2, y2: 渐变终点的坐标
  • gradientUnits: 坐标系统（userSpaceOnUse或objectBoundingBox）
  • gradientTransform: 渐变变换

径向渐变 <radialGradient>

径向渐变从一个中心点向外扩散。

径向渐变属性

  • cx, cy: 渐变中心点的坐标
  • r: 渐变的半径
  • fx, fy: 渐变的焦点坐标
  • gradientUnits: 坐标系统

渐变停止点 <stop>

stop元素定义渐变的颜色和位置。

stop属性

  • offset: 停止点的位置（0%到100%）
  • stop-color: 停止点的颜色
  • stop-opacity: 停止点的不透明度

渐变变换

可以对渐变应用变换。

渐变重用

定义在<defs>中的渐变可以被多个元素使用。
渐变是SVG中创建复杂视觉效果的重要工具。通过组合不同的渐变类型和停止点，可以实现丰富的色彩效果。
