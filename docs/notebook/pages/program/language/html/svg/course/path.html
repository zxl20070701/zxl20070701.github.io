路径

路径基础

SVG路径是使用<path>标签创建的，通过d属性定义路径数据。

路径命令

移动命令 (M, m)

M命令用于移动画笔到指定位置，不绘制线条。

直线命令 (L, l, H, h, V, v)

L命令绘制直线到指定坐标，H命令绘制水平线，V命令绘制垂直线。

曲线命令 (C, c, S, s, Q, q, T, t)

三次贝塞尔曲线 (C)

二次贝塞尔曲线 (Q)

弧线命令 (A, a)

弧线命令的参数：

  • rx: x轴半径
  • ry: y轴半径
  • x-axis-rotation: 椭圆的旋转角度
  • large-arc-flag: 大弧标志（0或1）
  • sweep-flag: 顺时针标志（0或1）
  • x: 终点x坐标
  • y: 终点y坐标

闭合路径 (Z, z)

Z命令用于闭合当前路径。

相对坐标与绝对坐标

大写字母命令使用绝对坐标，小写字母命令使用相对坐标。
路径命令可以组合使用，创建复杂的图形。理解每个命令的参数是掌握SVG路径的关键。
