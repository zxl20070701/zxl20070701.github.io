基本形状

矩形 <rect>

矩形是最基本的SVG形状之一，使用<rect>标签创建。

主要属性：

  • x: 矩形左上角的x坐标
  • y: 矩形左上角的y坐标
  • width: 矩形宽度
  • height: 矩形高度
  • rx: 圆角矩形x方向的圆角半径
  • ry: 圆角矩形y方向的圆角半径

圆形 <circle>

使用<circle>标签创建圆形。

主要属性：

  • cx: 圆心的x坐标
  • cy: 圆心的y坐标
  • r: 圆的半径

椭圆 <ellipse>

使用<ellipse>标签创建椭圆。

主要属性：

  • cx: 椭圆中心的x坐标
  • cy: 椭圆中心的y坐标
  • rx: 椭圆的x半径
  • ry: 椭圆的y半径

直线 <line>

使用<line>标签创建直线。

主要属性：

  • x1: 起点的x坐标
  • y1: 起点的y坐标
  • x2: 终点的x坐标
  • y2: 终点的y坐标

折线 <polyline>

使用<polyline>标签创建折线，由多个点连接而成。

主要属性：

  • points: 一系列用逗号或空格分隔的坐标点

多边形 <polygon>

使用<polygon>标签创建多边形，与折线类似，但会自动闭合。

主要属性：

  • points: 一系列用逗号或空格分隔的坐标点
所有基本形状都可以使用通用的属性如fill（填充色）、stroke（描边色）、stroke-width（描边宽度）等来设置样式。
