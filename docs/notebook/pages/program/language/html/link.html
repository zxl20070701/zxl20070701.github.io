link标签

link标签用于在HTML文档中定义外部资源的关系，最常见的用途是链接CSS样式表，但它也可以用于预加载资源、定义网站图标等。link标签是空元素，只包含属性，没有结束标签。

属性详解

rel属性

rel属性定义当前文档与链接资源之间的关系，是最重要的属性之一。

常用rel值

  • stylesheet：链接CSS样式表
  • icon：定义网站图标（favicon）
  • preload：预加载资源
  • prefetch：预获取资源
  • dns-prefetch：DNS预解析
  • preconnect：预连接
  • canonical：规范URL
  • alternate：替代版本（如RSS、移动版）

href属性

href属性指定链接资源的URL。

type属性

type属性指定链接资源的MIME类型。对于CSS样式表，通常是"text/css"。

media属性

media属性指定样式表适用的媒体类型或媒体查询。

hreflang属性

hreflang属性指定链接资源的语言。

sizes属性

sizes属性指定图标的大小，主要用于不同尺寸的favicon。

crossorigin属性

crossorigin属性用于配置CORS（跨域资源共享）请求。

integrity属性

integrity属性允许浏览器检查获取的资源是否被篡改，提供了额外的安全性。

as属性

as属性与rel="preload"一起使用，指定预加载资源的类型。

注意事项

  • link标签应该放在head部分
  • 对于CSS样式表，应该尽早加载以避免页面闪烁
  • 使用integrity属性可以提高安全性，但需要定期更新哈希值
  • 预加载资源时要谨慎，避免浪费带宽
  • 对于第三方资源，考虑使用crossorigin属性
