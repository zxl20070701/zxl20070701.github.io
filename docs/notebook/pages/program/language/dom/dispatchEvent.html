自定义事件的触发

dispatchEvent

向一个指定的事件目标派发一个Event，并以合适的顺序（同步地）调用所有受影响的EventListener，语法如下： 

    el.dispatchEvent(event);

比如我们有一个按钮，我们给他添加了onclick事件： 


<script>
function doBack(event) {
    console.log(event);
}
</script>

我们可以看见打印的内容： 

    PointerEvent {isTrusted: true, pointerId: 1, width: 1, height: 1, pressure: 0, …}

除了直接点击按钮外，你还可以dispatchEvent来模拟点击： 

dom.dispatchEvent(new PointerEvent('click', {
    clientX: 77,
    clientY: 77,
    // ...
}));
