控制台设置颜色和背景色

在Node.js的控制台输出中，你可以使用ANSI转义码来改变文本的字体颜色和背景色。ANSI转义码是一种在终端中控制文本格式的标准方法，包括颜色和背景色。

字体颜色 

黑色：\x1b[30m
红色：\x1b[31m
绿色：\x1b[32m
黄色：\x1b[33m
蓝色：\x1b[34m
洋红色（品红）：\x1b[35m
青色：\x1b[36m
白色：\x1b[37m

背景色 

黑色背景：\x1b[40m
红色背景：\x1b[41m
绿色背景：\x1b[42m
黄色背景：\x1b[43m
蓝色背景：\x1b[44m
洋红色（品红）背景：\x1b[45m
青色背景：\x1b[46m
白色背景：\x1b[47m

加粗文本 

加粗：\x1b[1m

重置颜色

要重置颜色设置到默认值，可以使用： 

重置：\x1b[0m

示例

以下是一个简单的示例： 

console.log('\x1b[31m红色文字\x1b[0m'); // 红色文字，重置颜色
console.log('\x1b[42m绿色背景文字\x1b[0m'); // 绿色背景文字，重置颜色
console.log('\x1b[34m\x1b[43m蓝色文字，黄色背景\x1b[0m'); // 蓝色文字，黄色背景，重置颜色
