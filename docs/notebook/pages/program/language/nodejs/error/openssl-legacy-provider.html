Error: error:0308010C:digital envelope routines::unsupported

错误描述

启动或打包项目的时候，报错（类似下面）：

js/chunk-f8275756.6fe3e9e5.js from Terser
Error: error:0308010C:digital envelope routines::unsupported

报错原因

出现这个错误是因为node.js V17版本中最近发布的OpenSSL3.0, 而OpenSSL3.0对允许算法和密钥大小增加了严格的限制，可能会对生态系统造成一些影响.

在node.js V17以前一些可以正常运行的的应用程序,但是在 V17 版本可能会抛出这个异常。

如何解决？

方案一：版本降级

尝试卸载Node.js 17+版本并重新安装Node.js 16+版本，然后再重新启动。

方案二：设置参数

在执行运行项目命令之前，先运行这个命令：

Linux&Unix 

    export NODE_OPTIONS=--openssl-legacy-provider

Windows 

    set NODE_OPTIONS=--openssl-legacy-provider
