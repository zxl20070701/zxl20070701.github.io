软链接

软链接（Symbolic Link）是一种特殊的文件类型，它包含了对另一个文件或目录的引用，而不是实际的数据。

创建

类 Unix 系统

可以使用 fs.symlink 方法来创建软链接，例如： 

const fs = require('fs');
const path = require('path');
 
// 目标文件的路径（可以是文件夹或文件，下同）
const target = path.join(__dirname, 'targetFile.txt');
// 软链接的路径
const symlink = path.join(__dirname, 'symlinkToTarget.txt');
 
// 创建软链接
fs.symlink(target, symlink, 'file', (err) => {
  if (err) {
    console.error('创建软链接失败:', err);
    return;
  }
  console.log('软链接创建成功');
});

Windows

可以使用 fs.symlink 并指定类型为 junction（仅适用于目录），或者你可以使用 Node.js 的 child_process 模块调用系统的 mklink 命令： 

const { exec } = require('child_process');
const target = 'C:\\path\\to\\target';
const symlink = 'C:\\path\\to\\symlink';
 
exec(`mklink ${symlink} ${target}`, (error, stdout, stderr) => {
  if (error) {
    console.error(`执行出错: ${error}`);
    return;
  }
  console.log(`软链接创建成功: ${stdout}`);
});
