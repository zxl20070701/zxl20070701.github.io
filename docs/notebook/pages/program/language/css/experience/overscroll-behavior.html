overscroll-behavior

属性用于在尝试滚动超出滚动边界时，关闭元素的滚动链（scroll chaining）或过度滚动反馈（overscroll affordance）。

滚动链：当在一个元素上过度滚动时，会导致父元素的滚动行为（默认行为）。

语法 

    overscroll-behavior: auto|contain|none|initial|inherit;

具体描述如下：

值 描述
auto 允许滚动链和过度滚动反馈行为（默认值）。
contain 允许过度滚动反馈行为，但不允许滚动链。
none 不允许过度滚动反馈或滚动链行为。
initial 将此属性设置为其默认值（比如继承无效等）。
inherit 从其父元素继承此属性。

缩写

属性是以下属性的简写形式：

  • overscroll-behavior-x
  • overscroll-behavior-y
