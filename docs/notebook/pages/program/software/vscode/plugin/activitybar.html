在左侧Activity Bar（活动栏）中添加一个按钮

效果展示

最终效果如下：

在Activity Bar（活动栏）中添加了一个按钮，点击以后在Primary Sidebar（主侧边栏）中打开了一个views页面。

配置

在 package.json 中配置好activitybar并定义使用的view： 

"contributes": {
    "viewsContainers": {
        "activitybar": [{
            "id": "Helper",
            "title": "Auxiliary",
            "icon": "images/logo.png"
        }]
    },
    "views": {
        "Helper": [
            {
            "id": "vscode-auxiliary.entry",
            "type": "webview",
            "name": "Welcome"
            }
        ]
    }
}

上面定义好activitybar，使用view/Helper作为视图，此视图id为vscode-auxiliary.entry。

注册WebviewViewProvider

为id是vscode-auxiliary.entry的视图定义提供者（extension.js文件中）： 

class UtilsWebviewViewProvider implements vscode.WebviewViewProvider {
    public resolveWebviewView(webviewView: vscode.WebviewView) {
        webviewView.webview.html = "页面内容";
    }
}

function activate(context) {
    context.subscriptions.push(
        vscode.window.registerWebviewViewProvider("vscode-auxiliary.entry", new UtilsWebviewViewProvider())
    ); 
}
