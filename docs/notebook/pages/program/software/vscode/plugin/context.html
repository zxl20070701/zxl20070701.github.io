添加右键菜单

效果展示

最终效果如下（以编辑器界面显示菜单为例）：

配置

定义子菜单

比如 “Auxiliary” -> "CSS" -> "iPhone中的安全显示区域" 这个菜单链中，有两个子菜单（“Auxiliary”、"CSS"）和一个按钮（"iPhone中的安全显示区域"），所以我们这里需要定义两个子菜单： 

"contributes": {
    "submenus": [{
        "id": "vscode-auxiliary.codeSnippet",
        "label": "Auxiliary"
    },
    {
        "id": "vscode-auxiliary.codeSnippet.css",
        "label": "CSS"
    }]
}

定义菜单

子菜单准备好了一会，直接定义菜单，并挂载好子菜单直接的关系即可： 

"contributes": {
    "menus": {
        "editor/context": [{
          "submenu": "vscode-auxiliary.codeSnippet",
          "group": "navigation"
        }],
        "vscode-auxiliary.codeSnippet": [{
          "submenu": "vscode-auxiliary.codeSnippet.css"
        }],
        "vscode-auxiliary.codeSnippet.css": [{
          "command": "vscode-auxiliary.codeSnippet.css.iphone-safe-view"
        }]
    }
}

叶子按钮是定义的一个命令。

显示位置

界面

editor/context（编辑器界面）、explorer/context（文件目录界面）、terminal/context（命令行界面）等。
