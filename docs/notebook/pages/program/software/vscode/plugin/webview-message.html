和Webview通讯

插件发消息，Webview接收消息

插件里的JS 

    panel.webview.postMessage("来自插件的数据");

Webview里的JS 

window.addEventListener('message',function(e){
    // 来自插件的数据
    console.log(e.data);
});

Webview发消息，插件接收消息

Webview里的JS 

var vscode = acquireVsCodeApi();

vscode.postMessage("来自H5的数据");

插件里的JS 

panel.webview.onDidReceiveMessage(function(data) {
    // 来自H5的数据
    console.log(data);
});
