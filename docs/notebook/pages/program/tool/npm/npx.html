命令行执行之NPX

安装 

    npm install -g npx

基本使用 

    npx <pkg> [参数]

例如： 

    npx oipage-cli -v

运行Github上项目 

    npx github:<username>/<pkg> [参数]

例如有个项目： https://github.com/oi-contrib/OIPage （OIPage项目可能不支持这样执行，因为缺乏入口脚本，只是一个例子），只需要： 

    npx github:oi-contrib/OIPage -v
注意，远程代码必须是一个模块，即必须包含package.json和入口脚本。
