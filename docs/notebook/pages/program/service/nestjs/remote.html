常用的请求相关

基础语法

GET请求 

import { Get, Query } from '@nestjs/common'

...
    @Get('/my-get')
    doDemo(@Query() query:any):any{
        return {}
    }
...

其中query就是Get请求携带的参数。

POST请求 

import { Post, Body } from '@nestjs/common'

...
    @Post('/my-post')
    doDemo(@Body() body:any):any{
        return {}
    }
...

其中body就是Post请求携带的参数。

更多装饰器

温馨提示：下面的装饰器如果没有特别说明，基本上所有的请求类型都可以使用。

@Request()

所有的请求信息都包含在其中。

@Headers(name?: string)

默认包含所有的请求头信息。

或者只想查看其中确定的请求头content-type，就可以这样：： 

import { Post, Headers } from '@nestjs/common'

...
    @Post('/my-post')
    doDemo(@Headers('content-type') headers:any):any{
        return {}
    }
...

@Header(name: string,value: string)

用于设置响应头，下面是一个简单的例子： 

import { Post, Header } from '@nestjs/common'

...
    @Post('/my-post')
    @Header('Content-Type','application/json; charset=utf-8')
    doDemo():any{
        return {}
    }
...

如果需要设置的响应头参数有多个，同样写多个Header装饰器即可。

更多常用装饰器

@Param(name?: string)

获取路由参数： 

import { Get, Param } from '@nestjs/common'

...
    @Get('/users/:id')
    getUser(@Param('id') id: string) {
        return { id };
    }
...

@HttpCode(statusCode: number)

设置响应状态码： 

import { Post, HttpCode } from '@nestjs/common'

...
    @Post()
    @HttpCode(201)
    create() {
        return 'This action adds a new user';
    }
...

@Redirect(url?: string, statusCode?: number)

重定向到指定URL： 

import { Get, Redirect } from '@nestjs/common'

...
    @Get('docs')
    @Redirect('https://docs.nestjs.com', 301)
    getDocs() {
        return { url: 'https://docs.nestjs.com' };
    }
...

模块和依赖注入

创建服务 

import { Injectable } from '@nestjs/common';

@Injectable()
export class UsersService {
    private readonly users = [];

    create(user: any) {
        this.users.push(user);
    }

    findAll(): any[] {
        return this.users;
    }
}

在控制器中使用服务 

import { Controller, Get, Post, Body } from '@nestjs/common';
import { UsersService } from './users.service';

@Controller('users')
export class UsersController {
    constructor(private readonly usersService: UsersService) {}

    @Post()
    create(@Body() user: any) {
        return this.usersService.create(user);
    }

    @Get()
    findAll() {
        return this.usersService.findAll();
    }
}

异常处理 

import { HttpException, HttpStatus } from '@nestjs/common';

...
    @Get(':id')
    findOne(@Param('id') id: string) {
        const user = this.usersService.findOne(id);
        if (!user) {
            throw new HttpException('User not found', HttpStatus.NOT_FOUND);
        }
        return user;
    }
...

中间件 

import { Injectable, NestMiddleware } from '@nestjs/common';
import { Request, Response, NextFunction } from 'express';

@Injectable()
export class LoggerMiddleware implements NestMiddleware {
    use(req: Request, res: Response, next: NextFunction) {
        console.log('Request...');
        next();
    }
}
