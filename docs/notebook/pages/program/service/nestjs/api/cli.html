常用命令 
    npm install -g @nestjs/cli

创建模块 

    nest g module 文件名称

创建控制器 

    nest g controller 文件名称 --no-spec

创建服务 

    nest g service 文件名称 --no-spec

创建守卫 

    nest g guard 文件名称 --no-spec

创建中间件 

    nest g middleware 文件名称 --no-spec

创建管道 

    nest g pipe 文件名称 --no-spec

创建拦截器 

    nest g interceptor 文件名称 --no-spec

创建过滤器 

    nest g filter 文件名称 --no-spec

创建资源 

    nest g resource 文件名称 --no-spec

启动项目 

    npm run start:dev

构建项目 

    npm run build
