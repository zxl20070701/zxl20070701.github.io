光照系统

光照是3D图形学中的重要概念，它能让场景看起来更加真实。WebGL中的光照主要包括环境光、漫反射光和镜面反射光。

光照类型

环境光 (Ambient Light)

环境光是场景中的基础光照，均匀地照亮所有物体，不考虑光源位置和方向： 

// 环境光计算
vec3 ambient = ambientStrength * lightColor;

漫反射光 (Diffuse Light)

漫反射光模拟光线照射到粗糙表面时的散射效果，依赖于光线方向和表面法线： 

// 计算漫反射
vec3 norm = normalize(normal);
vec3 lightDir = normalize(lightPos - fragPos);
float diff = max(dot(norm, lightDir), 0.0);
vec3 diffuse = diff * lightColor;

镜面反射光 (Specular Light)

镜面反射光模拟光线在光滑表面的高光反射： 

// 计算镜面反射
vec3 viewDir = normalize(viewPos - fragPos);
vec3 reflectDir = reflect(-lightDir, norm);
float spec = pow(max(dot(viewDir, reflectDir), 0.0), 32);
vec3 specular = specularStrength * spec * lightColor;

Phong光照模型

Phong光照模型结合上述三种光照： 

// Phong光照模型
vec3 result = (ambient + diffuse + specular) * objectColor;

光源类型

平行光 (Directional Light)

平行光模拟远处的光源，如太阳光，所有光线平行： 

// 平行光
vec3 lightDir = normalize(-light.direction);

点光源 (Point Light)

点光源从一个点向所有方向发射光线，强度随距离衰减： 

// 点光源衰减
float distance = length(light.position - fragPos);
float attenuation = 1.0 / (light.constant + light.linear * distance + 
    light.quadratic * (distance * distance));

聚光灯 (Spot Light)

聚光灯从一个点向特定方向发射锥形光线： 

// 聚光灯
vec3 lightDir = normalize(light.position - fragPos);
float theta = dot(lightDir, normalize(-light.direction));
float epsilon = light.cutOff - light.outerCutOff;
float intensity = clamp((theta - light.outerCutOff) / epsilon, 0.0, 1.0);

在着色器中实现光照

顶点着色器中传递法线和位置： 

// 顶点着色器
attribute vec3 a_normal;
attribute vec3 a_position;
varying vec3 v_normal;
varying vec3 v_fragPos;

uniform mat4 u_modelMatrix;
uniform mat4 u_viewMatrix;
uniform mat4 u_projectionMatrix;

void main() {
    v_fragPos = vec3(u_modelMatrix * vec4(a_position, 1.0));
    v_normal = mat3(transpose(inverse(u_modelMatrix))) * a_normal;
    gl_Position = u_projectionMatrix * u_viewMatrix * u_modelMatrix * vec4(a_position, 1.0);
}

片段着色器中计算最终颜色： 

// 片段着色器
precision mediump float;
varying vec3 v_normal;
varying vec3 v_fragPos;
uniform vec3 u_lightPos;
uniform vec3 u_viewPos;
uniform vec3 u_lightColor;
uniform vec3 u_objectColor;

void main() {
    // 环境光
    float ambientStrength = 0.1;
    vec3 ambient = ambientStrength * u_lightColor;
    
    // 漫反射光
    vec3 norm = normalize(v_normal);
    vec3 lightDir = normalize(u_lightPos - v_fragPos);
    float diff = max(dot(norm, lightDir), 0.0);
    vec3 diffuse = diff * u_lightColor;
    
    // 镜面反射光
    float specularStrength = 0.5;
    vec3 viewDir = normalize(u_viewPos - v_fragPos);
    vec3 reflectDir = reflect(-lightDir, norm);
    float spec = pow(max(dot(viewDir, reflectDir), 0.0), 32.0);
    vec3 specular = specularStrength * spec * u_lightColor;
    
    vec3 result = (ambient + diffuse + specular) * u_objectColor;
    gl_FragColor = vec4(result, 1.0);
}

多光源处理

在实际应用中，通常需要处理多个光源： 

// 处理多个光源
vec3 result = vec3(0.0);
for(int i = 0; i < NR_POINT_LIGHTS; i++)
    result += calcPointLight(pointLights[i], norm, fragPos, viewDir);
