WebGPU渲染管线

WebGPU的渲染管线是一个高度可配置的过程，它定义了从顶点数据到最终像素颜色的整个处理流程。

渲染管线概述

WebGPU渲染管线的主要阶段：

顶点输入 → 顶点着色器 → 图元装配 → 光栅化 → 片段着色器 → 输出合并

管线创建

基础管线配置 

const pipeline = device.createRenderPipeline({
    layout: 'auto', // 自动布局
    vertex: {
        module: vertexShaderModule,
        entryPoint: 'main',
        buffers: [/* 顶点缓冲区布局 */]
    },
    fragment: {
        module: fragmentShaderModule,
        entryPoint: 'main',
        targets: [{ format: presentationFormat }]
    },
    primitive: {
        topology: 'triangle-list', // 图元类型
        stripIndexFormat: undefined, // 索引格式
        frontFace: 'ccw', // 正面方向
        cullMode: 'none' // 剔除模式
    },
    depthStencil: {
        depthWriteEnabled: true,
        depthCompare: 'less',
        format: 'depth24plus'
    },
    multisample: {
        count: 1, // 采样数
        mask: 0xFFFFFFFF,
        alphaToCoverageEnabled: false
    }
});

图元装配

图元类型

  • point-list: 点列表
  • line-list: 线列表
  • line-strip: 线条带
  • triangle-list: 三角形列表
  • triangle-strip: 三角形条带

剔除和正面 

primitive: {
    topology: 'triangle-list',
    frontFace: 'ccw', // 逆时针为正面
    cullMode: 'back' // 剔除背面
}

管线布局

定义资源绑定布局： 

const bindGroupLayout = device.createBindGroupLayout({
    entries: [
        {
            binding: 0,
            visibility: GPUShaderStage.VERTEX | GPUShaderStage.FRAGMENT,
            buffer: { type: 'uniform' }
        },
        {
            binding: 1,
            visibility: GPUShaderStage.FRAGMENT,
            texture: { sampleType: 'float' }
        },
        {
            binding: 2,
            visibility: GPUShaderStage.FRAGMENT,
            sampler: { type: 'filtering' }
        }
    ]
});

const pipelineLayout = device.createPipelineLayout({
    bindGroupLayouts: [bindGroupLayout]
});

高级着色器技术

实例化渲染 

// 顶点着色器中的实例化
struct InstanceData {
    modelMatrix: mat4x4<f32>,
    color: vec4<f32>
};

@group(0) @binding(0) var<storage> instances: array<InstanceData>;

@vertex
fn main(@location(0) position: vec3<f32>,
        @builtin(instance_index) instanceIdx: u32) -> VertexOutput {
    var output: VertexOutput;
    let instance = instances[instanceIdx];
    output.position = u_projection * u_view * instance.modelMatrix * vec4<f32>(position, 1.0);
    output.color = instance.color;
    return output;
}

几何着色器替代方案

WebGPU没有传统的几何着色器，但可以通过计算着色器实现类似功能： 

// 计算着色器生成几何体
@compute @workgroup_size(64)
fn main(@builtin(global_invocation_id) global_id: vec3<u32>) {
    let index = global_id.x;
    if (index >= u_vertexCount) {
        return;
    }
    
    // 生成顶点数据
    vertices[index].position = generatePosition(index);
    vertices[index].normal = generateNormal(index);
}
