weex基础概念

什么是Weex

Weex是一个使用Web开发体验来开发高性能原生应用的框架。它结合了Web开发的灵活性和原生应用的性能体验。

Weex架构

  • JS Framework: 运行在JavaScript引擎中，负责JS代码的执行
  • JS Bridge: 连接JS和原生代码的桥梁
  • Render Engine: 负责将Virtual DOM渲染为原生组件
  • Native Components: 原生UI组件，提供高性能的用户体验

Weex工作原理

  1. 开发者编写.vue文件，包含template、style和script
  2. Weex编译器将.vue文件转换为JS Bundle
  3. JS Bundle通过JS Bridge与原生端通信
  4. 原生端根据指令创建和更新原生组件

Weex与Vue的关系

Weex基于Vue.js 2.x版本，但并非完全兼容。Weex移除了DOM相关的API，增加了原生组件和模块的支持。
