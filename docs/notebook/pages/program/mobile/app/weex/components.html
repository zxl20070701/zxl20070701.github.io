weex内置组件

容器组件

  • div: 基础的容器组件，相当于HTML的div
  • scroller: 可滚动的容器，支持垂直和水平滚动
  • list: 高性能的列表组件，支持单元格回收和复用
  • refresh: 下拉刷新组件，通常与list或scroller配合使用
  • loading: 上拉加载更多组件，通常与list或scroller配合使用

基础组件

  • text: 文本组件，用于显示文字内容
  • image: 图片组件，支持本地和网络图片
  • input: 输入框组件，支持文本、密码、数字等类型
  • textarea: 多行文本输入组件

表单组件

  • switch: 开关组件
  • checkbox: 复选框组件
  • radio: 单选框组件
  • slider: 滑块组件
  • picker: 选择器组件，支持日期、时间等

特殊组件

  • web: 网页容器，用于嵌入网页内容
  • video: 视频播放组件
  • a: 超链接组件，支持页面跳转
