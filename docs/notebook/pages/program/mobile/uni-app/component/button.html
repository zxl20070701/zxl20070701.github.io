button

属性说明

属性名 类型 默认值 说明
size String default 按钮的大小，有效值：default, mini
type String default 按钮的样式类型，有效值：primary, default, warn
plain Boolean false 按钮是否镂空，背景色透明
disabled Boolean false 是否禁用
loading Boolean false 名称前是否带 loading 图标
form-type String 用于 form 组件，点击分别会触发 form 组件的 submit/reset 事件，有效值：submit, reset
open-type String 开放能力，有效值：contact, share, getPhoneNumber, getUserInfo, launchApp, openSetting, feedback, getRealnameAuthInfo
