map

属性说明

属性名 类型 默认值 说明
longitude Number 中心经度
latitude Number 中心纬度
scale Number 16 缩放级别，取值范围为3-20
markers Array<Object> 标记点
polyline Array<Object> 路线
circles Array<Object> 圆
controls Array<Object> 控件
include-points Array<Object> 缩放视野以包含所有给定的坐标点
show-location Boolean false 显示带有方向的当前定位点
polygons Array<Object> 多边形
enable-3D Boolean false 是否显示3D楼块
show-compass Boolean false 是否显示指南针
enable-overlooking Boolean false 是否开启俯视
enable-zoom Boolean true 是否支持缩放
enable-scroll Boolean true 是否支持拖动
enable-rotate Boolean false 是否支持旋转
enable-satellite Boolean false 是否开启卫星图
enable-traffic Boolean false 是否开启实时路况
@markertap EventHandle 点击标记点时触发，e.detail = {markerId}
@callouttap EventHandle 点击标记点对应的气泡时触发，e.detail = {markerId}
@controltap EventHandle 点击控件时触发，e.detail = {controlId}
@regionchange EventHandle 视野发生变化时触发
@tap EventHandle 点击地图时触发
@updated EventHandle 在地图渲染更新完成时触发
