预览图片

uni.previewImage(OBJECT)

预览图片。 

// 从相册选择图片后预览
uni.chooseImage({
    success: function (res) {
        uni.previewImage({
            urls: res.tempFilePaths,
            current: res.tempFilePaths[0]
        });
    }
});

// 或者直接预览网络图片
uni.previewImage({
    urls: [
        'https://example.com/image1.jpg',
        'https://example.com/image2.jpg',
        'https://example.com/image3.jpg'
    ],
    current: 'https://example.com/image1.jpg'
});

参数说明

参数名 类型 必填 说明
urls Array<String> 是 需要预览的图片链接列表
current String 否 当前显示图片的链接，默认为 urls 的第一张
indicator Boolean 否 是否显示指示器（如：1/5），默认为 true
loop Boolean 否 是否可循环预览，默认为 false
