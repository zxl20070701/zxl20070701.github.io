消息提示框

uni.showToast(OBJECT)

显示消息提示框。 

uni.showToast({
    title: '操作成功',
    icon: 'success',
    duration: 2000
});

// 显示加载中
uni.showToast({
    title: '加载中',
    icon: 'loading',
    duration: 2000
});

// 不显示图标
uni.showToast({
    title: '提示信息',
    icon: 'none',
    duration: 2000
});

参数说明

参数名 类型 必填 说明
title String 是 提示的内容，长度与 icon 取值有关
icon String 否 图标，有效值详见下方说明
image String 否 自定义图标的本地路径，image 的优先级高于 icon
duration Number 否 提示的延迟时间，单位毫秒，默认：1500
mask Boolean 否 是否显示透明蒙层，防止触摸穿透，默认：false
position String 否 纯文本轻提示显示位置，有效值：top, center, bottom，默认：bottom

uni.hideToast()

隐藏消息提示框。 

uni.hideToast();

uni.showLoading(OBJECT)

显示 loading 提示框, 需主动调用 uni.hideLoading 才能关闭提示框。 

uni.showLoading({
    title: '加载中'
});

// 隐藏 loading
setTimeout(function () {
    uni.hideLoading();
}, 2000);

uni.hideLoading()

隐藏 loading 提示框。 

uni.hideLoading();
