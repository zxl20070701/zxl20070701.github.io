组件中无法修改子组件样式

默认情况下，微信小程序中自定义组件中再修改使用到的组件样式在H5中是可以的，而在小程序中会失败，此时，你可以在自定义组件中添加如下内容： 

export default {
    options: {
        styleIsolation: "shared"
    }
}

对于支付宝小程序，需要在 manifest.json 中添加如下配置： 

"mp-alipay": {
    styleIsolation: "shared"
}

v-for运行异常

上面的代码在微信小程序端会无法正常显示或别的错误，是因为key的问题，需要修改为： 

    :key="index"

slotMultipleInstance

微信小程序中，如果多个插槽无法正确解析，可以在 manifest.json 中添加如下配置： 

"mp-weixin": {
    "slotMultipleInstance": true
}
