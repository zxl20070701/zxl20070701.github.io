路由传参

$state.go()

page1.html 中有个button按钮，点击button实现状态切换的效果： 

<!-- 传递 两个参数：参数一为name, 参数二为age -->
<button ng-click="doit('zxl20070701','17')">Page 2</button>

其中doit方法如下： 

$scope.doit=function(name,age){
     $state.go("page2", {
        name,age
     });
};

下面是对应的路由配置： 

.state('page2',{
        url:"/page2/:name/:age",
        templateUrl: "views/page2.html",
        controller: 'page2Controller'
    })

目标页面取值方式： 

// {name: 'zxl20070701', age: '17'}
console.log($state.params);
