快速开始

安装必要的依赖： 

pip3 install deepagents langchain langchain-openai langchain-anthropic

比如我们本次安装的版本： 

deepagents==1.9.0
langchain==0.3.0
langchain-openai==0.2.0
langchain-anthropic==0.2.0

然后，准备好model后就可以创建智能体，然后执行任务了： 

from deepagents import create_deep_agent
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="qwen3.5",
    api_key="ollama",
    base_url="http://localhost:11434/v1",  # 设置自定义 LLM 地址
)

agent = create_deep_agent(
    model=llm,
    tools=[],
    system_prompt="",
)

result = agent.invoke({
    "messages": [{"role": "user", "content": "你是谁?"}]
})

print(result["messages"][-1]["content"])
