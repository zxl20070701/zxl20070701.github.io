快速开始

安装必要的依赖： 

npm i --save @mastra/core

比如我们本次安装的版本： 

{
    "@mastra/core": "^1.32.1"
}

基本的 Mastra 使用示例： 

import { Agent } from "@mastra/core/agent";

const myAgent = new Agent({
    id: "my-agent",
    name: "My Agent",
    instructions: ``,
    model: {
        providerId: "ollama",
        modelId: "qwen3.5",
        url: "http://127.0.0.1:11434/v1",
        apiKey: "",
    },
    tools: {},
});

myAgent.generate("你是谁？", {
    maxSteps: 10,
}).then(function (res) {
    console.log(res.text);
});
