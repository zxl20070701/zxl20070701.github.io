入门简介

一个开源的机器学习库，主要用于进行计算机视觉（CV）、自然语言处理（NLP）、语音识别等领域的研究和开发。

官方文档地址：https://docs.pytorch.org/docs/stable/index.html。

安装&引入

使用 pip 安装 pytorch： 

    pip install torch torchvision torchaudio

为了确保 PyTorch 已正确安装，我们可以通过执行以下 PyTorch 代码来验证是否安装成功： 

import torch

# 当前安装的 PyTorch 库的版本
print(torch.__version__)
# 检查 CUDA 是否可用
print(torch.cuda.is_available())

如果 torch.cuda.is_available() 输出 True，则说明 PyTorch 成功识别到你的 GPU。

架构总览

PyTorch 采用模块化设计，由多个相互协作的核心组件构成：

PyTorch 架构图
生态系统
torchvision torchtext torchaudio 其他专业库
核心
torch.nn
(神经网络) 		torch.optim
(优化器) 		torch.utils
(工具函数)
torch 核心
(张量计算) 		autograd
(自动微分) 		torch.utils.data
(数据加载)

执行流程

Python代码 → C++核心计算 → 底层CUDA/C库加速 → 返回结果。
