入门简介

和Tensorflow.js配套的可视化库。

官方中文地址：https://js.tensorflow.org/api_vis/latest/?hl=zh-cn。

安装&引入

NPM 

npm install --save @tensorflow/tfjs-vis

然后引入即可： 

import * as tfvis from '@tensorflow/tfjs-vis';

CDN 

<script src="https://cdn.jsdelivr.net/npm/@tensorflow/tfjs-vis"></script>
