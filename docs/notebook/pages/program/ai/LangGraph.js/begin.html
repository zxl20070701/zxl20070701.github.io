快速开始

安装必要的依赖： 

npm i --save @langchain/langgraph @langchain/core @langchain/openai

比如我们本次安装的版本： 

{
    "@langchain/langgraph": "^1.2.9",
    "@langchain/core": "^1.1.41",
    "@langchain/openai": "^1.4.4"
}

LangGraph 是一个用于构建有状态、多参与者 LLM 应用的库： 

import { ChatOpenAI } from "@langchain/openai";
import { StateGraph } from "@langchain/langgraph";
import { BaseMessage } from "@langchain/core/messages";

// 定义状态
interface GraphState {
    messages: BaseMessage[];
}

// 初始化 LLM
const llm = new ChatOpenAI({
    model: "qwen3.5",
    apiKey: "ollama",
    configuration: {
        baseURL: "http://localhost:11434/v1", // 设置自定义 LLM 地址
    },
});

// 定义节点函数
async function callModel(state: GraphState): Promise<Partial<GraphState>> {
    const messages = state.messages;
    const response = await llm.invoke(messages);
    return { messages: [...messages, response] };
}

// 创建图
const workflow = new StateGraph<GraphState>({
    channels: {
        messages: null,
    },
})
    .addNode("model", callModel) 添加节点
    .addEdge("__start__", "model") // 添加边
    .addEdge("model", "__end__");

// 编译图
const app = workflow.compile();

// 运行图
app.invoke({
    messages: [{ role: "user", content: "你好，请介绍一下 LangGraph。" }],
}).then((result: any) => {
    console.log(result.messages[result.messages.length - 1].content);
});

LangGraph 使用状态图的概念，允许定义复杂的工作流程和状态转换。
