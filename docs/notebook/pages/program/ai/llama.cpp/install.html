安装

llama.cpp 是一个用 C/C++ 编写的轻量级 LLM 推理库，支持多种操作系统，包括 macOS、Windows、Linux。

源码编译安装

llama.cpp 主要通过源码编译的方式进行安装，首先需要克隆仓库： 

git clone https://github.com/ggerganov/llama.cpp
cd llama.cpp

macOS

在 macOS 上，可以使用以下命令进行编译： 

cmake -B build
cmake --build build --config Release

依赖项

llama.cpp 基本编译不需要额外依赖，但如果需要特定功能，可能需要安装：

  • OpenBLAS：用于 CPU 加速
  • CUDA Toolkit：用于 NVIDIA GPU 加速
  • ROCm：用于 AMD GPU 加速

验证安装

编译完成后，可以通过以下命令验证是否成功： 

./build/bin/llama-cli --help

如果显示帮助信息，则说明编译成功。
