基本使用

llama.cpp 提供了多种可执行文件用于不同的任务，最基本的用法是使用 main 程序运行模型。

下载模型

首先，你需要下载一个兼容的模型文件（通常是 GGUF 格式）。可以从 Hugging Face （Huggingface 国内镜像站 hf-mirror.com ） 或其他来源获取：

访问 hf-mirror.com，搜索目标模型（如 “Qwen3.5-9B GGUF”），下载 ‌GGUF 格式‌ 的模型文件，这是 llama.cpp 支持的唯一模型格式。 
# 例如下载 Llama 2 7B 模型
wget https://huggingface.co/TheBloke/Llama-2-7B-Chat-GGUF/resolve/main/llama-2-7b-chat.Q4_K_M.gguf

基本推理

使用 main 程序运行模型： 

./build/bin/llama-cli -m models/llama-2-7b-chat.Q4_K_M.gguf -p "Hello, how are you?"

参数说明：

  • -m：指定模型文件路径
  • -p：指定提示词

交互式聊天

启用交互模式进行多轮对话： 

./build/bin/llama-cli -m models/llama-2-7b-chat.Q4_K_M.gguf -i

在交互模式下，你可以连续输入问题，模型会记住对话历史。

常用参数

参数 说明 示例
-m 模型文件路径 -m models/model.gguf
-p 提示词 -p "Hello, how are you?"
-i 交互模式 -i
-n 生成token数量 -n 256
-t 使用的线程数 -t 8
-c 上下文长度 -c 2048
-temp 温度参数（0-1） -temp 0.7
-b 批处理大小 -b 512
