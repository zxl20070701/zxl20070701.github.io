高级功能

llama.cpp 除了基本的推理功能外，还提供了许多高级功能，包括模型量化、API 服务器、嵌入生成等。

模型量化

模型量化可以显著减少模型大小和内存使用，同时保持合理的性能。llama.cpp 提供了强大的量化工具。

使用 quantize 工具 

# 查看可用的量化类型
./build/bin/llama-quantize --help

# 将 FP16 模型量化为 Q4_K_M
./build/bin/llama-quantize models/llama-2-7b-chat.gguf models/llama-2-7b-chat-Q4_K_M.gguf Q4_K_M

量化类型说明

量化类型 说明 相对大小
Q4_0 4位量化，旧格式 ~4.5GB (7B模型)
Q4_1 4位量化，改进版 ~4.8GB (7B模型)
Q4_K_S 4位量化，小尺寸 ~4.3GB (7B模型)
Q4_K_M 4位量化，平衡型 ~4.5GB (7B模型)
Q5_0 5位量化 ~5.3GB (7B模型)
Q5_1 5位量化，改进版 ~5.5GB (7B模型)
Q5_K_S 5位量化，小尺寸 ~5.1GB (7B模型)
Q5_K_M 5位量化，平衡型 ~5.3GB (7B模型)
Q6_K 6位量化 ~6.2GB (7B模型)
Q8_0 8位量化 ~7.5GB (7B模型)
F16 16位浮点 ~13.5GB (7B模型)
F32 32位浮点 ~26.5GB (7B模型)

API 服务器

llama.cpp 提供了 HTTP API 服务器，可以通过网络接口访问模型。

启动服务器 

# 启动服务器，监听 8080 端口
./build/bin/llama-server -m models/llama-2-7b-chat.Q4_K_M.gguf --host 0.0.0.0 --port 8080

# 使用多线程
./build/bin/llama-server -m models/llama-2-7b-chat.Q4_K_M.gguf -t 8 --host 0.0.0.0 --port 8080

API 端点

服务器启动后，可以使用以下 API 端点：

兼容 OpenAI 的 API 位于 http://localhost:8080/v1/
  • POST /completion：文本补全
  • POST /chat/completions：聊天补全
  • POST /embedding：生成嵌入向量
  • GET /models：列出可用模型
  • GET /health：健康检查

使用示例 

# 使用 curl 调用 API
curl http://localhost:8080/completion \
    -H "Content-Type: application/json" \
    -d '{
        "prompt": "Hello, how are you?",
        "n_predict": 128
    }'

嵌入生成

使用 embedding 工具生成文本的嵌入向量： 

# 生成单个文本的嵌入
./build/bin/llama-embedding -m models/llama-2-7b-chat.Q4_K_M.gguf -p "Hello world"

# 从文件读取文本
./build/bin/llama-embedding -m models/llama-2-7b-chat.Q4_K_M.gguf -f input.txt

性能优化

批处理

使用批处理可以提高吞吐量： 

./build/bin/llama-cli -m models/llama-2-7b-chat.Q4_K_M.gguf -p "Hello" -n 128 -b 512

内存映射

对于大模型，可以使用内存映射减少内存使用： 

./build/bin/llama-cli -m models/llama-2-7b-chat.Q4_K_M.gguf --mmap

内存锁定

锁定模型到内存可以提高性能（需要足够内存）： 

./build/bin/llama-cli -m models/llama-2-7b-chat.Q4_K_M.gguf --mlock

多 GPU 支持

如果有多个 GPU，可以指定使用哪些 GPU： 

# 使用 GPU 0 和 GPU 1
./build/bin/llama-cli -m models/llama-2-7b-chat.Q4_K_M.gguf -ngl 35 --tensor-split 3,1

# 仅使用 GPU 1
./build/bin/llama-cli -m models/llama-2-7b-chat.Q4_K_M.gguf -ngl 35 --main-gpu 1
