tf.tensor

语法 

    tf.tensor(values, shape?, dtype?)

通过给定的values，shape和dtype创建一个张量。

values

表示这个张量的值，类型包括：TypedArray | Array | WebGLData | WebGPUData（如果是TypedArray，可以是Uint8Array, Uint8ClampedArray, Int32Array, Float32Array）。

shape

应该是一个数组，表示数据的“形状”，比如： 

tf.tensor([1, 2, 3, 4, 5, 6], [2, 3]).print();

结果就是： 

[
    [1, 2, 3], 
    [4, 5, 6]
]

当然，你也可以不提供，直接： 

tf.tensor([[1, 2], [3, 4]]).print();

可以推断出 shape = [2, 3] 。

dtype

表示数据的类型，可以是：'float32'|'int32'|'bool'|'complex64'|'string'。
