tf.train.adam

语法 

    tf.train.adam(learningRate?, beta1?, beta2?, epsilon?)

创建 Adam 优化器。

learningRate

学习率，默认为 0.001。

beta1

第一个动量参数，默认为 0.9。

beta2

第二个动量参数，默认为 0.999。

epsilon

数值稳定性参数，默认为 1e-8。

示例 

const optimizer = tf.train.adam(0.001);
const model = tf.sequential({
  layers: [
    tf.layers.dense({ inputShape: [1], units: 1 })
  ]
});

// 编译模型
model.compile({
  optimizer: optimizer,
  loss: 'meanSquaredError'
});
