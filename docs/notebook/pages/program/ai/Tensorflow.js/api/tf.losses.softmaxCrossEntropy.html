tf.losses.softmaxCrossEntropy

语法 

    tf.losses.softmaxCrossEntropy(onehotLabels, logits, weights?, labelSmoothing?, reduction?)

计算 softmax 交叉熵损失。

onehotLabels

one-hot 编码的标签张量。

logits

网络的 logits 输出。

weights

可选的权重张量。

labelSmoothing

标签平滑参数，默认为 0。

reduction

减少类型，可以是 'none'、'mean' 或 'sum'，默认为 'mean'。

示例 

const labels = tf.tensor2d([[1, 0], [0, 1]], [2, 2]);
const logits = tf.tensor2d([[2, 1], [1, 2]], [2, 2]);
tf.losses.softmaxCrossEntropy(labels, logits).print();
