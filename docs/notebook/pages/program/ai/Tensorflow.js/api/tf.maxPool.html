tf.maxPool

语法 

    tf.maxPool(x, filterSize, strides, pad, dataFormat?, dimRoundingMode?)

最大池化操作。

x

输入张量，形状为 [batch, height, width, inChannels]。

filterSize

池化窗口大小，可以是单个数字或长度为 2 的数组 [filterHeight, filterWidth]。

strides

池化步长，可以是单个数字或长度为 2 的数组 [strideHeight, strideWidth]。

pad

填充方式，可以是 'valid' 或 'same'。

dataFormat

数据格式，默认为 'NHWC'。

示例 

const x = tf.tensor4d([1, 2, 3, 4], [1, 2, 2, 1]);
tf.maxPool(x, 2, 1, 'valid').print();

输出： 

[[[[4]]]]
