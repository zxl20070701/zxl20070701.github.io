tf.mean

语法 

    tf.mean(x, axis?, keepDims?)

计算张量的平均值。

x

输入张量。

axis

指定要减少的维度。如果为 null（默认值），则计算所有元素的平均值。

keepDims

如果为 true，则保留被减少的维度，长度设置为 1。

示例 

const x = tf.tensor2d([[1, 2], [3, 4]]);
tf.mean(x).print(); // 所有元素的平均值
tf.mean(x, 0).print(); // 按列计算平均值
tf.mean(x, 1).print(); // 按行计算平均值

输出： 

2.5
[2, 3]
[1.5, 3.5]
