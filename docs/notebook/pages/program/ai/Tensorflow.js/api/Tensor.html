Tensor
tf.Tensor extends TensorInfo

dataSync()

同步获取一个张量的值，比如我们有如下代码： 

let myT=tf.tensor([1, 2, 3, 4, 5, 6], [3, 2]);
console.log(myT.dataSync());

打印结果： 

Float32Array(6) [1, 2, 3, 4, 5, 6, buffer: ArrayBuffer(24), byteLength: 24, 
    byteOffset: 0, length: 6, Symbol(Symbol.toStringTag): 'Float32Array']
