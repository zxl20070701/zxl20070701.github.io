tf.conv2d

语法 

    tf.conv2d(x, filter, strides, pad, dataFormat?, dilations?, dimRoundingMode?)

二维卷积操作。

x

输入张量，形状为 [batch, height, width, inChannels]。

filter

卷积核张量，形状为 [filterHeight, filterWidth, inChannels, outChannels]。

strides

卷积步长，可以是单个数字或长度为 2 的数组 [strideHeight, strideWidth]。

pad

填充方式，可以是 'valid' 或 'same'。

dataFormat

数据格式，默认为 'NHWC'。

dilations

扩张率，可以是单个数字或长度为 2 的数组 [dilationHeight, dilationWidth]。

示例 

const x = tf.tensor4d([1, 2, 3, 4], [1, 2, 2, 1]);
const filter = tf.tensor4d([1, 1, 1, 1], [2, 2, 1, 1]);
tf.conv2d(x, filter, 1, 'valid').print();

输出： 

[[[[10]]]]
