tf.matMul

语法 

    tf.matMul(a, b, transposeA?, transposeB?)

矩阵乘法操作。

a

第一个矩阵。

b

第二个矩阵。

transposeA

如果为 true，则在乘法之前转置 a。

transposeB

如果为 true，则在乘法之前转置 b。

示例 

const a = tf.tensor2d([1, 2, 3, 4], [2, 2]);
const b = tf.tensor2d([5, 6, 7, 8], [2, 2]);
tf.matMul(a, b).print();

输出： 

[[19, 22],
 [43, 50]]
