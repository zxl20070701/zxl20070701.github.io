tf.train.sgd

语法 

    tf.train.sgd(learningRate)

创建随机梯度下降 (SGD) 优化器。

learningRate

学习率。

示例 

const optimizer = tf.train.sgd(0.01);
const model = tf.sequential({
  layers: [
    tf.layers.dense({ inputShape: [1], units: 1 })
  ]
});

// 编译模型
model.compile({
  optimizer: optimizer,
  loss: 'meanSquaredError'
});
