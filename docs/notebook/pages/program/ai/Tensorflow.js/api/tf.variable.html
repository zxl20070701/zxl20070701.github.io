tf.variable

语法 

    tf.variable(initialValue, trainable?, name?, dtype?);
  1. initialValue也就是用一个张量来初始化；
  2. trainable是布尔类型，默认true，可变；
  3. name是字符串类型，默认使用一个唯一的id；
  4. dtype数据类型。

例子 

var myTensor = tf.tensor2d([[1, 2], [3, 4], [5, 6]]);
    
var myVariable=tf.variable(myTensor);
myVariable.print();

打印结果： 

Tensor
  [[1, 2],
   [3, 4],
   [5, 6]]
