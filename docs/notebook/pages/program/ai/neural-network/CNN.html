卷积神经网络（CNN）

卷积神经网络主要由这几类层构成：输入层、卷积层，ReLU层、池化（Pooling）层和全连接层（实际应用中往往将卷积层与ReLU层共同称之为卷积层）。

卷积层

将原来的对象变小一些，但又能保证原来的信息尽量多的保留下来。

过滤器

过滤器的作用是寻找图片的特征。

每一个过滤器中的数值，都是算法自己学习来的，不需要我们费心去设置。需要我们做的有：

  1. 设置过滤器的大小（比如3x3）
  2. 设置过滤器滑动的步幅数（比如1）
  3. 设置过滤器的个数（比如4个）

ReLU层

卷积神经网络中常用的激活函数层，通过公式max(0, x)实现。
