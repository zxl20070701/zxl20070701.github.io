快速开始

安装必要的依赖： 

pip3 install langgraph langchain-core langchain-openai

比如我们本次安装的版本： 

anggraph==0.2.0
langchain-core==0.3.0
langchain-openai==0.2.0

LangGraph 是一个用于构建有状态、多参与者 LLM 应用的库： 

from typing import TypedDict, List, Optional
from langchain_openai import ChatOpenAI
from langgraph.graph import StateGraph
from langchain_core.messages import BaseMessage, HumanMessage

# 定义状态
class GraphState(TypedDict):
    messages: List[BaseMessage]

# 初始化 LLM
llm = ChatOpenAI(
    model="qwen3.5",
    api_key="ollama",
    base_url="http://localhost:11434/v1"  # 设置自定义 LLM 地址
)

# 定义节点函数
def call_model(state: GraphState) -> GraphState:
    messages = state["messages"]
    response = llm.invoke(messages)
    return {"messages": messages + [response]}

# 创建图
workflow = StateGraph(GraphState)

# 添加节点
workflow.add_node("model", call_model)

# 添加边
workflow.add_edge("__start__", "model")
workflow.add_edge("model", "__end__")

# 编译图
app = workflow.compile()

# 运行图
result = app.invoke({
    "messages": [HumanMessage(content="你好，请介绍一下 LangGraph。")]
})

print(result["messages"][-1].content)

LangGraph 使用状态图的概念，允许定义复杂的工作流程和状态转换。
