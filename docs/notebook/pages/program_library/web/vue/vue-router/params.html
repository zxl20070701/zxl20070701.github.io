参数传递

利用Query路由传参

利用URL上的键值来传递参数的，会将参数暴露在URL上： 

import {useRoute} from 'vue-router'
const route=useRoute()

// 带查询参数，结果是 /register?plan=private
router.push({ path: '/register', query: { plan: 'private' } })

获取的地方： 

import {useRoute} from 'vue-router'
const route=useRoute()

console.log(route.query.plan) // 'private'

利用params传递

将数据储存在后台内存之中，当页面刷新之后其数据会丢失： 

import {useRoute} from 'vue-router'
const route=useRoute()

router.push({ name: 'user', params: { username: 'eduardo' } })

获取的地方： 

import {useRoute} from 'vue-router'
const route=useRoute()

console.log(route.params.username) // 'eduardo'

使用state

参数不显示在链接后面。 

import {useRoute} from 'vue-router'
const route=useRoute()

router.push({ path: '/register', state: { plan: 'private' } })

获取的地方： 

console.log(history.state.plan) // 'private'
