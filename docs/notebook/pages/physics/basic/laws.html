物理定律与公式

物理定律是描述自然界中物理现象基本规律的科学表述。这些定律通过数学公式表达，是理解自然世界的基础。

经典力学定律

牛顿运动定律

  1. 第一定律（惯性定律）：物体在不受外力作用时，保持静止或匀速直线运动状态
    公式：当 F = 0 时，a = 0
  2. 第二定律（加速度定律）：物体的加速度与作用力成正比，与质量成反比
    公式：F = ma（力 = 质量 × 加速度）
  3. 第三定律（作用与反作用定律）：作用力与反作用力大小相等，方向相反
    公式：F₁₂ = -F₂₁

万有引力定律

任何两个有质量的物体之间都存在引力，引力大小与两物体质量的乘积成正比，与它们距离的平方成反比。

公式：F = G(m₁m₂)/r²

其中：

  • F - 引力大小
  • G - 万有引力常数（6.67×10⁻¹¹ N·m²/kg²）
  • m₁, m₂ - 两个物体的质量
  • r - 两物体质心间的距离

动量守恒定律

在没有外力作用的系统中，系统的总动量保持不变。

公式：m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'

能量守恒定律

能量既不会凭空产生，也不会凭空消失，只能从一种形式转换为另一种形式，或者从一个物体转移到另一个物体。

公式：E₁ + E₂ = E₁' + E₂'

热力学定律

热力学第一定律

系统内能的增加等于系统吸收的热量减去系统对外做的功。

公式：ΔU = Q - W

热力学第二定律

热量总是自发地从高温物体传向低温物体，不可能自发地从低温物体传向高温物体。

电磁学定律

库仑定律

两个点电荷之间的作用力与它们的电荷量的乘积成正比，与它们距离的平方成反比。

公式：F = k(q₁q₂)/r²

其中：

  • k - 库仑常数（8.99×10⁹ N·m²/C²）
  • q₁, q₂ - 两个电荷的电量
  • r - 两电荷间的距离

欧姆定律

导体中的电流与导体两端的电压成正比，与导体的电阻成反比。

公式：I = U/R

法拉第电磁感应定律

闭合电路中感应电动势的大小与穿过该电路的磁通量的变化率成正比。

公式：ε = -N(ΔΦ/Δt)

光学定律

反射定律

入射光线、反射光线和法线在同一平面内，入射角等于反射角。

公式：θᵢ = θᵣ

折射定律（斯涅尔定律）

入射光线、折射光线和法线在同一平面内，入射角的正弦与折射角的正弦之比等于两种介质折射率之比。

公式：n₁sinθ₁ = n₂sinθ₂

现代物理学定律

质能方程

质量和能量是等价的，可以相互转换。

公式：E = mc²

光电效应方程

光子的能量等于逸出功加上电子的最大初动能。

公式：hν = W + Eₖ

生活中的物理公式应用

  • 自由落体：h = ½gt²（计算物体下落高度）
  • 抛物运动：x = v₀cosθ·t, y = v₀sinθ·t - ½gt²（计算抛射物轨迹）
  • 简谐振动：x = Acos(ωt + φ)（描述弹簧振子运动）
  • 理想气体状态方程：PV = nRT（描述气体状态变化）
  • 功率计算：P = W/t = Fv（计算机械功率）
