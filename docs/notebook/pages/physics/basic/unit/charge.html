电荷单位

国际单位是库仑，符号C。

单位 符号 转换
库仑 C 1C = 1A·s
毫库仑 mC 1C = 1000mC
微库仑 μC 1mC = 1000μC

什么是电荷？

电荷是物质的一种基本属性，就像质量一样。电荷分为正电荷和负电荷，同种电荷相斥，异种电荷相吸。你可以把电荷想象成一种"电力货币"，它决定了物体之间电相互作用的大小。

生活中的电荷

  • 静电：冬天脱毛衣时的"噼啪"声和火花，就是积累的静电荷放电现象，大约有几微库仑
  • 闪电：一次闪电携带约15-350库仑的电荷，相当于数百万个家用电池的总和
  • 手机电池：一部智能手机电池容量约2000-4000毫安时，相当于7200-14400库仑
  • 人体静电：人在地毯上行走后积累的静电荷约0.1-1微库仑，但电压可达几千伏
  • 电容器：相机闪光灯的电容器可储存约0.01-0.1库仑的电荷

不同单位的应用场景

  • C（库仑）：大容量电池、电容器、雷电等大型电荷量测量
  • mC（毫库仑）：中型电子设备、闪光灯、小型电容器
  • μC（微库仑）：静电现象、微小电容器、精密电子元件
