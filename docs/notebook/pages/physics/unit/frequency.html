频率单位

国际单位是赫兹，符号Hz。

单位 符号 转换
吉赫 GHz 1GHz = 10⁹Hz
兆赫 MHz 1MHz = 10⁶Hz
千赫 kHz 1kHz = 1000Hz
赫兹 Hz 1Hz = 1s⁻¹

什么是频率？

频率就是某件事情在单位时间内重复的次数。1赫兹表示每秒钟发生1次。你可以把频率想象成心跳的快慢——心跳越快，频率越高。频率越高，振动或波动的速度就越快。

生活中的频率

  • 心跳：正常成年人安静时心率约60-100次/分钟，相当于1-1.7赫兹
  • 交流电：中国和欧洲家用交流电频率50赫兹，美国60赫兹，这就是为什么有些进口电器需要转换器
  • 广播：FM广播频率范围88-108兆赫，AM广播540-1600千赫
  • WiFi：家用WiFi通常使用2.4吉赫或5吉赫频段
  • 5G网络：5G网络使用频段可达几十吉赫，传输速度更快
  • 可见光：人眼可见光的频率约430-750太赫兹（1太赫兹=10¹²赫兹）
  • 蜜蜂翅膀：蜜蜂飞行时翅膀振动频率约230赫兹，这就是为什么我们能听到嗡嗡声

不同单位的应用场景

  • GHz（吉赫）：现代计算机CPU、手机处理器、微波通信等高速电子设备
  • MHz（兆赫）：老式计算机、无线电广播、电视信号、早期手机网络
  • kHz（千赫）：音频处理、老式无线电、超声波清洗等中等频率应用
  • Hz（赫兹）：机械振动、电力系统、音频低频、基础物理实验
