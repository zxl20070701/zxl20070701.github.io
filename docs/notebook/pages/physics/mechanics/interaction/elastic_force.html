弹力

弹力是物体因形变而产生的恢复力，遵循胡克定律。对于理想弹性体，弹力与形变量成正比：

["join", "F", "=", "-", "kx"]

其中，F是弹力大小，k是弹性系数，x是形变量（负号表示弹力方向与形变方向相反）。

胡克定律的适用范围

胡克定律仅在弹性限度内成立：

  • 弹性限度：材料能够完全恢复原状的最大应力
  • 比例极限：应力与应变成严格正比关系的最大应力
  • 屈服点：材料开始发生塑性变形的应力点

弹性势能

弹性形变物体储存的势能为：

["join", "Eₚ", "=", ["division", "kx²", "2"]]

该公式适用于遵循胡克定律的弹性体。

弹性系数

弹性系数k取决于材料的性质和几何形状：

  • 弹簧：k = Gd⁴/(8nD³)，其中G是剪切模量，d是钢丝直径，n是有效圈数，D是弹簧中径
  • 杆件拉伸：k = EA/L，其中E是弹性模量，A是截面积，L是长度
  • 梁的弯曲：k = 3EI/L³，其中I是截面惯性矩

弹力的微观解释

从微观角度看，弹力源于原子间相互作用势：

  • 键能曲线：原子间势能随距离变化的非对称曲线
  • 平衡位置：势能最小值对应的原子间距
  • 回复力：偏离平衡位置时产生的恢复力

复杂弹性系统

多个弹簧组合时的等效弹性系数：

  • 串联：1/k_eq = 1/k₁ + 1/k₂ + ... + 1/kₙ
  • 并联：k_eq = k₁ + k₂ + ... + kₙ

非线性弹性

当形变较大时，胡克定律不再适用：

  • 非线性弹性：应力-应变关系为非线性
  • 超弹性材料：如橡胶，可承受大变形
  • 粘弹性：同时表现出弹性和粘性特征

动态弹性

考虑惯性效应时的弹性振动：

  • 简谐振动：m(d²x/dt²) + kx = 0
  • 阻尼振动：m(d²x/dt²) + c(dx/dt) + kx = 0
  • 受迫振动：m(d²x/dt²) + c(dx/dt) + kx = F₀cos(ωt)
