安装

在 Windows 上安装 SQLite

打开https://www.sqlite.org/download.html下载页面，比如下载 sqlite-tools-win-x64-3500400.zip压缩文件。

温馨提示：如果是ARM芯片，参考上面的，此时应该下载sqlite-tools-win-arm64-3500400.zip即可。

然后把压缩包解压，比如到C:\sqlite中，然后把路径C:\sqlite添加到PATH环境变量中。

然后在命令行执行下面命令： 

sqlite3

一起顺利的话，就可以看见类似下面的打印： 

SQLite version 3.50.4 2025-07-30 19:33:53
Enter ".help" for usage hints.
Connected to a transient in-memory database.
Use ".open FILENAME" to reopen on a persistent database.
sqlite>

至此，你已经完成了在Windows上的安装。

在 Mac OS X 上安装 SQLite

最新版本的 Mac OS X 会预安装 SQLite，但是如果没有可用的安装，只需按照如下步骤进行：

打开https://www.sqlite.org/download.html下载页面，比如下载 sqlite-tools-osx-x64-3500400.zip压缩文件。

温馨提示：如果是ARM芯片，参考上面的，此时应该下载sqlite-tools-osx-arm64-3500400.zip即可。

然后把压缩包解压，比如到/Users/zxl20070701/sqlite中，然后把路径/Users/zxl20070701/sqlite添加到PATH环境变量中。

你可以按照上述Windows讲解的进行验证。
