创建第一个项目

使用 django-admin 来创建 HelloWorld 项目： 

    django-admin startproject HelloWorld

创建完成后我们可以查看下项目的目录结构： 

HelloWorld/                  
    manage.py                
    HelloWorld/              
        __init__.py         
        settings.py          
        urls.py             
        asgi.py             
        wsgi.py

项目根目录 (HelloWorld/)中的manage.py是Django 命令行工具入口，用于运行开发服务器、数据库迁移等操作。

主项目配置目录 (HelloWorld/HelloWorld/) ：

文件/目录 作用
__init__.py 空文件，告诉 Python 该目录是一个包。
settings.py 核心配置文件，包含：
- 数据库设置 (DATABASES)
- 静态文件路径 (STATIC_URL)
- 应用注册 (INSTALLED_APPS)
- 调试模式 (DEBUG=True/False)
urls.py 主路由配置文件，定义 URL 路径与视图的映射关系。
asgi.py ASGI 服务器配置（用于异步 Web 服务器，如 Daphne）。
wsgi.py WSGI 服务器配置（用于传统 Web 服务器，如 Gunicorn、uWSGI）。

接下来我们进入 HelloWorld 目录输入以下命令，更新数据库结构： 

    python3 manage.py migrate

可以看见此时，项目根目录 (HelloWorld/)中多了一个db.sqlite3，其是SQLite 数据库文件（默认数据库，开发环境使用）。

现在，终于可以启动了： 

    python3 manage.py runserver 0.0.0.0:8080

启动成功会发现，主项目配置目录 (HelloWorld/HelloWorld/)中多了一个文件夹__pycache__，其是Python 字节码缓存目录（自动生成，无需手动修改）。
