环境搭建

安装Python

Windows 平台安装 Python

打开 WEB 浏览器访问 https://www.python.org/downloads/windows/ ，页面截图如下：

上面截图显示，当前的最新版本（Stable Releases）是 Python 3.12.1 - Dec. 8, 2023

然后直接下载其中的 Download Windows installer (64-bit)安装包。下载好以后，和普通软件一样，进行安装即可。

在安装过程中，你会遇到类似下面的界面：

此时，一定要记得勾选类似 Add Python 3.6 to PATH 的选项哦，这非常重要。

Mac 平台安装 Python

打开 WEB 浏览器访问 https://www.python.org/downloads/macos/ 后选择需要的版本下载后，和上述类似进行安装即可。

运行 Python

新建一个文本文件，取名index.py并输入如下内容： 

#!/usr/bin/python3
# coding=utf-8

print("你好呀！")

实际截图如下：

如何在命令行输入下面命令运行： 

    python3 ./index.py

运行效果如下：

可以看见，命令行打印了“你好呀！”几个字。

至此，环境就准备好了。
