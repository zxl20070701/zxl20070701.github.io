## 如何开发？

本项目的打包工具等采用自定义设计开发，开发的时候，你需要执行下面的命令启动开发服务器：

```
npm run dev
```

启动成功以后，在浏览器直接访问即可： http://127.0.0.1:20000/

如果开发完毕需要发布，执行下面命令进行打包：

```
npm run build
```
